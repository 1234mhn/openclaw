### 韩语学习
用户对照第26课台词"금방 전화할게"展开讨论：
- **-을게(요)** 词尾：向对方承诺/约定要做某事，只能用于第一人称
- **-ㄴ다고 하다** 引用别人说的话 → 但实际上引用的是"自己说过的话"当理由
- **-ㄴ다고 하니까** = "因为说了要……"

#### 关于引用别人说话的四种句型总结：
1. **陈述（他说怎么怎么样）** → 动词-ㄴ/는다고 하다 / 形容词-다고 하다 / 名词-이라고 하다
2. **命令（他让干什么）** → -라고 하다
3. **邀请（他说一起干什么）** → -자고 하다
4. **疑问（他问什么）** → -냐고 하다
5. **照搬原话** → "……"라고 하다
6. **口语缩略版** → -대요/-래요/-재요/-냬요

#### 第26课两句台词汇总（小白视角）：

**① 금방 전화할게** = 马上打给你
- -을게(요) = 我答应你我会做

**② 근데 막상 간다고 하니까 좀 무서워졌어요** = 但真到要去的时候，又有点害怕了
- -ㄴ다고 하니까 = 因为说要去（把自己说过的话当理由）
- 막상 = 真到那时候（理论vs现实）
- 무서워지다 = 变得害怕

用户纠正点：引用别人说的话就是引用别人说的话，不是自己说的话。以后不搞混。
