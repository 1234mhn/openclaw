# 五维词汇深度测试网站

**时间：** 2026-05-20
**状态：** 🛠 迭代中
**文件：** `/root/.openclaw/workspace/词汇五维测试.html`

## 产品定位
纯前端初高中英语词汇深度诊断工具，从5个维度测量单词掌握程度。

## 核心功能
- 5级词库（入门800~拔高5000+）
- 自适应抽题测词汇量（带级别封顶）
- 每词5步深度测试（读音/词性/搭配/辨句/造句）
- Canvas雷达图报告 + 截图导出

## 技术栈
HTML + CSS + JS + Canvas + html2canvas
