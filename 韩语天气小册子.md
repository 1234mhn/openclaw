          🌤️  韩语天气 · 自学手册  🌤️

          "今天天气怎么样？从这句开始聊起来！"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤷  "韩语学了字母，然后呢？"

跟韩国人聊天，最不容易冷场的话题就是——天气。
不是因为天气多有意思，而是因为每个人都生活在天气里。

这本手册不教语法，不列枯燥的表格。
就按场景走，学完一趴练一趴，练完再往下。

               ☀️  先把这些词认熟

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▎  ☀️  맑다  [막따]  —  晴朗

     "天上一朵云都没有，太阳直晒"
     
     짧은 말  →  맑은 날  =  晴朗的日子
     한마디    →  오늘 하늘이 맑아서 기분이 좋아요.
                   =  今天天空很晴朗，心情真好。

▎  ⛅  흐리다  [흐리다]  —  阴天

     "整个天灰蒙蒙的，像蒙了一层布"
     
     짧은 말  →  흐린 날씨  =  阴天
     한마디    →  내일도 계속 흐릴 거예요.
                   =  明天也会一直阴天。

▎  ☁️  구름  [구름]  —  云

     짧은 말  →  구름이 많다  =  云很多
     한마디    →  하늘에 구름이 많아요. 곧 비가 올까요?
                   =  天上云很多，马上要下雨了吧？

▎  🌧️  비  [비]  —  雨

     짧은 말  →  비가 오다  =  下雨
     한마디    →  갑자기 비가 오기 시작했어요.
                   =  突然开始下雨了。

▎  ❄️  눈  [눈]  —  雪

     짧은 말  →  눈이 오다  =  下雪
     한마디    →  올겨울에 첫눈이 왔어요! 
                   =  今年冬天初雪来了！

▎  🌬️  바람  [바람]  —  风

     짧은 말  →  바람이 불다  =  刮风
     한마디    →  바람이 많이 불어서 추워요.
                   =  风很大，很冷。

▎  🌫️  안개  [안개]  —  雾

     짧은 말  →  안개가 끼다  =  起雾
     한마디    →  오늘 아침에 안개가 많이 꼈어요.
                   =  今天早上雾很大。

▎  🌡️  기온  [기온]  —  气温

     짧은 말  →  기온이 높다/낮다  =  气温高/低
     한마디    →  오늘 최고 기온이 35도예요.
                   =  今天最高气温35度。

▎  🌡️  영상 / 영하  [영상 / 영하]  —  零上 / 零下

     짧은 말  →  영상 10도 / 영하 5도  =  零上10度/零下5度
     한마디    →  겨울에는 영하 10도까지 내려가요.
                   =  冬天会降到零下10度。

▎  💧  습도  [습또]  —  湿度

     짧은 말  →  습도가 높다  =  湿度高
     한마디    →  오늘 습도가 높아서 꿉꿉해요.
                   =  今天湿度高，潮潮的。

▎  💨  건조하다  [건조하다]  —  干燥

     짧은 말  →  건조한 날씨  =  干燥的天气
     한마디    →  겨울에는 날씨가 건조해서 물을 많이 마셔야 돼요.
                   =  冬天天气干燥，得多喝水。

▎  💧  습하다  [수파다]  ← 送气音化: ㅂ+ㅎ→ㅍ  —  潮湿

     짧은 말  →  습한 여름  =  潮湿的夏天
     한마디    →  장마철에는 날씨가 습해서 빨래가 잘 안 말라요.
                   =  梅雨季天气潮湿，衣服不容易干。

▎  🔥  덥다  [덥따]  —  热

     "韩国人夏天一天说八百遍"
     
     짧은 말  →  더운 여름  =  炎热的夏天
     한마디    →  오늘 진짜 더워요. 에어컨 틀어야겠어요.
                   =  今天真的好热，得开空调了。

▎  🧊  춥다  [춥따]  —  冷

     "冬天不说了，日常必备"
     
     짧은 말  →  추운 겨울  =  寒冷的冬天
     한마디    →  날씨가 추워졌어요. 감기 조심하세요.
                   =  天气变冷了，小心感冒。

▎  🌸  따뜻하다  [따뜨타다]  ← 送气音化  —  暖和

     짧은 말  →  따뜻한 봄날  =  温暖的春日
     한마디    →  봄이 되어서 날씨가 따뜻해졌어요.
                   =  春天到了，天气变暖和了。

▎  🍂  시원하다  [시원하다]  —  凉快（舒服的凉爽）

     "不是'冷'，是秋风吹过来的那种舒服"
     
     짧은 말  →  시원한 바람  =  凉爽的风
     한마디    →  가을 바람이 시원해서 산책하기 좋아요.
                   =  秋天的风很凉快，适合散步。

▎  🥶  쌀쌀하다  [쌀쌀하다]  —  凉飕飕

     "比시원하다冷一档，'有点冷了'"
     
     짧은 말  →  쌀쌀한 아침  =  凉飕飕的早晨
     한마디    →  아침에는 쌀쌀하니까 옷을 따뜻하게 입으세요.
                   =  早上凉飕飕的，穿暖和点。

▎  🫠  꿉꿉하다  [꿉꾸파다]  —  潮闷不舒服 ⭐韩国人夏天高频

     "汗闷在衣服里的感觉，中文没有完美对应"
     
     짧은 말  →  꿉꿉한 방  =  潮闷的房间
     한마디    →  장마철이라 방 안이 꿉꿉해요.
                   =  梅雨季，房间里潮闷闷的。

▎  🌋  후텁지근하다  [후텁찌근하다]  —  闷热潮湿 ⭐很地道

     "比꿉꿉하다更强烈，又湿又热透不过气"
     
     짧은 말  →  후텁지근한 여름밤  =  闷热的夏夜
     한마디    →  비 온 후에 날씨가 후텁지근해요.
                   =  雨后天气又湿又热。

▎  ☀️  쨍하다  [쨍하다]  —  阳光刺眼

     "不是一般的好天气，是'好到睁不开眼'"
     
     짧은 말  →  쨍한 햇살  =  刺眼的阳光
     한마디    →  오늘 햇살이 쨍해서 선글라스 꼭 챙기세요.
                   =  今天阳光刺眼，一定带墨镜。

▎  🔥  무덥다  [무덥따]  —  闷热

     짧은 말  →  무더운 날씨  =  闷热的天气
     한마디    →  무더운 여름에는 아이스커피가 최고예요.
                   =  闷热的夏天，冰咖啡最棒。

▎  🧊  한파  [한파]  —  寒潮

     짧은 말  →  한파 주의보  =  寒潮警报
     한마디    →  내일 한파가 온다고 하니까 옷을 따뜻하게 입으세요.
                   =  说明天寒潮来了，穿暖和点。

▎  🌡️  포근하다  [포근하다]  —  温暖舒适

     짧은 말  →  포근한 봄날  =  暖洋洋的春日
     한마디    →  날씨가 포근해서 나가고 싶어요.
                   =  天气暖洋洋的，好想出去。

▎  🌡️  일교차  [일교차]  —  昼夜温差

     짧은 말  →  일교차가 크다  =  昼夜温差大
     한마디    →  가을에는 일교차가 크니까 옷을 여러 겹 입으세요.

     "韩国人最常说的两个天气词之一"
     
     짧은 말  →  비가 오다  =  下雨
     한마디    →  비가 오니까 우산을 가져가세요.
                   =  下雨了，请带伞。

▎  ☂️  비가 많이 오다   —  雨下得大

     짧은 말  →  비가 많이 오는 날  =  下大雨的日子
     한마디    →  어젯밤에 비가 많이 와서 잠을 못 잤어요.
                   =  昨晚雨太大，没睡好。

▎  ☀️  비가 그치다   —  雨停

     "[비가 그처썌요] ← 连音+紧音化"
     
     짧은 말  →  비가 그친 후  =  雨停之后
     한마디    →  드디어 비가 그쳤어요. 이제 나갈 수 있어요.
                   =  终于雨停了。现在可以出门了。

▎  ❄️  눈   [눈]  —  雪

     "[누니 와요] ← 连音！"
     
     짧은 말  →  눈이 오다  =  下雪
     한마디    →  내일 눈이 온대요. 같이 눈사람 만들래요?
                   =  说明天会下雪。一起堆雪人吗？

▎  🌊  장마   [장마]  —  梅雨季（6月底~7月连下一个月）

     "一到六月韩国人就开始哀嚎"
     
     짧은 말  →  장마 기간  =  梅雨期
     한마디    →  장마가 시작됐는데 언제 끝날까요?
                   =  梅雨季开始了，什么时候结束呢？

▎  ⛈️  폭우   [포구]  ← 连音  —  暴雨

     짧은 말  →  폭우 경보  =  暴雨警报
     한마디    →  오늘 폭우가 내려서 학교에 못 갔어요.
                   =  今天下暴雨，没去成学校。

▎  ❄️  폭설   [폭썰]  ← 紧音化  —  暴雪

     짧은 말  →  폭설이 내리다  =  下暴雪
     한마디    →  폭설 때문에 비행기가 취소됐어요.
                   =  因为暴雪，航班取消了。

▎  🌧️  소나기   [소나기]  —  阵雨

     "突然下、很快停的雨"
     
     짧은 말  →  소나기가 오다  =  下阵雨
     한마디    →  갑자기 소나기가 와서 비를 맞았어요.
                   =  突然下了阵雨，被淋了。

    A: 오늘 날씨 좋네요~ ☀️
       今天天气真好～
    
    B: 네, 나가기 딱 좋은 날씨예요.
       嗯，出门刚刚好。
    
    A: 주말에도 맑을까요?

    B: 글쎄요, 일기예보를 봐야 알 것 같아요.
       这个嘛，得看天气预报。

    >  딱 = 正好 / 글쎄요 = 怎么说呢～


▎  🌧️  场景：下雨天碰到邻居

    A: 비가 많이 오네요! 우산 있어요?
       雨下好大！有伞吗？
    
    B: 네, 있어요. 감사합니다!
       嗯，有。谢谢！
    
    A: 다행이네요. 저는 우산이 없어서 큰일이에요.
       那就好。我没伞，麻烦了。
    
    B: 제 우산 같이 쓸래요?
       要用我的伞一起吗？

    >  다행이다 = 万幸 / 큰일이다 = 出大事了


▎  🥶  场景：冬天关心朋友

    A: 오늘 엄청 추워요. 따뜻하게 입으세요!
       今天超级冷。穿暖和点！
    
    B: 네, 걱정 마세요!
       嗯，别担心！
    
    A: 감기 조심해야 돼요. 요즘 독감이 유행이에요.
       得小心感冒。最近流感在流行。

    >  엄청 = 超级、巨（比진짜还夸张）
    >  걱정 마세요 = 别担心


▎  😷  场景：雾霾天吐槽

    >  꼭 = 一定 / 틀다 = 打开（机器）

          🌟  进阶！地道韩国人表达

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

这些词教科书不一定有，但韩国人天天用：

▎  ☀️  하늘이 쨍해요  [쨍하다]

    "太阳好刺眼"
    
    짧은 말  →  쨍한 햇볕  =  刺眼的阳光
    한마디    →  오후에 하늘이 쨍해서 밖에 나가기 힘들었어요.
                  =  下午阳光太刺眼，出门都困难。

    → 读出来就有种被阳光直射的感觉


▎  🌧️  날씨가 꿉꿉해요  [꿉꾸파요]

    "天气潮闷不舒服"
    
    짧은 말  →  꿉꿉한 여름날  =  潮闷的夏日
    한마디    →  비 온 후에 날씨가 꿉꿉해서 샤워했어요.
                  =  雨后天气潮闷，洗了个澡。

    → 夏天在韩国，使用率仅次于"덥다"！


▎  🌋  장마철이라 후텁지근해요  [후텁찌근해요]

    "梅雨季又湿又热"
    
    짧은 말  →  후텁지근한 밤  =  闷热的夜晚
    한마디    →  장마철이라 후텁지근해서 잠이 잘 안 와요.
                  =  梅雨季闷热，睡不好觉。

    → 韩国人一说这个词就开始叹气 😮‍💨


▎  🌸  꽃샘추위  [꼳쌤추위]  ← 紧音化

    "倒春寒"
    
    字面意思：꽃(花) + 샘(嫉妒) + 추위(寒冷)
    = "花嫉妒的寒冷"——花刚要开，冷空气回来捣乱
    
    짧은 말  →  꽃샘추위가 오다  =  倒春寒来了
    한마디    →  벌써 4월인데 꽃샘추위가 와서 다시 추워졌어요.
                  =  已经四月了却来了倒春寒，又变冷了。


▎  🌡️  일교차가 크다  [일교차]

    "昼夜温差大"
    
    짧은 말  →  일교차 큰 날  =  昼夜温差大的日子
    한마디    →  가을에는 일교차가 크니까 옷 여러 겹 입으세요.
                  =  秋天昼夜温差大，多穿几层。

    → 春秋两季韩国人天天挂在嘴边


▎  😮‍💨  눅진하다  [눅찐하다]

    "黏糊糊的潮湿"
    
    짧은 말  →  눅진한 날씨  =  黏糊的天气
    한마디    →  비가 온 후에 공기가 눅진해서 불쾌해요.
                  =  雨后空气黏糊糊的，很不舒服。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

①  "ㅎ" 碰到元音 →  直接消失！

    좋아요 → [조아요]    不是"조하요"！
    많아요 → [마나요]    不是"만하요"！
    꿉꿉해요 → [꿉꾸패요]

② 收音 ㄱㄷㅂ 后面碰到松音 →  变紧！

    덥다 → [덥따]        不是"덥다"
    맑다 → [막따]        不是"말ㄱ다"
    습도 → [습또]        不是"습도"
    폭설 → [폭썰]

③ ㅂ + ㄴ  →  ㅁ（鼻音化）

    덥네요 → [덤네요]    不是"덥네요"！
    춥네요 → [춤네요]

记住这三条，你的韩语发音就自然一半了！🎉
