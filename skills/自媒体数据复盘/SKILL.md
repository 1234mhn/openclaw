## 一、平台算法核心指标

### 📕 小红书算法权重

| 优先级 | 指标 | 说明 | 评判标准 |
|:----:|:----|:------|:--------|
| 1️⃣ | **收藏率** | 内容是否有用，最核心指标 | <3%一般 >5%🔥 >8%🔥🔥 |
| 2️⃣ | **点击率** | 封面+标题是否吸引人 | <5%差 >8%好 >12%🔥 |
| 3️⃣ | **完读率** | 用户是否看完 | >60%🔥 <30%⚠️ |
| 4️⃣ | **点赞率** | 内容是否讨喜 | <3%一般 >5%🔥 |
| 5️⃣ | **评论率** | 互动深度 | <0.5%一般 >1%🔥 |
| 6️⃣ | **互动时效性** | 发布后1小时内互动越高，推流越猛 | 前1h是关键窗口 |

| 优先级 | 指标 | 说明 | 评判标准 |
|:----:|:----|:------|:--------|

| 2️⃣ | **完播率** | 是否完整看完 | >30%🔥 <15%⚠️ |
| 3️⃣ | **平均观看时长** | 看得越久越推 | >15s🔥 <7s⚠️ |
| 4️⃣ | **点赞率** | 内容是否讨喜 | <3%一般 >5%🔥 |
| 5️⃣ | **评论率** | 互动深度 | |
| 6️⃣ | **分享率** | 分享=有价值 | |
| 7️⃣ | **关注转化率** | 看完是否关注 | |

## 二、15项功能详解

### 📝 内容分析（功能1~5）

#### 1️⃣ 每日发布内容分析
``` 每天发布后，记录：
- 发布时间
- 内容主题/标题
- 封面风格
- 标签列表
- 发布后1小时/6小时/24小时数据
```

#### 2️⃣ 第二天晚上复盘
``` 用户报数后，对比昨日数据：
- 涨了还是跌了？
- 为什么涨/跌？（从算法指标分析）
- 给明日优化建议
```

#### 3️⃣ 好保留、差改掉
| 类型 | 判断标准 | 行动 |
|:----|:---------|:-----|
| 🔥 好内容 | 收藏率>8%或完播率>30% | 保持方向，周内再发同类 |
| ⚠️ 一般 | 指标在平均水平 | 看哪个指标低，优化后重试 |
| ❄️ 差内容 | 2s跳出率>50%或收藏率<3% | 换封面/换开头/换标题 |

#### 4️⃣ 竞品对比
```
搜索同类型账号同主题内容：
- 他们的封面有什么不同？
- 标题关键词是什么？
- 发布时间？
- 互动情况？
→ 找出差距，给出调整方向
```

#### 5️⃣ 怎么留住用户
| 平台 | 留住用户的关键 |
|:----|:--------------|
| 📕 小红书 | 内容干货+收藏价值+封面吸引人 |
| 🎵 抖音 | 前3秒钩子+节奏紧凑+结尾引导关注 |

### 📈 策略优化（功能6~10）

#### 6️⃣ 周趋势追踪
```
📈 本周数据曲线
- 曝光/观看趋势
- 哪些天数据好？为什么？
- 哪些天数据差？为什么？
- 下周方向建议
```

#### 7️⃣ 标题/封面/标签分析
```
标记每篇笔记的：
🏷️ 标题类型（提问型/数字型/悬念型）
📸 封面风格（纯文字/图文/截图）
🔖 标签组合（哪些标签带来了流量）

→ 找出你的「爆款公式」
```

#### 8️⃣ 最佳发布时间分析
```
追踪每次发布时间与6小时数据的关系：
7:00~8:00 → 数据如何？
12:00~13:00 → 数据如何？
18:00~20:00 → 数据如何？
21:00~22:00 → 数据如何？

→ 找到你的黄金发布时间
```

#### 9️⃣ 粉丝画像分析
```
你的内容吸引了什么人？
- 年龄分布
- 性别比例
- 兴趣标签
→ 根据画像调整内容方向
```

#### 🔟 内容日历
```
📅 本周计划
周一：XX内容
周二：XX内容
周三：XX内容
周四：XX内容
周五：XX内容
周末：休息/日常

规律更新比一次性爆更有效
```

### 💰 变现准备（功能11~15）

#### 1️⃣1️⃣ 粉丝增长追踪
```
每日涨粉数、哪条内容涨粉最多
涨粉高的内容有什么共同点？
→ 复制涨粉密码
```

#### 1️⃣2️⃣ 高价值内容识别
```
标记适合做付费的内容：
✅ 收藏率>10% → 适合打包成「韩语资料包」
✅ 评论区求教程 → 适合做成付费课程
✅ 同主题内容>3条 → 可以合集收费
```

#### 1️⃣3️⃣ 引流转化追踪
```
如果挂了链接/引导私域：
- 有多少人点进来？
- 转化率多少？
- 哪类内容的引流效果最好？
```

#### 1️⃣4️⃣ 内容资产积累
```
按主题分类已发内容：
📁 出行类：打车/问路/乘车
📁 饮食类：点餐/邀请吃饭
📁 社交类：打招呼/问候/告别

→ 积累到一定量，直接打包成产品变现
```

#### 1️⃣5️⃣ 变现路径建议
```
根据数据表现，定期推荐变现方向：
- 内容收藏率高 → 适合卖资料包/课程
- 粉丝粘性好 → 适合做社群/陪跑
- 曝光量大 → 适合接广告
```

## 三、日常工作流

### 每日流程

```
你发布内容 → 第二天晚上报数给我（10秒）
       ↓
我按算法分析数据 → 对比昨日/上周 → 出优化建议
       ↓
你按建议调整下一条内容 → 循环
```

### 每周流程

```
每周日 → 出一份周报，包含：
   1. 本周数据汇总
   2. 🔥 最佳内容分析（为什么好）
   3. ❄️ 最差内容分析（为什么差）
   4. 竞品对比
   5. 下周内容日历
```

### 每月流程

```
每月1号 → 出一份完整策略报告，包含：
   1. 本月数据总览
   2. 粉丝增长分析
   3. 变现可行性评估
   4. 下月内容策略
```
