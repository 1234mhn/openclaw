# 📊 自媒体数据复盘系统

> 核心理念：数据不是看的，是用来指导行动的。

---

## 一、数据来源

用户每天花**10秒**告诉我数字，或发截图，我来OCR读取：

```
格式：「小红书 观看300 点赞15 收藏8 评论2 涨粉5」
格式：「抖音 播放200 点赞6 评论1 涨粉2」
```

---

## 二、数据分析维度

### 1. 单日数据卡

| 项目 | 小红书 | 抖音 |
|:----|:------|:-----|
| 👀 观看/播放 | | |
| 👍 点赞 | | |
| 📁 收藏（含合集） | | |
| 💬 评论 | | |
| 👥 涨粉 | | |
| 📈 互动率 | (点赞+收藏+评论)÷观看 | (点赞+评论+分享)÷播放 |

### 2. 关键指标评分

| 指标 | 计算公式 | 小红书标准 | 抖音标准 |
|:----|:---------|:---------:|:-------:|
| 点赞率 | 点赞÷观看 | <3%一般 >5%🔥 | <3%一般 >5%🔥 |
| 收藏率 | 收藏÷观看 | <3%一般 >5%🔥 >8%🔥🔥 | — |
| 评论率 | 评论÷观看 | <0.5%一般 >1%🔥 | <0.5%一般 >1%🔥 |
| 2s跳出率 | 2s跳出÷播放 | — | <30%🔥 >50%⚠️ |
| 5s完播率 | 看完5s÷播放 | — | >30%🔥 <20%⚠️ |

### 3. 趋势分析

```
📈 上升趋势  → 方向对了，继续发
📉 下降趋势  → 需要调整内容形式
🔄 波动不定  → 还没找到稳定方向
```

### 4. 跨平台对比

```
同一天/同主题的内容，小红书和抖音的表现差异分析：
- 小红书火、抖音冷 → 内容偏干货，适合长文
- 抖音火、小红书冷 → 内容偏短快，适合碎片化
```

## 四、内容优化建议

### 小红书优化方向

| 问题 | 建议 |
|:----|:-----|
| 观看高、收藏低 | 内容不够干货，增加实用性 |
| 收藏高、评论低 | 结尾加提问引导 |
| 观看低 | 优化封面和标题，封面加大字标题 |
| 完播率下降 | 结尾加钩子「关注看下集」 |

### 抖音优化方向

| 问题 | 建议 |
|:----|:-----|
| 2s跳出率>50% | 前3秒必须直接上冲突/原声/悬念 |
| 完播率<20% | 把视频缩短到15~20秒 |
| 评论为0 | 结尾加提问或投票 |
| 播放量<200 | 观察前3秒是不是太长了 |

## 五、内容日历规划

根据数据分析结果，规划下周发布计划：

```
📅 下周计划（6.28~7.04）

周一：发第X课「主题」（复制上周爆款方向）
周二：发日常/翻车（涨粉向）
周三：发第X+1课「主题」
周四：答疑/互动（收集评论区问题）
周五：发合集/对比（总结向）
周末：休息 or 发幕后
```
