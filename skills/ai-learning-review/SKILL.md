# AI/ML 深度学习复盘

## 核心理念

不考记忆，考理解。每个知识点经过四轮打磨：

```
一句话概括 → 追问深化 → 串联旧知 → 应用检验
```

## 四轮复盘法

### 第一轮：一句话概括
要求用户用**一句话**说清楚概念本质。

> Q: 「线性回归是什么？」
> A: 「找一条直线去拟合数据点。」

> Q: 「逻辑回归是什么？」
> A: 「用Sigmoid函数把线性输出压缩到0~1之间做分类。」

### 第二轮：追问深化
基于答案追问，把理解往深推。

> 「这条直线怎么才算'最好'？」→ 引出损失函数/MSE
> 「0~1之间的值怎么变成类别？」→ 引出阈值/决策边界

### 第三轮：串联旧知识
连接之前学过的内容，形成知识网络。

> 「这个跟你之前学的线性回归比，区别在哪？」
> 「这个损失函数跟MSE比，优缺点分别是什么？」

### 第四轮：应用检验
让知识落地到用户的具体项目场景。

> 「这个技术能在你的UNet项目上用吗？」
> 「如果数据不够怎么办？有什么办法？」

## 复盘记录模板

每次复盘后，在对应知识卡片末尾追加：

```
---
🔄 复盘记录（第X轮 YYYY-MM-DD）
一句话概括：XXX
追问点：XXX
串联：与XXX关联
应用：可用于XXX
理解程度：L1初识 / L2理解 / L3应用 / L4内化
---
```

## 引导原则

1. 答不上来 → 不直接给答案，先引导思考
2. 引导无效 → 用类比/举例解释
3. 解释完 → 让用户用自己的话复述一遍
4. 答对了 → 追问一个"为什么"，确认不是蒙的
