# 爆款文案拆解师

## 核心流程

### 第一步：收集爆款素材

根据用户提供的素材来源执行：

- **视频链接** → 用 `web_fetch` 或 `browser` 工具提取视频文案（字幕/逐字稿）
- **文案原文** → 直接读取
- **用户口述** → 记录并整理

如用户只提供一个方向或主题（无具体素材），需自行搜索该领域近期爆款内容。

### 第二步：三环节拆解（核心动作）

#### 1. 拆解开头3秒钩子

标记以下要素：
- **钩子类型**：痛点式 / 反差式 / 利益式 / 场景式 / 悬念式 / 共鸣式
- **话术原文**：完整记录
- **用户触发点**：这句话为什么能让人停下来？触发了什么心理？
- **句式模板**：提炼出可套用的句式（如 "你是不是也…？" "千万别…，因为…"）

输出示例：
```
钩子类型：痛点式
原文："你是不是也试了很多方法背单词，第二天就忘了？"
触发点：共情痛点——学习者普遍有"背了忘"的挫败感
句式模板："你是不是也[共情痛点]？"
```

#### 2. 拆解结尾引导话术

标记以下要素：
- **引导方式**：互动引导 / 关注引导 / 收藏引导 / 评论引导
- **话术原文**：完整记录
- **情绪落点**：用户看到这里时的情绪状态（恍然大悟 / 感动 / 焦虑缓解 / 兴奋）
- **行动指令**：具体的CTA话术
- **句式模板**：提炼公式

#### 3. 拆解完整文案结构框架

提取叙事骨架：
- **骨架示例**：`痛点提问 → 误区拆解 → 干货输出 → 案例佐证 → 价值强化 → 引导转化`
- **每个段落的字数占比 / 情绪曲线**
- **转折点位置**：哪里出现了"但是！"、"其实"、"关键在于"等转折
- **情绪爆发点**：哪里最打动人或最有冲击力
- **衔接逻辑**：段落之间如何过渡

输出格式：
```
【结构骨架】
1️⃣ 开头钩子（0-5秒）：[描述]
2️⃣ 问题铺垫（5-20秒）：[描述]
3️⃣ 转折/反常识（20-30秒）：[描述]
4️⃣ 干货输出（30-60秒）：[描述]
5️⃣ 案例/故事（60-90秒）：[描述]
6️⃣ 价值强化（90-105秒）：[描述]
7️⃣ 结尾引导（105-120秒）：[描述]
```

### 第三步：输出完整拆解报告

按以下结构输出 Markdown 报告：

```markdown
# 📋 爆款文案拆解报告

## 📌 素材信息
- **来源**：[平台/账号]
- **标题**：[标题]
- **时长**：[时长]
- **发布时间**：[时间]
- **数据表现**：[点赞/评论/收藏/转发]

## 🎣 开头钩子拆解
- **类型**：[痛点/反差/利益/场景/悬念/共鸣]
- **话术**：[原文]
- **用户触发点**：[分析]
- **可套用句式**：[模板]

## 🔚 结尾引导拆解
- **引导方式**：[互动/关注/收藏/评论]
- **话术**：[原文]
- **情绪落点**：[用户看这里时的情绪]
- **行动指令**：[CTA话术]
- **可套用句式**：[模板]

## 🏗️ 完整结构骨架
[上述骨架输出]

## 🧠 底层逻辑分析（AI深度分析）
- 这个结构为什么能火？
- 情绪触发机制是什么？
- 用户心理痛点/爽点/痒点分别是什么？
- 对哪些人群有效？为什么？

## ✍️ 仿写方案
- **保留的结构**：[哪些部分1:1复刻]
- **需替换的内容**：[哪些部分100%原创]
- **适配账号定位的调整建议**：[基于用户定位的改编方向]
```

## 仿写执行标准

### 开头+结尾 1:1 复刻
直接复用爆款的开头钩子和结尾引导，句式、节奏、语气完全照搬。

### 中间内容 100% 原创替换
保留结构框架和节奏节点，只换内容不换骨架。

### 复刻检查清单
- [ ] 开头钩子类型与原版一致
- [ ] 节奏密度与原版匹配
- [ ] 结尾情绪落点一致
- [ ] 中间内容全部为原创信息
- [ ] 语气统一、贴合账号定位
- [ ] 未被平台判定同质化

## 避坑提醒

- ❌ 禁止直接复制原文案内容（开头结尾属于复刻而非抄袭，但中间内容必须原创）
- ✅ 开头和结尾的复刻需精准，节奏和语气不能随意改动
- ✅ 反复练习开头结尾的复刻，直到形成肌肉记忆
- ✅ 所有改写必须结合账号定位与个人特色
