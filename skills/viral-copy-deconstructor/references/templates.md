# 爆款文案模板库

> 使用时按需查阅，不要一次性全读。根据拆解到的钩子类型或引导方式，针对性查对应模板。

---

## 一、开头钩子模板库

### 1. 痛点式
直击用户痛点，让用户产生"这不就是我吗？"的共鸣。

| 模板句式 | 示例 |
|---------|------|
| "你是不是也[痛点描述]？" | "你是不是也试了很多方法背单词，第二天就忘了？" |
| "有没有人跟我一样[痛点场景]？" | "有没有人跟我一样，一到写简历就卡壳？" |
| "[某个看似努力的行为]，其实[真相]？" | "每天背100个单词，其实效率最低的方式" |
| "[问题]到底怎么解决？" | "单词背了就忘到底怎么破？" |
| "如果你在[具体场景]，这条视频一定要看完" | "如果你在准备考研英语，这条视频一定要看完" |

### 2. 反差式
用出乎意料的信息制造认知冲击。

| 模板句式 | 示例 |
|---------|------|
| "我做了[很长时间]，才意识到[反常识观点]" | "我做了5年英语老师，才意识到背单词是效率最低的方式" |
| "大部分人认为[常识]，但真相是[反常识]" | "大部分人觉得语法要系统学，但真相是——母语者根本不学语法" |
| "[惊人事实]，你信吗？" | "考过雅思的人里，70%没背完过一本单词书，你信吗？" |
| "我[某个身份]，但今天要说的可能会得罪同行" | "我是一名英语老师，但今天要说的话可能会得罪同行" |
| "[大家都在做的事]，其实是错的" | "每天打卡背单词，其实是错的" |

### 3. 利益式
直接告诉用户看完能获得什么。

| 模板句式 | 示例 |
|---------|------|
| "看完这条视频，你将学会[具体价值]" | "看完这条视频，你将学会一个不用死记硬背的单词方法" |
| "只要[X分钟]，教你[具体技能]" | "只要1分钟，教你一个能让口语变流利的小技巧" |
| "不花一分钱，就能[利益点]" | "不花一分钱，就能让你的英语发音好听10倍" |
| "[数字]个[干货]，最后一个是王炸" | "3个免费工具帮你提高英语听力，最后一个是王炸" |
| "[动词]+这个[技巧/方法]，效果立竿见影" | "用这个背单词方法，效果立竿见影" |

### 4. 场景式
营造具体场景，让用户代入。

| 模板句式 | 示例 |
|---------|------|
| "想象一下：[场景描述]" | "想象一下：你在国外餐厅，想点菜却不知道怎么开口" |
| "当你[场景]，你会不会[问题]？" | "当你在会议上被突然点名发言，会不会大脑一片空白？" |
| "[时间/地点]，[状态]，然后[问题来了]" | "凌晨两点，你在赶论文，然后发现漏了一段关键文献" |
| "[身份/角色]都经历过的事：[场景]" | "每个职场人都经历过的事：面试被问到英语时假装听懂了" |

### 5. 悬念式
先抛一个悬念，诱导用户继续看下去。

| 模板句式 | 示例 |
|---------|------|
| "[一个决定/一件事]，让我的[X]发生了翻天覆地的变化" | "一个决定，让我的英语水平发生了翻天覆地的变化" |
| "终于有人把[复杂概念]讲清楚了" | "终于有人把词根记忆法讲清楚了" |
| "我后悔没早知道[X]" | "我后悔没早知道这个背单词的方法" |
| "[平台/领域]最不敢说的真相" | "英语培训机构最不敢说的真相" |

### 6. 共鸣式
用用户内心的真实想法建立连接。

| 模板句式 | 示例 |
|---------|------|
| "有没有那么一瞬间，你觉得自己[某种感受]" | "有没有那么一瞬间，你觉得自己英语怎么也学不好" |
| "我知道你[状态]，我以前也这样" | "我知道你学英语很焦虑，我以前也这样" |
| "[心声]，懂的人点个赞" | "学英语最痛苦的不是难，是没时间，懂的人点个赞" |

## 二、结尾引导模板库

### 1. 关注引导

| 模板句式 | 情绪落点 | 示例 |
|---------|---------|------|
| "我是[X]，关注我，带你[持续价值]" | 期待感 | "我是楠楠，关注我，带你拆解更多爆款文案" |
| "还没关注的抓紧了，因为我会持续分享[价值]" | 紧迫感 | "还没关注的抓紧了，因为我会持续分享更多英语学习方法" |
| "关注我，下期告诉你[预告内容]" | 好奇心 | "关注我，下期告诉你英语听力怎么练最有效" |

### 2. 互动引导

| 模板句式 | 情绪落点 | 示例 |
|---------|---------|------|
| "你属于第一种还是第二种？评论区告诉我" | 站队心理 | "你属于'背了忘派'还是'随缘学派'？评论区告诉我" |
| "你的[相关经历]是什么？评论区聊聊" | 分享欲 | "你的英语学习方法是什么？评论区聊聊" |
| "觉得有道理的，点个赞让更多人看到" | 认同感 | "觉得有用的，点个赞让更多人看到" |
| "[问题]？评论区扣1/扣2" | 低门槛参与 | "你是学生党还是职场人？评论区扣1学生党、扣2职场人" |

### 3. 收藏引导

| 模板句式 | 情绪落点 | 示例 |
|---------|---------|------|
| "这条建议收藏，因为[价值]" | 怕错过 | "这条建议收藏，因为不是每条视频都这么干货" |
| "先点赞收藏，以后慢慢看" | 轻松感 | "先点赞收藏，以后学英语用得上" |
| "我整理了[X]清单，建议收藏起来反复看" | 囤积欲 | "我整理了10个英语学习工具，建议收藏起来反复看" |

### 4. 评论引导

| 模板句式 | 情绪落点 | 示例 |
|---------|---------|------|
| "评论区留下你的[内容]，我来[后续动作]" | 期待反馈 | "评论区留下你的英语水平，我来给你推荐学习方法" |
| "有什么问题，评论区问我" | 解惑需求 | "有什么英语学习上的问题，评论区问我" |
| "你们还想看什么？评论区告诉我" | 被重视感 | "你们还想看哪个视频拆解？评论区告诉我" |

### 5. 综合转化（多引导叠加）

```
"觉得有用点个赞 👍 （认同感）
还没关注的抓紧了 🔔 （紧迫感）
想看我拆解更多视频的，评论区告诉我你想看什么 📝 （互动+调研）
我是楠楠，下期见 👋 （品牌记忆）"
```

## 三、常用文案骨架库

### 骨架A：痛点 → 误区 → 干货 → 案例 → 转化

| 段落 | 作用 | 时长占比 | 情绪曲线 |
|------|------|---------|---------|
| 1. 开头钩子 | 抓住注意力 | 5% | 😣 痛点 |
| 2. 问题描述（共情） | 建立信任 | 15% | 😔 无奈 |
| 3. 误区拆解 | 打破认知 | 20% | 🤔 疑惑→😲 震惊 |
| 4. 干货方案 | 输出价值 | 30% | 😌 恍然大悟 |
| 5. 案例佐证 | 增强可信 | 15% | 😄 信心 |
| 6. 价值强化 | 刺激行动 | 10% | 💪 动力 |
| 7. 引导转化 | 获取互动 | 5% | 🤝 连接 |

**适用场景：** 知识科普、学习方法、干货分享类

## 四、常见钩子组合策略

| 组合方式 | 示例 | 效果 |
|---------|------|------|
| 痛点 + 利益 | "你是不是背了忘？3个方法让你记得牢" | 先戳痛再给药，留存率极高 |
| 反差 + 悬念 | "做了5年英语老师，才敢说的真相——" | 制造期待，强烈好奇心 |
| 场景 + 共鸣 | "凌晨两点赶论文的你，一定经历过这个" | 高代入感 |
| 数据 + 反常识 | "90%的人背单词的方法都是错的" | 权威+冲击 |
| 利益 + 紧迫感 | "不花1分钱的英语学习方法，看完就删" | 价值+稀缺 |
