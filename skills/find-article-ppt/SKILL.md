# 外刊精讲PPT素材生成

## 流程

### 第一步：找外刊

搜索最新英文外刊文章，**不限于任何平台**：

**可选来源：**
- Science News / Science News for Students
- The Guardian
- National Geographic
- BBC News / BBC Learning English
- CNN / Reuters
- The New York Times
- The Economist
- 其他英文媒体或杂志

**选文标准：**
- **最新**：优先当天或近3天
- **难度适合**：内容可读，生词量适中
- **话题有趣**：科技 / 环保 / 动物 / 健康 / 生活 / 文化等
- **篇幅不限**：内容决定PPT页数，不预设固定页数
- **优先选带音频的文章**（BBC / VOA / Science News等常有音频）

### 第二步：生成PPT素材文档

生成Markdown格式文档，结构如下：

#### P1 封面
```
标题：[文章标题]
副标题：外刊精读 · [来源]
来源：[出处 + 日期]
```

#### 正文段（每段一页，页数不固定）
每段格式：
```
🎵 音频：[来源/时间段/链接]（如有）

**原文：**
[段落原文]

**中文大意：**
[段落翻译]
```

#### 重点词汇讲解（每页2-3个词，共5-8个词）
每个单词格式：
```
**单词** /音标/ (词性 中文释义)
- **英英释义：** [英文解释]
- **词性：** [名词/动词/形容词/短语等]
- **用法：** [用法说明]
- **例句：**
  1. [例句] [翻译]
  2. [例句] [翻译]
- **拓展：** [同义词/反义词/派生词]（可选）
```

#### 重点句型仿写（2-3个句子）
```
**句型①**
**原文：** [原句]
**结构：** [句型结构拆解]
**仿写：**
1. [仿写句] [翻译]
2. [仿写句] [翻译]
```

#### 课后小测验（4-6题）

### 第三步：输出

输出一份完整Markdown文件，用户可直接甩到豆包生成PPT。

文件命名：`外刊精讲_[主题]_[日期].md`
