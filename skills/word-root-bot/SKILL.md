# 词根查词Bot

## 功能
用户发一个单词过来，bot自动：
1. 查词根数据库 → 如果有匹配，输出结构化解构
2. 无匹配 → AI按词根规则实时拆解
3. 输出格式统一

## 数据源
- 内置词根数据库：/root/.openclaw/workspace/word-root-db.json
- AI补充拆解（当数据库无匹配时）

## 输出格式
```
📖 [单词]
词根：[词根]=[含义]  |  来源：[拉丁语/希腊语]

🔍 拆解：[前缀+词根+后缀] → [释义]

📌 词性：[词性]
💡 例句：[英文例句]

📎 关联词：[同词根的其他单词]
```
