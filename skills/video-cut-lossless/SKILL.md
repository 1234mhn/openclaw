# 无损剪辑助手
核心规则：**视频原始数据流完整拷贝，零压缩、零转码，画质和原文件完全一致**。音频高质量编码+0.2秒淡入淡出消除爆音。

## 最终确认格式（不可变更）
```
① 📺 先发视频（原画裁剪，2-3分钟分2-3段）
② 📝 再发台词+翻译+单词
```

## 工作流程
1. 从百度网盘取视频素材（不去外部网站搜索）
2. 使用脚本裁剪2-3分钟片段，原画质
3. 先用whisper转写确认实际台词
4. 再配上正确的翻译+重点单词
5. 先发视频 → 再发文字
6. 每天早7点自动推送一个场景

## 核心特性
1. 视频流：直接拷贝，无二次编码，画质100%还原原文件
2. 音频流：高质量编码+0.2秒淡入淡出消除爆音
3. 极速处理：无损切割，速度远高于重编码剪辑

## 调用方式
```bash
bash scripts/cut_lossless.sh <源视频> <开始秒> <结束秒> <输出路径>
```

## 重要规则
- 台词必须从视频转写获得，禁止自己编造或漏掉
- 视频里说什么，台词就写什么
- 来源仅限于百度网盘，不去B站/YouTube等平台搜索
