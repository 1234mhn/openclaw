# 2026小红书算法详解

## 一、流量分发链路

```
发布笔记 → 审核收录 → 打标签匹配 → 冷启动池(0~200)
    → CES评分 → 初级池(200~500) → CES评分 → 中级池(500~5000)
    → 人工审核 → 热门池(1w+)
```

## 二、CES评分模型（2026版）

```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

### 权重意义

**关注(×8)：** 平台最看重长期粉丝价值(LTV)。
- 策略：每篇结尾都要暗示"关注我有更多"
- 人设大于单篇爆款

**评论(×4) / 转发(×4)：** 社交传播力。
- 评论区互动越多，算法越认为内容有讨论价值
- 引导评论的性价比远高于引导点赞

**点赞(×1) / 收藏(×1)：** 基础互动信号。
- 收藏=二次曝光，2026年收藏过的笔记会出现在推荐流
- 藏播比>1%算有价值，>3%算优质

## 三、各流量池的核心考核

### 冷启动池（0~200浏览）
- **时间窗口：** 发布后前2小时
- **考核：** 点击率(CTR)
- 推送对象：粉丝+标签匹配兴趣人群
- 如果CTR低→停止分发
- **关键：封面+标题**

### 初级池（200~500浏览）
- **考核：** CES互动分
- 重点看评论、收藏、关注转化
- 你的173播放在此阶段

### 中级池（500~5000浏览）
- **考核：** 完播率+停留时长
- 内容节奏和结构决定能否通过
- 视频前3秒是生命线

### 热门池（1w+浏览）
- 人工审核介入
- 确认无违规后大规模曝光

## 四、2026年两大变化

### 1. 搜索流量占30%
- 用户直接搜索关键词找内容
- 标题/正文前30字/标签必须埋词
- 例：「韩剧 日常口语 自学 跟读 第2集」

### 2. 从"吸引眼球"到"留住时间"
- 完播率权重大幅提升
- 标题党+空洞内容会被降权
- 实用价值>情绪价值>娱乐价值（对学习类账号）

## 五、新人冷启动SOP

1. **养号3-5天**：正常浏览同赛道内容，打标签
2. **黄金2小时**：发布后2小时内引导基础互动
3. **检测收录**：2小时后搜标题看能不能搜到
4. **复盘**：
   - 播<100 → 封面标题问题
   - 播~500 → 互动引导问题
   - 播~2000 → 完播率问题
