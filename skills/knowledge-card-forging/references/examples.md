# 卡片示例集

## 示例卡片：纳瓦尔之盾

```
📇 卡片名称：纳瓦尔之盾
🔖 卡片ID：KC-BOOK-001
📊 等级：L1
💡 核心洞察
社会能培训你的技能，社会就能培训别人替代你。只有你自己学到的独特组合才是护城河。

📐 行动指南
1. 每周问自己一次："这个技能有几个人能替代我？"
2. 不要只学"大家都在学"的东西，要找自己的组合技能
3. 公开输出你组合出来的独特理解，形成作品资产

🟢 绿灯（适用场景）
· 决定报什么课程/培训班之前
· 考虑换工作/创业方向时
· 评估当前职业竞争力

🔴 红灯（不适用场景）
· 纯粹为了兴趣/消遣的学习，不需要考虑替代性
· 基础义务教育阶段

🔗 关联卡片
· [补充] KC-BOOK-002：个人即公司-IPO学习模式
```

## 四铁律检查清单

锻造完卡片后逐条问用户：

- [ ] **原子性** — 这张卡只讲了一件事吗？有没有混进来第二个知识点？
- [ ] **可行动** — 读完这张卡，读者能立刻做一件事吗？行动指南足够具体吗？
- [ ] **可证伪** — 红绿灯齐全吗？红灯有没有写清楚？（不能空着）
- [ ] **诚实性** — 绿灯场景是你真实经历过的吗？还是纸上谈兵？
