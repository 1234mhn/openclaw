# 🎸 吉他每日练习计划

**🏷️ 主题：** 吉他弹唱进阶 + 指弹入门

## 📋 总纲

- **每天20分钟**，不多不少，关键是坚持

## 📅 周一/三/五 · 弹唱强化日

### 任务① C大调音阶（5min）
```
目的：熟悉指板、练手指独立性、为即兴打基础

C大调指型图（从6弦到1弦）：
  6弦 3品(G) 5品(A) → 5弦 3品(C) 5品(D) → 4弦 2品(E) 3品(F) 5品(G)
  3弦 2品(A) …

简易版（C和弦位置，不用换手型）：
  e|---0-1-3---   Mi Fa Sol
  B|---0-1-3---   Si Do Re
  G|---0-2-----   Sol La
  D|---0-2-3---   Re Mi Fa
  A|---0-2-3---   La Si Do
  E|---0-1-3---   Mi Fa Sol

速度：节拍器 60bpm → 慢慢提速
要求：每个音清晰均匀，一周内不看指板也能弹
```

### 任务② 和弦转换特训（5min，一/三/五）或 指法+Travis（二/四）
```
选两个你切换最慢的和弦，比如：
  → F → G 或 Bm → D 或 C → Am → F → G（经典四和弦）
方法：设60秒倒计时，看能切换多少次
目标：第一天 X次 → 一周后 X+20次
记录：每次练完记次数，看进步
```

### 任务③ 节奏型练习（5min）
```
本周主练4种节奏型，每天选一种反复练：

① 下 下上 上下上   （经典扫弦）
② 下 空空 上       （切音/制音）
③ 下 上 下 上      （八拍，配合切音）
④ 附点节奏：下  下上  上下上

要求：右手放松，节奏稳定，配合节拍器
```

### 任务④ 歌曲弹唱（5min）
```
选一首歌的副歌部分，反复练到不用想和弦
本周推荐：晴天（周杰伦）/ 那些年（胡夏）
重点：先弹熟，再加唱
```

## 📅 周二/四 · 指弹入门日

### 任务① C大调音阶（5min）
```
从6弦3品（G）到1弦8品（C），上行+下行
指法：1-2-3-4 每品一个手指
速度：节拍器 60-70bpm
目标：一周内练到不看指板也能弹
```

### 任务② 指弹基础指法（5min）
```
右手标准指法分配：
  p（拇指）→ 6/5/4弦
  i（食指）→ 3弦
  m（中指）→ 2弦
  a（无名指）→ 1弦

练习动作：
① p→i→m→a 交替（每根弦拨4次）
② p→i→m→a 爬弦（654321弦依次）
③ Am和弦分解：p(5)→i(3)→m(2)→a(1)
```

### 任务③ Travis Picking（5min）
```
经典指弹节奏型：
  拍号：4/4拍
  拇指：低音6/5/4弦交替（每拍一个）
  食指中指：在高音弦上同步扫动

练习：
  ① Am和弦上练：p→i→m→a 循环
  ② 加入和弦转换：Am→C→G→Em
  ③ 保持节奏稳定，不要忽快忽慢
```

### 任务④ 简单旋律（5min）
```
本周练：《小星星》或《生日快乐》
要求：
① 先练右手旋律（单音）
② 再加入左手和弦伴奏
③ 慢速→中速→原速
```

## 📅 周六/日 · 自由弹唱日

### 任务① 热身（5min）
```
① 爬格子 2min
② C大调音阶 1min  
③ 和弦转换 2min（选自己最弱的两个和弦）
```

### 任务② 录制弹唱/指弹（10min）
```
① 选一首这周练的歌
② 录3遍
③ 选最好的一遍发给我或存着

注意：不要追求完美，录3遍选最好的就行
```

### 任务③ 复习本周（5min）
```
① 翻看这周的笔记/练习记录
② 找出自己进步的地方（比如和弦转换快了5次）
③ 找出下周需要加强的地方
```
