# 养花经验

**📂 分类：** 知识库/基础知识/通用技能/园艺

---

## 📖 通用养护知识

### 一、土壤酸碱度
- **蓝莓/茉莉/三角梅：** 三者均喜微酸性土，调酸、配土、养护方法可互通
- **三角梅：** 喜欢微酸性土壤（pH 5.5-6.5）；偏碱会黄叶、徒长、不开花

### 二、养护物料明细

#### 1. 发酵淘米水
- **🧪 成分：** 大量氮磷钾 + 多种有机酸、维生素、微量元素
- **🎯 适用：** 蓝莓、三角梅、茉莉等所有喜酸植物
- **📋 用法：** 每月浇灌1次
- **💡 作用：** 调酸 + 补充养分，促花、保果

#### 2. 鸡蛋壳
- **🧪 成分：** 碳酸钙（主）+ 磷、镁、钾等微量元素
- **📋 处理：** 洗净→彻底晒干→碾碎→埋于花盆边缘
- **💡 作用：** 温和补磷、补钙，促花促果，无烧根风险

#### 3. 过期啤酒
- **🧪 成分：** 氨基酸、糖类、多种微量元素 + 弱酸性物质
- **📋 用法：** 必须兑水稀释后使用
- **💡 作用：** 浇灌→补营养+调酸；喷施/擦拭→清洁叶面、提亮叶色、叶面肥

### 三、通用养护准则

1. 🌱 **土壤：** 统一微酸性土，发酵淘米水+稀释啤酒维持酸度
2. 💧 **浇水：** 晾晒后自来水；清晨/傍晚浇；表土干2厘米再浇透，禁托盘积水
3. ☀️ **光照：** 充足光照，嫩叶/嫩枝避开正午强光
4. ✂️ **修剪：** 定期清老叶、病叶、剪徒长枝，保证通风透光
5. 🥚 **补肥：** 晒干碾碎鸡蛋壳埋盆边，天然补磷，适配开花、结果需求

## 🫐 蓝莓养护

### 一、枝条识别 → 修剪 → 叶片问题 → 日常管护

#### 1. 枝条识别
- **修剪位置：** 叶柄基部修剪，切口平整不留残桩，减少养分消耗
- **枝条区分：** 长条木质化枝条 = 主枝/主枝延长枝；主枝侧边长出的细弱枝条 = 结果分支（挂有幼果）
- **掐顶促枝：** 主枝长到理想高度可掐顶，促分枝，增加来年结果量；顶端徒长嫩梢建议剪掉

#### 2. 修剪实操
- 剪掉主枝顶端徒长梢 → 养分集中供给结果分支，促进幼果生长
- 全部清理老叶 → 提升通风、透光性，减少病害，降低养分消耗

#### 3. 新叶异常处理
- **症状：** 新叶出现红褐斑驳
- **原因：** ①正午强光轻微灼伤新叶；②坐果期养分优先供给果实，新叶养分不足
- **处理：** 不用摘除，自然恢复；摆放改为上午见光、下午遮阴，避开正午暴晒

#### 4. 叶面喷水
- **可以喷：** 细水雾，冲浮尘、增湿度
- **禁忌：** 叶腋、叶心不能积水；喷后放通风处彻底晾干，防止闷湿烂叶

#### 5. 修剪后配套实操

| 项目 | 操作 |
|------|------|
| 🌱 松土 | 盆边浅松，深度1厘米，避开细根、幼果 |
| 💧 浇水 | 盆土表层向下2厘米干透再浇透，花盆托盘严禁积水 |
| ☀️ 光照 | 每日4-6小时光照，嫩叶避开正午强光 |
| 🥚 补肥 | 鸡蛋壳处理后埋盆边，助力果实膨大、转色、增甜 |

### 二、黄叶 / 焦叶 / 掉果原因与方案

#### 1. 叶片问题核心原因
**① 晒伤（主因）**
- 蓝莓叶片薄，挂果期养分优先供果实，抗晒能力极差
- 正午暴晒会破坏叶绿素、烤伤叶片，导致发黄、焦褐、掉果
- **处理：** 挂果期避开正午暴晒，只给上午柔和光照

**② 土壤碱化（自来水导致）**
- 蓝莓专用土+珍珠岩+椰糠的配土没问题
- 但自来水会让土壤慢慢变碱
- 蓝莓喜酸性土（pH 4.5-5.5），缺酸就吸收不了铁，持续黄叶

#### 2. 调酸方案

| 方式 | 方法 | 用途 |
|------|------|------|
| 🌲 长效调酸 | 腐熟松树皮/松针铺面或混土 | 长期维持酸性环境 |
| ⚡ 快速调酸 | 硫酸亚铁兑水灌根 | 黄叶急救 |
| ⚠️ 禁忌 | 新鲜松针、鸡蛋壳不能直接埋 | 会烧根、招虫 |

#### 3. 挂果期养护要点
- ☀️ 光照：避开正午暴晒，只给上午柔和光照
- 💧 浇水：控水不积水，避免烂根
- 🪴 土壤：维持酸性，定期检查叶片颜色判断酸度

## 🌺 三角梅养护

### 一、土壤偏好
- 喜欢微酸性土壤（pH 5.5-6.5）；偏碱会黄叶、徒长、不开花
- 定期浇发酵淘米水调酸

### 二、日常养护要点

| 项目 | 操作 |
|------|------|
| 🌱 土壤 | 维持微酸性，定期浇发酵淘米水调酸 |
| ✂️ 修剪 | 及时剪徒长枝，控株型、促分枝、多开花 |
| 💧 浇水 | 见干见湿，不干不浇，盆内+托盘严禁积水 |
| ☀️ 环境 | 保证充足光照、环境通风 |

| ☀️ 光照 | 充足光照，嫩叶/嫩枝避开正午强光 |
| ✂️ 修剪 | 定期清老叶、病叶、剪徒长枝，保证通风透光 |

## 🍓 草莓养护

### 一、苗期养护（出苗8棵以上）

| 要点 | 做法 |
|------|------|
| ☀️ 光照 | 散光养护，别暴晒 |
| 💧 补水 | 轻喷水保湿，别大水浇灌 |
| 🚫 位置 | 别总挪动，让小苗适应环境 |

### 二、想要结更大果
- **疏苗：** 后期适当疏苗，给健壮苗留空间
- **疏花疏果：** 摘掉过多花/小果，集中养分
- **补肥：** 膨果期补磷钾肥
- **光照：** 保证温和光照，避免暴晒

### 三、补光方案
- **无自然光时：** 用全光谱植物补光灯模拟阳光
- **优点：** 安全不灼伤植株，可精准控制光照时长

## 🧪 三大元素缺乏症状速查

### 1. 缺氮（长叶子、长身体的）

**症状：** 全株发黄、长得矮、叶子小、老叶先黄
- 叶子浅绿→发黄，从下面老叶子开始黄
- 植株瘦小、不长高、分枝少
- 长得弱不拉几，像没吃饭

> 📝 口诀：**缺氮黄叶子，植株长得细**

---

### 2. 缺磷（长根、开花、结果的）

**症状：** 叶子发紫、长得慢、不开花、根系弱
- 叶子暗绿、发紫、发红（最典型）
- 长得极慢，矮壮不长
- 开花少、结果小、根系不发达
- 耐寒差，容易冻伤

> 📝 口诀：**缺磷叶发紫，开花不给力**

---

### 3. 缺钾（壮杆、抗病、保果实的）

**症状：** 叶边焦枯、像烧焦、容易倒伏、烂根烂果
- 叶子边缘干枯、焦黄卷曲，中间还是绿的
- 茎秆细弱，一吹就倒
- 果子小、口感差、容易生病

> 📝 口诀：**缺钾叶焦边，杆子软无力**

## ❓ 遇到的问题总汇

### 蓝莓
- 蓝莓新叶出现红褐斑驳，误以为是病害
- 蓝莓叶片发黄焦褐，之前误以为只是缺肥
- 挂果期叶片对光照敏感，之前忽略了遮阴
- 原来各类喜酸植物养护逻辑可互通，之前未建立通用框架

### 草莓
- 草莓苗多但不一定每棵都能长大果
- 补光灯能否替代自然光——全光谱灯可行

### 三角梅
- 三角梅与蓝莓/茉莉同为喜酸植物，养护逻辑可互通
- 偏碱土壤会导致不开花，之前未重视酸度管理

### 茉莉
- 茉莉养护可直接复用蓝莓/三角梅的通用框架

## 🔧 解决过程

1. 观察新叶异常 → 分析排除病害 → 确定是光照过强 + 坐果期养分分配
2. 对比蓝莓、三角梅、茉莉的习性 → 发现土壤酸碱度是共通关键
3. 整理三种物料的成分和用法 → 建立通用养护知识框架
4. 观察草莓出苗数量 → 需要疏苗 → 分析膨果因素 → 确认疏花疏果+磷钾肥+补光方案

## 🚀 应用落地

- 蓝莓修剪后配套操作（松土/浇水/光照/补肥）已落地执行
- 三角梅日常养护按通用准则操作
- 茉莉养护可直接复用此框架
- 挂果期蓝莓放到上午见光、下午遮阴的位置
- 硫酸亚铁灌根应急调酸
- 后续补腐熟松树皮作长效调酸
- 草莓保持散光养护，进入膨果期后补磷钾肥
- 后续可补充：除虫/换盆/越冬等专项知识
