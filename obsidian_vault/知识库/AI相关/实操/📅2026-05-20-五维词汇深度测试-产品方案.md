# 五维词汇深度测试 — 产品方案

**📅 时间：** 2026-05-20
**🏷️ 主题：** 五维词汇深度测试网站方案
**📂 分类：** 知识库/AI相关/实操

---

## 📖 大体内容

### 产品定位
面向初高中学生的**词汇深度诊断工具**，从五个维度测量一个单词的掌握程度，而不是简单地测"认不认识"。

### 词汇定级（5级）
| 级别 | 词量 | 对标 |
|------|------|------|
| 🌱 入门级 | 0-800 | 小学毕业 |
| 🌿 基础级 | 800-1600 | 初一~初二 |
| 🌳 进阶级 | 1600-2500 | 初三~高一 |
| 🌲 高考级 | 2500-3500 | 高二~高三 |
| 🏔️ 拔高级 | 3500-5000+ | 四级/拔尖 |

### 测试结构（两层法）

**第一层：自适应抽题测词汇量**
- 从目标级别开始，每5个词调整一次难度
- 认识多→上调，认识少→下调
- 约测完自动估算词汇量
- 误差控制在 ±200 词

**第二层：深度诊断（10个词×5步）**
每个词走5步，从浅到深：

| 步 | 维度 | 题型 | 考什么 |
|----|------|------|--------|
| 1 | 🔊 读音 | 4个音标选正确项 | 会不会读 |
| 2 | 🏷️ 词性 | 可多选含陷阱 | 知道几种词性 |
| 3 | 🔧 短语搭配 | 选正确搭配 | 常和谁搭 |
| 4 | ✍️ 语境辨句 | 选正确句子 | 看得懂吗 |
| 5 | 🖊️ 主动造句 | 打字造句 | 能不能自己用出来 |

### 报告内容
- 词汇量估算 + 对应级别
- 五维雷达图（Canvas绘制）
- 各维度掌握度百分比
- 诊断结论（最弱维度+学习建议）
- 错词清单（哪些词、哪个维度错了）
- 📸 一键保存报告图（html2canvas导出PNG）

### 技术实现
- 纯前端：HTML + CSS + JS 单页应用
- 不依赖后端，localStorage 存数据
- 手机端适配良好
- 词库预制：首批66个词（全5级）

### 词库数据格式
```
每个词完整字段：
- word: 单词
- p: 音标
- pt: 干扰音标
- pos: 正确词性数组
- ptrap: 陷阱词性数组
- col: {c:正确搭配, t:陷阱搭配}
- sen: {c:正确句子, t:陷阱句子}
- hint: 造句提示
```

## ❓ 遇到的问题 / 不理解的点

- 造句判分如何更准确？当前方案：简单判断（包含该词+有动词+长度≥6）
- 后续可以接入灵眸AI API 做智能造句判分

## 🔧 解决过程

1. 用户提出想做单词测试网站
2. 讨论后确定核心方向：词汇量+读音+词性+用法+会不会用
3. 参考 Vocabulary Levels Test (Nation, 1983) 方法论确定自适应抽样
4. 最终确定为5级词库 + 两层筛选 + 五维深度测试
5. 前端代码已生成并调试完成
