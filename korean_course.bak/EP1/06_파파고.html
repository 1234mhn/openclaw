단어문법

다 알아들었어요?

→ 都听懂了？

동사알아듣다
→ 听懂、领会
알다(知道) + 아 + 듣다(听) → 听明白。ㄷ→ㄹ 不规则: 알아들으면, 알아들었어요
💬 한국어 잘 알아들어요. → 我能听懂韩语。
대조듣다 vs 알아듣다 vs 들리다
▸ 듣다 = 听 (主动) ／ 들리다 = 被听到 (被动)
▸ 알아듣다 = 听明白 (理解到了)
🎵 音变说明
› 알아들었어요 → [아라드러써요] (连音化)

그냥 들린 겁니다, 몰래 들은 거죠

→ 只是听到了 / 是偷听的吧

동사들리다
→ 被听到、能听见（피동사）
듣다 → 들리다。"我听到了"= 들렸어요，"自动传到耳朵"
💬 노래가 잘 들려요. → 歌很清楚。
부사몰래
→ 偷偷地
"모르게" 不让别人知道。반대 = 대놓고(公然地)
💬 몰래 들어왔어. → 偷偷进来的。
문법~ㄴ/은 거죠
→ 是……的吧（推测+确认）
~ㄴ 것이지요 缩略，比 ~았/었지요 多了"行为本质"的味道
🎵 音变说明
› 들린 겁니다 → [들린검니다] (鼻音化)

중국 사람인 척하면서

→ 装作中国人

문법~인 척하다 / ~는 척하다
→ 装作……、假装……
名词 + 인 척 ／ 动词 + 는 척 ／ 形容词 + ~(으)ㄴ 척
💬 자는 척했어요. → 装睡了。
💬 모르는 척하지 마. → 别装不知道。
💬 부자인 척했어. → 装富。
문법~(으)면서
→ 一边……一边……
同时进行的动作。批评语气："明明……还(同时)……"
💬 노래 들으면서 공부해요. → 边听歌边学习。

아니, 본 척한 게 아니고 본 거고

→ 不是装作看，是真的看

문법~ㄴ 게 아니고 ~ㄴ 거고
→ 不是……而是……
"本척한 게 아니고 본 거" = 不是装看，是真看。强调否定+修正。
💬 화난 게 아니고 피곤한 거예요. → 不是生气，是累了。
감탄사아니
→ 不(是)、欸（口语反驳/惊讶）
💬 아니, 그게 아니라… → 不是，我说的不是那个…

말이 되는 소리를 해요

→ 说点像样的话好不好

관용구말이 되다 / 안 되다
→ 像话／不像话
"말이 안 돼" = 不像话/没道理 ／ "말이 되는 소리" = 有道理的话
💬 그게 말이 돼? → 那像话吗？
💬 말도 안 돼! → 太离谱了！
관용관련 표현
▸ 말도 안 되는 소리 = 不像话的话
▸ 말이 되는 소리를 하다 = 说点像话的
▸ 말 같지도 않다 = 简直不像人话

자기가 무슨 파파고인가

→ 你以为自己是Papago翻译机吗

고유명사파파고 (Papago)
→ 韩国 NAVER 公司开发的翻译App，韩国人最常用
语源 = Papagaio（葡萄牙语"鹦鹉"）。剧中讽刺男主"什么语言都翻"。
호칭자기
→ ① 自己 ② 你（口语年长者→年轻人，或情侣间）
这里"자기" 表"你"，带轻微讽刺/不屑（무희 比 호진 大）
💬 자기 일이나 잘하세요. → 管好你自己的事。
문법무슨 ~인가
→ 难道是……吗（讽刺反问）
💬 자기가 무슨 의사야? → 你以为你是医生啊？

아이가 알레르기 때문에 호흡 곤란으로 쓰러졌다고

→ 孩子因过敏呼吸困难晕倒了

명사알레르기
→ 过敏 (Allergy)
💬 견과 알레르기가 있어요. → 我对坚果过敏。
명사호흡 곤란
→ 呼吸困难
汉字 呼吸困难 = 호흡(呼吸) + 곤란(困难)
동사쓰러지다
→ 倒下、晕倒
💬 더위에 쓰러졌어요. → 中暑晕倒了。
문법~ 때문에
→ 因为（名词原因）
名词 + 때문에 ／ 形容词/动词 + 기 때문에
💬 비 때문에 못 갔어요. → 因为下雨没去成。
문법~다고 (전달)
→ ……(让对方传达)
陈述间接引语 = ~다고 하다。这里 호진 让店员"告知救护车"，用 ~다고 알려 주세요
📌 单词·语法 总览

📍 单词

▸ 알아듣다 — 听懂

▸ 들리다 — 被听到

▸ 몰래 — 偷偷

▸ 알레르기 — 过敏

▸ 쓰러지다 — 晕倒

▸ 호흡 곤란 — 呼吸困难

▸ 파파고 — Papago翻译App

📍 核心语法

▸ ~인/는 척하다 — 假装……

💬 안 듣는 척했어요. → 装作没听。

▸ ~(으)면서 — 一边……一边……

💬 웃으면서 말했어. → 笑着说的。

▸ ~ 때문에 — 因为……

💬 너 때문에 늦었어. → 因为你迟到了。

▸ ~ㄴ 게 아니고 ~ㄴ 거 — 不是……而是……

💬 화난 게 아니고 슬픈 거야. → 不是生气是难过。

▸ 말이 되다 — 像话/合理

💬 그게 말이 안 돼. → 那不像话。

💡 小贴士

① 파파고(Papago) 是韩国 NAVER 出的翻译App，类似中国的"百度翻译"。剧名"翻译爱情"中处处有它。

② 들리다 vs 듣다：들리다 强调"声音传来"(被动)，듣다 是"主动听"。"몰래 듣다"才是偷听。

③ ~인 척하다 是高频生活用语，年轻人爱用 "~척" 单独使用："척하지 마!"(别装!)

④ "자기" 用法很微妙：情侣间是"亲爱的"，对下属/晚辈则是"你"，必须看场合判断。
