부사예전에
→ 以前
💬 예전에 한국에 살았어요. → 以前住过韩国。
동사찍다
→ 拍（照片/电视/电影）
💬 사진 찍어 주세요. → 请帮我拍照。
💬 영화를 찍고 있어요. → 在拍电影。
문법~(으)ㄴ 적이 있다/없다
→ 有/没有……过的经验
动词 ~(으)ㄴ + 적(经验) + 이 있다/없다
💬 한국에 가 본 적이 있어요. → 去过韩国。
💬 김치를 먹어 본 적이 없어요. → 没吃过泡菜。
🎵 音变说明
› 찍은 적이 → [찌근저기] (连音化)

아주 신인 때, NG를 너무 많이 내서

→ 还是新人的时候，NG出得太多

명사신인
→ 新人
汉字词 新人。반대말 = 베테랑(老手)
💬 신인 가수예요. → 是新人歌手。
표현NG를 내다
→ 出NG（演员表演失误）
"내다" = 引发、发出。NG 发音 [엔지]
부사너무
→ 太、过于
现代口语已经中性化，可表示"非常"。书面建议用 매우/아주
🎵 音变说明
› 많이 내서 → [마니내서] (ㅎ脱落)

그때 대사는 아주 또렷하게 기억하고 있어요

→ 那时的台词记得清清楚楚

명사대사
→ 台词
💬 대사를 외워요. → 背台词。
부사또렷하게
→ 清晰地、清楚地
또렷하다(清晰的) → 또렷하게(副词形)
💬 또렷하게 발음해 주세요. → 请发音清楚一点。
문법~고 있다
→ 正在……、……着
动作进行 / 状态持续。기억하고 있다 = 记着。
💬 지금 밥을 먹고 있어요. → 现在正在吃饭。
🎵 音变说明
› 또렷하게 → [또려타게] (激音化: ㄷ+ㅎ→ㅌ)

아, 진짜? 얼마나 못했길래?

→ 真的吗？演得多差啊？

의문사얼마나
→ 多么、多少
💬 얼마나 기다렸어요? → 等了多久？
문법~길래
→ 因为/到底有多……（带好奇/反问语气）
表"究竟……才会这样"，常与 얼마나 / 어떻게 搭配
💬 얼마나 맛있길래? → 到底有多好吃？
💬 얼마나 예쁘길래 다들 난리야? → 到底多漂亮，大家这么激动？
🎵 音变说明
› 못했길래 → [모탣낄래] (激音化+紧音化)

궁금하네, 유튜브에 있나?

→ 真好奇，YouTube上有吗？

형용사궁금하다
→ 好奇、想知道
💬 결과가 궁금해요. → 好奇结果。
문법~네
→ ……啊（感叹/发现）
感叹刚发现的事或自言自语。주말이네! = (原来)是周末啊！
문법~나?
→ ……吗？（自问/委婉疑问）
用于自言自语或非正式提问。比 ~아요? 更轻松。
💬 비 오나? → 下雨了吗？
🎵 音变说明
› 유튜브 → [유튜브] (외래어 그대로)

아! 흑역사!

→ 啊！黑历史！

명사흑역사
→ 黑历史（不愿回顾的过去）
汉字词 黑历史 = 검을 흑(黑) + 역사(历史)。源于日语动画《高达》系列。
💬 그건 내 흑역사야. → 那是我的黑历史。
💬 흑역사를 들켰어. → 黑历史被发现了。
회화剧中梗
→ 日本人 히로 不会发 흑(heuk)，发成 허격사/하격사，最后 무희 听懂猜中是 흑역사。
韩日跨语言喜剧梗：ㅎ + ㅡ + ㄱ 这种闭音节对日语母语者来说很难。
🎵 音变说明
› 흑역사 → [흥녁싸] (鼻音化+ㄴ添加+紧音化)
› ㄱ (终声) + ㅇ → ㅇ变ㄴ → 흥
› ㅕ + ㄱ(쌍받침效应) → 녁

두 사람 촬영 초반에 비해 엄청 많이 친해졌네요

→ 跟拍摄初期相比，两人变得很熟了呢

명사초반
→ 初期、前期
초반(初盘) ↔ 중반(中盘) ↔ 후반(后盘)
문법~에 비해
→ 与……相比
~에 비해서/비하여 的口语形。汉字词 비하다 = 比之
💬 가격에 비해 좋아요. → 跟价格比起来挺好。
문법~아/어지다
→ 变得……（状态变化）
친하다(熟) → 친해지다(变熟) ／ 좋다 → 좋아지다 / 예쁘다 → 예뻐지다
💬 날씨가 따뜻해졌어요. → 天气变暖了。
🎵 音变说明
› 친해졌네요 → [치내전네요] (鼻音化)

이제 베프네, 베프야

→ 现在是好朋友了，铁哥们

유행어베프
→ Best Friend 的缩略，"最好的朋友"
베스트 프렌드 → 베+프 (年轻人常用语)
💬 우리는 베프야! → 我们是好基友！
💬 베프랑 여행 가요. → 跟好朋友去旅行。
유행어 모음常见韩式英缩略
▸ 인싸 = INSIDER 受欢迎的人 ／ 아싸 = OUTSIDER 边缘人
▸ 셀카 = SELF CAMERA 自拍 ／ 솔까 = 솔직히 까놓고 말해서
▸ 갑분싸 = 갑자기 분위기 싸해짐 (气氛突然冷)
🎵 音变说明
› 베프네 → [베프네] (语气词 ~네 表感叹)
📌 单词·语法 总览

📍 单词

▸ 신인 — 新人

▸ 대사 — 台词

▸ 또렷하다 — 清晰

▸ 흑역사 — 黑历史

▸ 친하다 — 关系好

▸ 베프 — 好朋友/铁哥们

▸ 초반 — 初期

📍 核心语法

▸ ~(으)ㄴ 적이 있다 — 有过……的经验

💬 일본에 가 본 적이 있어요. → 去过日本。

▸ ~길래 — 到底有多……

💬 얼마나 비싸길래 그래요? → 到底多贵啊？

▸ ~고 있다 — 正在……

💬 책을 읽고 있어요. → 正在读书。

▸ ~에 비해 — 相比……

💬 작년에 비해 추워요. → 比去年冷。

▸ ~아/어지다 — 变得……

💬 한국어가 늘어졌어요. → 韩语变好了。

💡 小贴士

① 흑역사 是来自日本《高达》动画的"黑历史"，韩国年轻人也常用，男女主因这个词产生默契。

② 韩剧里 NG 念 [엔지]，엔진 是发动机要分清。

③ 베프(best friend)、인싸/아싸、갑분싸 都是高频年轻人用语。

④ 表达"经验"用 ~(으)ㄴ 적이 있다，回答时常跟 ~본 적이 있다(尝试过)。
