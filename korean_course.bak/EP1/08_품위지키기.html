문법

눈치챘으니까 아무 말 마요

→ 既然察觉到了就什么都别说

명사눈치
→ 眼色、察言观色的能力
韩国文化超核心词。눈치(=eye-vibe)，韩国人重视看场合"读空气"。
💬 눈치 빠르다 = 反应快/会看眼色
💬 눈치 없다 = 不会看眼色
💬 눈치를 보다 = 看人脸色
동사눈치채다
→ 察觉、识破
눈치(察觉) + 채다(夺取/抓住)
💬 눈치챘어요? → 察觉到了？
문법~지 마요 / 마세요
→ 别做……
"아무 말 마" = 아무 말 하지 마 的口语缩略

근데 어떡하지? 나 눈물 나올 거 같아

→ 怎么办，我快哭出来了

접속사근데
→ 但是、不过（口语）
그런데 的口语缩略
명사눈물
→ 眼泪
눈(眼) + 물(水) → 字面"眼水"
💬 눈물이 나요. → 流眼泪。
문법~ㄹ 거 같아
→ 好像要……了
动词 ~(으)ㄹ 것 같다 = 看起来要发生
💬 비 올 거 같아. → 好像要下雨。

저 여자 앞에서 울면 나 진짜 죽어 버릴 거야

→ 在那女人面前哭我会真的去死的

문법~아/어 버리다
→ ……掉、(完全/彻底地)做完
버리다(扔掉) 后缀化 → 表彻底完成、可惜或放弃情绪
💬 다 먹어 버렸어. → 全吃完了。
💬 잊어버렸어. → 忘掉了。
관용죽어 버릴 거야
→ "去死了""丢死人了"（夸张表达）
日常夸张说法，不是真要死。比如 "쪽팔려 죽겠어" = 丢死人了
문법~(으)면
→ 如果……
💬 비 오면 안 갈 거야. → 下雨就不去。

그럼 이렇게 있어요. 내가 수습해 볼게요

→ 那你这样待着。我来收拾

동사수습하다
→ 收拾、应对（局面）
汉字 收拾。常见: 사태를 수습하다(应对事态)
💬 일을 수습해야 돼. → 必须收拾局面。
문법~아/어 보다
→ 试着……
💬 한번 먹어 봐요. → 试着尝一下。
문법~ㄹ게요
→ 我会……（意志/承诺）
用于对方在场时表达"我来做"。比 ~겠어요 更口语温和。
💬 내가 갈게. → 我会去。

최대한 품위 있게

→ 尽量有尊严地

부사최대한
→ 尽量、最大限度地
💬 최대한 빨리 가요. → 尽快走。
표현품위 있게
→ 有品位地、有尊严地
품위(=品位) + 있다(有) + 게(地)
문법~게 (副词化)
→ ……地（修饰动词）
形容词 + 게 → 副词。예쁘게(漂亮地) ／ 빠르게(快地) ／ 행복하게(幸福地)

오해하셨나 본데, 우리는 함께 여행 중이고

→ 您是不是误会了，我们正在一起旅行

동사오해하다
→ 误解、误会
💬 오해하지 마세요. → 请别误会。
문법~나 보다
→ 看来……、似乎……（推测）
~았/었나 봐 ／ ~나 봐. 根据外部线索判断他人状态。
💬 비가 오나 봐요. → 看来下雨了。
문법~중이다
→ 正在……
名词 + 중(中)。회의 중(开会中) / 식사 중(用餐中)
💬 운전 중이에요. → 正在开车。
🎵 音变说明
› 오해하셨나 → [오해하션나] (鼻音化)

이 사람 마음을 다치게 할 말은 전하지 않겠습니다

→ 会伤害这个人内心的话，我不会传达

명사마음
→ 心、心思、心情
韩语万能词。마음을 다치다 = 心受伤 ／ 마음에 들다 = 中意 ／ 마음먹다 = 下决心
동사다치다 / 다치게 하다
→ 受伤 / 使……受伤
다치다(自动) → 다치게 하다(使动)
문법~게 하다
→ 使……做……（使役）
动词 + 게 하다 = 让某人做某事
💬 울게 했어요. → 让(他)哭了。
문법~지 않겠습니다
→ 不会……（强烈拒绝/承诺）
~지 않다(不) + ~겠습니다(意志正式)

괜찮아졌으면 이만 가죠

→ 既然好些了就走吧

표현괜찮아지다
→ 变好、好起来
괜찮다(没事) + 아지다(变化) = 变得没事/好转
부사이만
→ 到此、就这样
💬 이만 마치겠습니다. → 到此结束。
💬 이만 가요. → 我先走了。
문법~죠 (제안)
→ ……吧（提议）
~지요 缩略，柔性提议。"가죠" = 走吧

📍 核心语法

▸ ~아/어 버리다 — 彻底……掉

💬 잊어 버려. → 忘掉吧。

▸ ~ㄹ게요 — 我会……

💬 도와줄게요. → 我会帮忙。

▸ ~게 하다 — 让某人做……

💬 친구를 웃게 했어. → 让朋友笑了。

▸ ~나 보다 — 似乎……

💬 자나 봐요. → 看来在睡。

▸ ~중이다 — 正在……

💬 회의 중이에요. → 在开会。

💡 小贴士

① 눈치 是韩国文化关键词。"눈치를 잘 보다"(会看眼色) 是韩国职场必备技能。

② ~아/어 버리다 表达"彻底"，常带情绪色彩(惋惜/解脱)。"잊어버려" = 忘了吧(放下吧)。

③ 이만 가요 是离场客气话，比 그냥 가요 更有礼。商务/正式场合常用。

④ 호진 这一段救场是男主"作为通译师"的最佳定义：选择什么不译，本身就是一种翻译。
