## 📖 如何使用本手册

本手册是 **EP1视频配套的深度自学资料**，每一节对应一个已发布的视频。

### 建议学习流程

```
① 刷小红书 → 看我的视频（先感受语境和发音）
② 打开本手册 → 看「场景铺垫」了解背景
③ 逐句学习 → 左边看台词+翻译+单词拆解
④ 右边记笔记 → 写自己的理解/疑问/造句
⑤ 回看视频 → 带着理解再看一遍，加深记忆
```

每个场景包括：


## 🎬 拍摄现场·抱怨与指令

> 🎭 **场景背景：** 车武熙被通知来拍摄现场做翻译，本以为是什么高端场合，盛装打扮穿着高跟鞋来了。结果发现拍摄地点在荒郊野外、满地泥泞……
> **💡 这节学什么：** 抱怨、后悔、指令表达。韩语里「早知道就…」「本该…才对」「请把…拿正」这些日常高频表达全在这段里。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 이런 데인 줄 알았으면 운동화를 신으라고 했어야지

**中文：** → 早知道是这种地方，就该让我穿运动鞋

- **📍 단어이런 데**
  → 这种地方（데 = 곳，口语化）
  💬 이런 데 자주 와요? → 你经常来这种地方吗？
  💬 이런 데 처음이에요. → 这种地方我是第一次来。
- **📍 단어운동화 / 신다**
  → 运动鞋 / 穿（鞋袜类）
  신다 = 穿鞋袜 / 입다 = 穿衣 / 쓰다 = 戴帽戴眼镜 / 끼다 = 戴戒指手套 / 차다 = 戴手表
  💬 운동화 신는 게 편해요. → 穿运动鞋舒服。
- **📌 문법~ㄴ/은 줄 알았으면**
  → 早知道是……的话（事后假设，常带后悔）
  ~ㄴ/은 줄 알다(以为是…) + ~았/었으면(如果…的话)
  💬 비 올 줄 알았으면 우산을 가져왔지. → 早知道下雨就把伞带来了。
  💬 이렇게 어려운 줄 알았으면 안 했어. → 早知道这么难就不做了。
- **📌 문법~(으)라고 하다**
  → 让……做（命令的间接引语）
  신다 → 신으라고 하다(让穿) / 가다 → 가라고 하다(让去)
  💬 엄마가 약을 먹으라고 했어요. → 妈妈让我吃药。
  💬 선생님이 일찍 오라고 했어요. → 老师让我早点来。
- **📌 문법~았/었어야지**
  → 本该……才对（埋怨/后悔语气）
  💬 일찍 일어났어야지! → 就该早点起来啊！
  💬 미리 말했어야지. → 就该提前说嘛。

🎵 ›줄 알았으면→ [주 라라쓰면] (连音化：ㄹ 받침 + ㅇ → 顺移)

🎵 ›신으라고→ [시느라고] (连音化：ㄴ 받침 + ㅇ → 顺移)

🎵 ›했어야지→ [해써야지] (连音化：ㅆ 받침 + ㅇ → 顺移到下一音节)

### 台词2

**韩语：** 아, 잔뜩 차려입으라더니 이게 뭐야

**中文：** → 啊，让我盛装打扮，结果是这种场面？

- **📍 단어잔뜩**
  → 满满地、狠狠地（程度副词）
  💬 잔뜩 먹었어요. → 吃得饱饱的。
  💬 화가 잔뜩 났어요. → 气得不行。
- **📍 단어차려입다**
  → 盛装打扮、穿戴整齐
  차리다(整理/准备) + 입다(穿) → 把衣服讲究地穿好
  💬 오늘 뭐 이렇게 차려입었어? → 今天怎么穿这么正式？
  💬 정장으로 잔뜩 차려입었어요. → 穿了一身西装盛装。
- **📌 문법~(으)라더니**
  → 之前让……，结果却……（前后矛盾/对比）
  ~(으)라고 하더니 的缩略
  💬 일찍 오라더니 본인은 늦었네. → 让我早点来，自己反倒迟到了。
  💬 조용히 하라더니 자기가 더 시끄럽네. → 让别人安静，自己更吵。
- ** 회화이게 뭐야**
  → 这算什么 / 这是什么啊（不满/质疑语气）
  💬 이게 다 뭐야? → 这都是些什么啊？

🎵 ›잔뜩→ [잔뜩] (词中本来就是紧音 ㄸ)

🎵 ›차려입으라더니→ [차려이브라더니] (连音化：ㅂ 받침 + ㅇ → 顺移)

🎵 ›뭐야→ [뭐야] (口语自然连读)

### 台词3

**韩语：** 아, 다리 아파

**中文：** → 啊，腿好疼

- **📍 단어다리**
  → 腿（注意：다리 也有"桥"的意思，靠语境分辨）
  💬 다리가 길어요. → 腿很长。
  💬 다리가 저려요. → 腿麻了。
- **📍 단어아프다**
  → 疼、痛、生病（으脱落不规则）
  아프다 + 아 → 아파（으脱落） / 아프다 + 았어요 → 아팠어요
  💬 머리가 아파요. → 头疼。
  💬 어디가 아프세요? → 您哪里不舒服？
- ** 회화다리 아파**
  → 腿疼（口语省略主格 가，半语反映心情）
  敬语：다리가 아파요 → 半语口语：다리 아파

🎵 ›아파→ 아프다 + 아 = 아파（으脱落不规则，词干末尾 으 + 아/어 时 으 脱落）

### 台词4

**韩语：** 차무희 씨, 오디오 들어갑니다

**中文：** → 车武熙小姐，音频要进了（导播提示）

- **📍 단어~씨**
  → ……先生 / 小姐（成人之间的尊称，接姓名后）
  💬 김민수 씨, 안녕하세요. → 金敏秀先生你好。
- **📍 단어오디오**
  → 音频、音响（外来语 audio）
  💬 오디오 체크할게요. → 我来检查一下音频。
- **📍 단어들어가다**
  → 进入、进去；（节目/录制场景）开始进入某状态
  들다(进) + 가다(去) 复合动词。剧组术语：오디오/카메라/방송 들어갑니다 = 现在进入音频/镜头/直播
  💬 회의실에 들어갔어요. → 进了会议室。
  💬 곧 방송 들어갑니다. → 节目马上开始。
- **📌 문법~ㅂ니다 / 습니다**
  → 最正式的"格式体"陈述句尾
  动词 + ㅂ니다(母音/ㄹ结尾) / 습니다(收音结尾)。现场对讲、播报、商务最常用。
  💬 곧 도착합니다. → 马上到。

🎵 ›들어갑니다→ [드러감니다] (连音 + 鼻音化：ㅂ + ㄴ → ㅁ + ㄴ)

🎵 ›차무희 씨→ [차무히 씨] (注意 ㅎ 在母音间常弱化)

### 台词5

**韩语：** 액션 캠 똑바로 들어 주세요

**中文：** → 请把运动相机拿正

- **📍 단어액션 캠**
  → 运动相机 (action cam，外来语缩写)
  💬 액션 캠으로 찍었어요. → 用运动相机拍的。
- **📍 단어똑바로**
  → 笔直地、端正地、好好地（强调"对/正"）
  💬 똑바로 앉으세요. → 请坐直。
  💬 똑바로 말해. → 给我老实说。
- **📍 단어들다**
  → 拿、举（用手抬起或握住）
  注意区分：들다(拿/举) ≠ 들어가다(进入)。这里是"举/握"。
  💬 가방 좀 들어 주세요. → 请帮我拎一下包。
- **📌 문법~아/어 주세요**
  → 请（帮我）做……（请求 + 受惠）
  动词词干 + 아/어 주세요。比单纯命令更礼貌。
  💬 문을 열어 주세요. → 请帮我开门。
  💬 사진 찍어 주세요. → 请帮我拍张照。

🎵 ›똑바로→ [똑빠로] (紧音化：ㄱ 받침 后的 ㅂ → ㅃ)

🎵 ›들어→ [드러] (连音化：ㄹ 받침 + ㅇ → 顺移)

### 台词6

**韩语：** 됐나요?

**中文：** → 这样行了吗？

- **📍 단어되다 / 됐다**
  → 行、可以、成了。됐다 = 되었다 的口语缩略
  高频万能动词。"행了"、"够了"、"算了"都能用 됐어요
  💬 됐어요, 그만! → 够了，别说了！
  💬 이만하면 됐어요. → 这样就差不多了。
- **📌 문법~나요?**
  → ……吗？（柔和、含蓄的疑问尾，比 ~아요? 更礼貌随和）
  动词/形容词 + 나요?（过去 았/었나요?）。常见于客服、确认。
  💬 시간 있으세요? = 시간 있나요? → 有时间吗？
  💬 다 끝났나요? → 都结束了吗？

🎵 ›됐나요→ [됀나요] (鼻音化：ㅆ 收音遇 ㄴ → ㄴ)

### 台词7

**韩语：** 아, 굽 다 나가겠네

**中文：** → 啊，鞋跟都要废了

- **📍 단어굽**
  → 鞋跟（특히 高跟鞋的跟）
  💬 굽이 너무 높아요. → 鞋跟太高了。
  💬 굽이 부러졌어요. → 鞋跟断了。
- **📍 단어다**
  → 全部、都（程度副词）
  💬 다 먹었어요. → 都吃完了。
- **📍 단어나가다**
  → 出去；这里口语义 = 坏掉、报废、不能用
  "X 나가다" 在口语里常表示 X 失灵/坏了：퓨즈가 나갔어 = 保险丝烧了 / 정신이 나갔어 = 神不守舍了
  💬 노트북이 나갔어요. → 笔记本坏了。
- **📌 문법~겠~**
  → 表示推测、即将（这里 = "快要…了"）
  动词词干 + 겠 + 终结词尾。表猜测/预感。
  💬 비가 오겠네. → 看来要下雨了。
  💬 늦겠어요. → 要迟到了。
- **📌 문법~네**
  → 半语感叹尾（"啊~"、"呢"），表新发现/感叹
  💬 진짜 예쁘네. → 真漂亮啊。

🎵 ›굽 다→ [굽 따] (紧音化：ㅂ 收音 + ㄷ → ㄸ)

🎵 ›나가겠네→ [나가겐네] (鼻音化：ㅆ → ㄴ 在 ㄴ 前)

### 台词8

**韩语：** 저기요, 거기 우리 매니저 있죠?

**中文：** → 喂，那边有我经纪人吧？

- ** 회화저기요**
  → "喂"/"不好意思"（叫人引起注意，店员/陌生人通用）
  💬 저기요, 길 좀 물어볼게요. → 不好意思，问个路。
  💬 저기요, 여기 메뉴판 좀 주세요. → 服务员，给一下菜单。
- **📍 단어거기**
  → 那里（指对方那边/听者一侧）
  여기(这边·我侧) / 거기(那边·你侧) / 저기(那边·远处)
- **📍 단어매니저**
  → 经纪人 (manager，外来语)
  💬 매니저한테 물어볼게요. → 我去问问经纪人。
- **📌 문법~죠? (~지요?)**
  → ……吧？（向对方确认已知信息）
  ~지요? 的口语缩略。下降语调时表"是这样吧"，上升语调时表确认询问。
  💬 여기 있죠? → 在这儿吧？
  💬 김 선생님이시죠? → 您是金老师吧？

🎵 ›있죠→ [읻쬬] (받침 ㅆ → ㄷ + ㅈ 紧音化为 ㅉ)

### 台词9

**韩语：** 네, 누나 말씀하세요

**中文：** → 是，姐你说

- **📍 단어누나**
  → 姐姐（男性对姐姐的称呼）
  男称：형(哥) / 누나(姐) ；女称：오빠(哥) / 언니(姐)。剧中是男经纪人称呼武熙。
  💬 누나, 이거 뭐예요? → 姐，这是什么？
- **📍 단어말씀**
  → 话（말 的尊敬形）
  말씀하다 = 말하다 的尊敬形。"请讲、您说"
  💬 할 말씀 있으세요? → 您有话要说吗？
- **📌 문법~(으)세요**
  → 请……（尊敬命令尾，温和的请求）
  动词词干 + (으)세요。这里 말씀하 + 세요 = 请说。
  💬 들어오세요. → 请进。
  💬 천천히 말씀하세요. → 您慢慢说。
- ** 회화네**
  → 是、好（应答词，万能）

🎵 ›말씀하세요→ [말쓰마세요] (连音化：ㅁ 后 ㅎ 弱化)

### 台词10

**韩语：** 이거 구두 증정이야, 반납이야?

**中文：** → 这双高跟鞋是送的，还是要还的？

- **📍 단어구두**
  → 皮鞋 / 高跟鞋（区别于运动鞋 운동화）
  💬 이 구두 예뻐요. → 这双鞋真好看。
- **📍 단어증정**
  → 赠送（书面词，常用于品牌/活动）
  💬 사은품 증정 이벤트. → 赠品赠送活动。
  💬 이건 증정품이에요. → 这是赠品。
- **📍 단어반납**
  → 归还、退还（图书/借品/试用装常用）
  💬 도서관에서 책을 반납했어요. → 在图书馆还书了。
  💬 회사 노트북은 퇴사할 때 반납해야 돼요. → 公司笔记本离职时要还。
- **📌 문법A이야, B이야?**
  → 是 A 还是 B ？（半语二选一疑问）
  名词 + 이야(收音) / 야(无收音) ?。敬语版：A예요, B예요?
  💬 커피야, 콜라야? → 咖啡还是可乐？
  💬 농담이야, 진심이야? → 是开玩笑还是认真的？

🎵 ›증정이야→ [증정이야] (无音变，平铺直读)

🎵 ›반납이야→ [반나비야] (连音化：ㅂ 받침 + ㅇ → 顺移)

### 台词11

**韩语：** 어, 그건 반납이에요

**中文：** → 嗯，那个是要还的

- ** 회화어**
  → 嗯（半语应答；比 네 随意，对熟人用）
- **📍 단어그건**
  → "그것은" 的缩略 = 那个（话题主语）
  이건(this) / 그건(that) / 저건(that over there) — 都是 ~것은 缩略
  💬 그건 제 거예요. → 那个是我的。
- **📌 문법~이에요 / 예요**
  → 是……（"~이다" 敬语）
  收音结尾 + 이에요 / 无收音 + 예요。반납 有收音 ㅂ → 반납이에요
  💬 학생이에요. → 是学生。
  💬 제 친구예요. → 是我朋友。

### 台词12

**韩语：** 까지지 않게 조심하세요

**中文：** → 小心别刮花了

- **📍 단어까지다**
  → 擦破、剐蹭、磨损（皮/漆面）

## 🎬 城堡通译·通译师初登场

> 🎭 **场景背景：** 转场到城堡拍摄，男主朱浩进作为通译师第一次正式登场。面对「那里不能靠」「通译师怎么突然插话了」的质疑，男主用冷静专业的态度一一化解。但两人之间的空气开始微妙起来——「是在担心我吗？怕我被风吹走？」
> **💡 这节学什么：** 禁止指令(~면 안 된다)、反问语气(~나?)、「是用…写的(쓰여 있다)」等实用表达。

### 台词1

**韩语：** 어, 거기 기대면 안 됩니다, 물러서세요

**中文：** → 那里不能靠，请退后

- ** 동사기대다**
  → 倚靠、靠着
  💬 벽에 기대지 마세요. → 不要靠墙。
- **📌 문법~(으)면 안 됩니다**
  → 不能/不可以做……（禁止）
  条件 ~(으)면 + 안 되다(不行)。比 ~지 마세요 更书面、正式。
  💬 여기서 사진 찍으면 안 돼요. → 这里不能拍照。
- ** 동사물러서다**
  → 退后、退开
  물러(=退) + 서다(=站) → 站到后面
  💬 한 발짝 물러서 주세요. → 请退后一步。

🎵 ›안 됩니다→ [안됨니다] (鼻音化: ㅂ+ㄴ→ㅁㄴ)

- ** 부사불쑥**
  → 突然、冷不防
  💬 불쑥 찾아오지 마. → 别突然找上门来。

**韩语：** 나 걱정해 주는 거예요? 바람에 날아갈까 봐?

### 台词4

**韩语：** 그 앞의 표지판에 쓰여 있습니다

## 🎬 黑历史·语言梗

> 🎭 **场景背景：** 休息时间闲聊，朱浩进透露自己以前在这里拍过戏——还是新人的时候NG太多，成了「黑历史」。女主打趣说要去YouTube搜搜看，两人关系在笑声中拉近。「现在是好朋友了！」
> **💡 这节学什么：** 「有过…的经历(~ㄴ 적이 있다)」「记得清清楚楚(또렷하게 기억하다)」「黑历史(흑역사)」等表达。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 예전에 여기서 드라마를 찍은 적이 있어요

  💬 한국에 가 본 적이 있어요. → 去过韩国。
  💬 김치를 먹어 본 적이 없어요. → 没吃过泡菜。

  💬 날씨가 따뜻해졌어요. → 天气变暖了。

## 🎬 告白通译·真心话

> 🎭 **场景背景：** 气氛忽然变了。朱浩进让车武熙做翻译，却说出了一句不像工作指令的话——「我真心喜欢车武熙这个人」。通过翻译来告白，既认真又有距离感，让人分不清是真心还是玩笑。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 지금 뭐라고 한 거예요?

**中文：** → 刚才说什么了？

- **📌 문법~라고 하다**
  → 说……(间接引语)
  陈述句间接引语：动词어요/예요 → ~다고 하다 ／ 名词 → ~(이)라고 하다
  💬 학생이라고 했어요. → 说是学生。
- **📌 문법~ㄴ/은 거예요?**
  → 是……的吗？（确认事实）
  ~ㄴ 것이에요? 的口语形。比 ~었어요? 更带追问意味。
  💬 어디 간 거예요? → 你去哪儿了？

🎵 ›뭐라고 한 거예요→ [뭐라고한거에요] (流畅连读)

---

### 台词2

**韩语：** 지금 인사하는 거예요?

**中文：** → 现在是在打招呼吗？

- ** 명사인사**
  → 问候、招呼
  汉字词 人事。常见搭配：인사하다(打招呼) / 작별 인사(告别) / 첫인사(初次问候)
- **📌 문법~는 거예요?**
  → 在做……吗？（询问当下行为意图）
  动词 ~는 것 + 이에요 = 做……这件事
  💬 뭘 보는 거예요? → 在看什么呢？

🎵 ›인사하는→ [인사하는] (清晰)

---

### 台词3

**韩语：** 주호진 씨 통역해요

**中文：** → 朱浩进先生，请翻译

- ** 명사통역**
  → 口译
  통역(口译) ↔ 번역(笔译)。동사: 통역하다, 통역해 주다
- ** 호칭~씨**
  → ……先生／女士（中性礼貌称呼）
  名字 + 씨。比 님(尊敬) 平和，比 야/아(亲昵) 礼貌。
  💬 김민수 씨, 안녕하세요. → 金敏洙先生您好。
- **📌 문법~해요**
  → 既可陈述也可命令(口语命令)
  语调上扬=疑问 / 平=陈述 / 降调=命令(轻命令)

### 台词4

**韩语：** 당신이 말을 전해 줘야 내가 답을 하지

**中文：** → 你得把话传过来，我才能回答呀

- ** 동사전하다**
  → 传达、转达
  💬 안부를 전해 주세요. → 请代我问好。
- **📌 문법~아/어야**
  → 必须……才能……（条件）
  필요조건 = 必要条件。后续句意"才能/才会"。
  💬 일찍 자야 안 늦어요. → 早睡才不会迟到。
- **📌 문법~지**
  → ……呀／嘛（自然结论语气）
  ~지 결句尾，常表"理所当然""明显的事"
  💬 그렇게 해야 되지. → 当然得那样啊。

🎵 ›줘야→ [줘야] (자연 발음)

### 台词5

**韩语：** 준비한 작별 인사는 하지 않아도 되겠네요

**中文：** → 看来准备好的告别问候用不上了

- ** 명사작별 인사**
  → 告别、道别
  作别人事 = 분별 인사 / 마지막 인사
- **📌 문법~지 않아도 되다**
  → 不用……也行
  ~지 않다(不) + ~아/어도 되다(也行)
  💬 안 와도 돼요. → 不来也没关系。
- **📌 문법~겠네요**
  → 看来……呢（推测＋感叹）
  겠 (推测) + 네요 (感叹)。表"看样子是这样"
  💬 비가 오겠네요. → 看来要下雨。

### 台词6

**韩语：** 통역해 드리겠습니다

**中文：** → 我来帮您翻译

- ** 敬语~아/어 드리다**
  → ~아/어 주다 的尊敬形
  주다(给) → 드리다(奉献给)。对上位者的"为您做某事"
  💬 가르쳐 드릴게요. → 我来教您。
  💬 도와 드리겠습니다. → 我来帮您。
- **📌 문법~겠습니다**
  → ……(将要)、(我会……)
  表意志 + 礼貌。공식적인 말투（正式场合）。
  💬 잘 부탁드리겠습니다. → 请多指教。

🎵 ›드리겠습니다→ [드리겓씀니다] (紧音化+鼻音化)

---

### 台词7

**韩语：** 나는 진심으로 차무희라는 사람이 좋습니다

**中文：** → 我真心喜欢车武熙这个人

- ** 명사진심**
  → 真心
  진심으로 = 真心地。반대 = 거짓(虚假)
  💬 진심이에요. → 是真心的。
- **📌 문법~(이)라는**
  → 叫做……的；称为……的
  ~(이)라고 하는 的缩略。后接名词，定义/介绍。
  💬 김치라는 음식 알아요? → 知道叫泡菜的食物吗？
  💬 사랑이라는 게 뭐예요? → 所谓爱情是什么？
- **📌 문법이 좋다 / 을 좋아하다**
  → "喜欢"两种说法
  N이/가 좋다 = N(对我而言)是好的(感觉) ／ N을/를 좋아하다 = 喜欢N(动作)
  💬 너가 좋아. = 너를 좋아해. → 喜欢你。

🎵 ›좋습니다→ [조씀니다] (ㅎ脱落+紧音化+鼻音化)

---

### 台词8

**韩语：** 제대로 전달이 됐습니까?

**中文：** → 准确传达到了吗？

- ** 부사제대로**
  → 好好地、正确地
  💬 제대로 좀 해 봐. → 好好做。
- ** 명사전달**
  → 传达、传递
  汉字词 传达。동사: 전달하다 / 전달되다(被传达)
  💬 메시지를 전달했어요. → 传达了消息。
- **📌 문법~이/가 되다**
  → 成为……、变得……
  전달이 되다 = 被传达。被动 1 - 名词 + 이/가 되다

🎵 ›됐습니까→ [됟씀니까] (紧音化+鼻音化)

## 🎬 韩式比喻

> 🎭 **场景背景：** 拍摄结束后，前辈调侃朱浩进翻译的功劳：「我说得乱七八糟你也能翻得明明白白」（개떡같이 말해도 찰떡같이 통역）。这是韩语里最经典的比喻式表达之一。后面还冒出一句「굼벵이도 구르는 재주가 있다」(蛴螬也有滚动的本事)。
> **💡 这节学什么：** 韩语里的「俗语/比喻」！개떡·찰떡·굼벵이——学地道韩式比喻，让韩语表达更有味。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 늙은이 쫓아다니면서 통역하느라 수고 많았어

**中文：** → 跟着我这老头子翻译辛苦了

- ** 명사늙은이**
  → 老头子（自嘲/略带贬义）
  늙다(老) + 은(冠形) + 이(人)。委婉敬称: 어르신
- ** 동사쫓아다니다**
  → 跟着、追着跑
  쫓다(追) + 아 + 다니다(往返)
  💬 강아지가 나를 쫓아다녀요. → 小狗跟着我。
- **📌 문법~느라(고)**
  → 因为做……（消耗时间/精力）
  表负面后果常用："因为忙做A，所以B(没做/没能)"
  💬 공부하느라 잠을 못 잤어요. → 学习累得没睡。
- ** 표현수고 많았어요**
  → 辛苦了
  敬语: 수고하셨습니다 ／ 平等: 수고했어 ／ 注意: 수고하세요 不能对长辈说！

🎵 ›늙은이→ [늘그니] (双终声+连音化)

---

### 台词2

**韩语：** 개떡같이 말해도 찰떡같이 통역해 주는 거는

**中文：** → 我说得乱七八糟你也能翻得明明白白

- ** 속담개떡같이 말해도 찰떡같이 알아듣다**
  → 别人说得不清不楚也能完全理解（=韩国超高频谚语）
  개떡(粗糙糙的米糕，粗糙混乱) ↔ 찰떡(粘糕，紧实漂亮)。"乱说也准懂"
  💬 우리 엄마는 개떡같이 말해도 찰떡같이 알아들어요. → 我妈即使我说得乱七八糟也能完全听懂。
- ** 유사 속담관련 음식 비유**
  ▸ 떡 줄 사람은 생각도 안 하는데 김칫국부터 마신다 = 还没给糕，自己先喝泡菜汤(自作多情)

🎵 ›개떡같이→ [개떡까치] (紧音化+ㅌ→ㅊ口蓋音化)

🎵 ›찰떡같이→ [찰떡까치] (同上)

---

### 台词3

**韩语：** 스페인 가는 거를 좀 미루지?

**中文：** → 去西班牙的事推迟一下吧？

- ** 동사미루다**
  → 推迟、拖延
  💬 회의를 미뤘어요. → 把会议推迟了。
  💬 일을 미루지 마. → 别拖延工作。
- **📌 문법~지?**
  → 不是……吗？／……吧？（提议/确认）
  尾音上扬+轻语气=委婉提议。"미루지?" = "推一下不行吗？"

🎵 ›가는 거를→ [가는거를] (口语中常说 "가는거 좀")

---

### 台词4

**韩语：** 기어이 일본까지 끌고 와 써먹으셨으면 됐지

**中文：** → 硬把人拽到日本来用都用了

- ** 부사기어이**
  → 硬要、终究、非要
  💬 기어이 가야겠어? → 非要去吗？
- ** 동사끌고 오다**
  → 拽过来、带过来
  끌다(拉) + 고 + 오다(来)
- ** 동사써먹다**
  → 用上、利用上(略含负面)
  쓰다(用) + 어 + 먹다(掉)。"用掉、消耗"
  💬 그 지식을 써먹을 데가 없어요. → 那知识没地方用。
- **📌 문법~(으)면 됐지**
  → ……就行了/就够了吧（带反讽/无奈语气）
  💬 그만하면 됐지. → 那样就够了吧。

### 台词5

**韩语：** 그거 너무 뻔뻔하신 거 아닙니까?

**中文：** → 那也太厚脸皮了吧？

- ** 형용사뻔뻔하다**
  → 厚脸皮、不知羞耻
  💬 정말 뻔뻔하다. → 真厚脸皮。
  💬 뻔뻔한 사람이에요. → 厚脸皮的人。
- ** 유의어관련 형용사**
  ▸ 부끄러움이 없다 = 不知羞
- **📌 문법~신 거 아닙니까?**
  → 不是太……了吗？（敬语反问）
  ~시 + ㄴ 거 = 您是……的吗。带轻微抗议语气。

---

### 台词6

**韩语：** 술 줄이세요

**中文：** → 少喝点酒

- ** 동사줄이다**
  → 减少、缩小
  줄다(减少) → 줄이다(使减少)。사동사
  💬 살을 줄이고 싶어요. → 想减肥。
  💬 소리를 줄여 주세요. → 把声音调小。
- ** 관용술 줄이다 / 끊다**
  → 减少喝酒 / 戒酒
  💬 담배 끊었어요. → 戒烟了。

---

### 台词7

**韩语：** 일본 온 김에 가 볼 데가 있어서요

**中文：** → 趁来日本想去个地方

- **📌 문법~(으)ㄴ 김에**
  → 趁……的机会、顺便……
  ~(으)ㄴ 김(势头/机会) + 에。"既然已经……不如再……"
  💬 시장 간 김에 우유도 사 와. → 既然去市场了顺便买点牛奶。
  💬 일어난 김에 운동할까? → 既然起来了不如运动？
- **📌 문법~아/어 보다**
  → 试着……、~过
  "가 보다" = 去看看 / 去过
- ** 대명사데**
  → 地方
  "가 볼 데" = 想去看看的地方。곳 의 구어체

### 台词8

**韩语：** 굼벵이도 구르는 재주가 있다더니

**中文：** → 都说蛴螬也有滚动的本事

- ** 속담굼벵이도 구르는 재주가 있다**
  → 即使是看似无能的人也有自己的本事(俗语)
  굼벵이 = 蛴螬（看似笨重的虫子）／ 구르다 = 滚 ／ 재주 = 本事。是夸赞性俗语。
- **📌 문법~다더니**
  → 都说……(果然如此)
  ~다고 하더니 缩略。引用过去听过的话+(果然/没想到)。
  💬 한국 김치 맛있다더니, 정말 맛있어요! → 都说韩国泡菜好吃，果然！

🎵 ›재주가→ [재주가] (清晰)

🎵 ›있다더니→ [읻따더니] (紧音化)

## 🎬 初次相遇·Papago事件

> 🎭 **场景背景：** 车武熙偷偷听了朱浩进不想让人知道的对话，被当场抓包。「都听懂了？」「只是听到了，是偷听的吧」「装作中国人？」「你以为自己是Papago翻译机吗？」——乌龙又搞笑的第一次「正式」交流。
> **💡 这节学什么：** 「假装…(~척하다)」「说点像样的话(말이 되는 소리)」「用Papago被甩太悲惨了」等口语表达。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 다 알아들었어요?

**中文：** → 都听懂了？

- ** 동사알아듣다**
  → 听懂、领会
  알다(知道) + 아 + 듣다(听) → 听明白。ㄷ→ㄹ 不规则: 알아들으면, 알아들었어요
  💬 한국어 잘 알아들어요. → 我能听懂韩语。
- ** 대조듣다 vs 알아듣다 vs 들리다**
  ▸ 듣다 = 听 (主动) ／ 들리다 = 被听到 (被动)

🎵 ›알아들었어요→ [아라드러써요] (连音化)

---

### 台词2

**韩语：** 그냥 들린 겁니다, 몰래 들은 거죠

**中文：** → 只是听到了 / 是偷听的吧

- ** 동사들리다**
  → 被听到、能听见（피동사）
  듣다 → 들리다。"我听到了"= 들렸어요，"自动传到耳朵"
  💬 노래가 잘 들려요. → 歌很清楚。
- ** 부사몰래**
  → 偷偷地
  "모르게" 不让别人知道。반대 = 대놓고(公然地)
  💬 몰래 들어왔어. → 偷偷进来的。
- **📌 문법~ㄴ/은 거죠**
  → 是……的吧（推测+确认）
  ~ㄴ 것이지요 缩略，比 ~았/었지요 多了"行为本质"的味道

🎵 ›들린 겁니다→ [들린검니다] (鼻音化)

---

### 台词3

**韩语：** 중국 사람인 척하면서

**中文：** → 装作中国人

- **📌 문법~인 척하다 / ~는 척하다**
  → 装作……、假装……
  名词 + 인 척 ／ 动词 + 는 척 ／ 形容词 + ~(으)ㄴ 척
  💬 자는 척했어요. → 装睡了。
  💬 모르는 척하지 마. → 别装不知道。
  💬 부자인 척했어. → 装富。
- **📌 문법~(으)면서**
  → 一边……一边……
  同时进行的动作。批评语气："明明……还(同时)……"
  💬 노래 들으면서 공부해요. → 边听歌边学习。

---

### 台词4

**韩语：** 아니, 본 척한 게 아니고 본 거고

**中文：** → 不是装作看，是真的看

- **📌 문법~ㄴ 게 아니고 ~ㄴ 거고**
  → 不是……而是……
  "本척한 게 아니고 본 거" = 不是装看，是真看。强调否定+修正。
  💬 화난 게 아니고 피곤한 거예요. → 不是生气，是累了。
- ** 감탄사아니**
  → 不(是)、欸（口语反驳/惊讶）
  💬 아니, 그게 아니라… → 不是，我说的不是那个…

---

### 台词5

**韩语：** 말이 되는 소리를 해요

**中文：** → 说点像样的话好不好

- ** 관용구말이 되다 / 안 되다**
  → 像话／不像话
  "말이 안 돼" = 不像话/没道理 ／ "말이 되는 소리" = 有道理的话
  💬 그게 말이 돼? → 那像话吗？
  💬 말도 안 돼! → 太离谱了！
- ** 관용관련 표현**
  ▸ 말도 안 되는 소리 = 不像话的话

---

### 台词6

**韩语：** 자기가 무슨 파파고인가

**中文：** → 你以为自己是Papago翻译机吗

- ** 고유명사파파고 (Papago)**
  → 韩国 NAVER 公司开发的翻译App，韩国人最常用
  语源 = Papagaio（葡萄牙语"鹦鹉"）。剧中讽刺男主"什么语言都翻"。
- ** 호칭자기**
  → ① 自己 ② 你（口语年长者→年轻人，或情侣间）
  这里"자기" 表"你"，带轻微讽刺/不屑（무희 比 호진 大）
  💬 자기 일이나 잘하세요. → 管好你自己的事。
- **📌 문법무슨 ~인가**
  → 难道是……吗（讽刺反问）
  💬 자기가 무슨 의사야? → 你以为你是医生啊？

### 台词7

**韩语：** 아이가 알레르기 때문에 호흡 곤란으로 쓰러졌다고

**中文：** → 孩子因过敏呼吸困难晕倒了

- ** 명사알레르기**
  → 过敏 (Allergy)
  💬 견과 알레르기가 있어요. → 我对坚果过敏。
- ** 명사호흡 곤란**
  → 呼吸困难
  汉字 呼吸困难 = 호흡(呼吸) + 곤란(困难)
- ** 동사쓰러지다**
  → 倒下、晕倒
  💬 더위에 쓰러졌어요. → 中暑晕倒了。
- **📌 문법~ 때문에**
  → 因为（名词原因）
  名词 + 때문에 ／ 形容词/动词 + 기 때문에
  💬 비 때문에 못 갔어요. → 因为下雨没去成。
- **📌 문법~다고 (전달)**
  → ……(让对方传达)
  陈述间接引语 = ~다고 하다。这里 호진 让店员"告知救护车"，用 ~다고 알려 주세요

## 🎬 我的爱情请翻译·女主告白

> 🎭 **场景背景：** 全剧最催泪的一段。车武熙在朱浩进面前坦白自己的感情经历——被单方面分手、卑微地等待、想要保留尊严地结束……最后说出那句最经典的请求：「내 사랑도 통역해 주세요. 부탁해요」（也请翻译我的爱情。拜托你了。）
> **💡 这节学什么：** 感情相关的高阶表达！「单方面通知分手」「就算寒酸又窝囊，我的也是爱情」「不想这样收场」「保留尊严地结束」。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 남의 치정 싸움을 통역하고 싶지는 않습니다

**中文：** → 我不想替别人翻译三角恋的争吵

- ** 명사남**
  → 别人、他人
  💬 남의 일에 끼어들지 마. → 别管别人的事。
- ** 명사치정**
  → 痴情/感情纠纷
  汉字 痴情。常见: 치정 싸움 = 三角恋纠纷 ／ 치정극 = 情感剧
- ** 명사싸움**
  → 争吵、打架
  动词 싸우다 → 名词 싸움
- **📌 문법~고 싶지(는) 않다**
  → (倒是)不想……
  ~고 싶다(想) + 否定 + 주제 ~는 = 强调"恰恰不想"
  💬 가고 싶지는 않아. → 倒是不想去。

### 台词2

**韩语：** 파파고로 까이면 너무 비참할 거 같아서 그래요

**中文：** → 用Papago被甩太悲惨了，所以我才……

- ** 슬랭까이다**
  → 被甩、被拒绝（口语）
  까다(剥/损) → 까이다(被剥/被损)。年轻人常用："고백 까였어" = 告白被甩
  💬 어제 까였어요. → 昨天被甩了。
- ** 형용사비참하다**
  → 悲惨
  汉字 悲惨。同义 = 처참하다(更书面)
  💬 비참한 결말이었어요. → 是悲惨的结局。
- **📌 문법~ㄹ 거 같아서 그래요**
  → (我)是觉得会……才这样
  ~ㄹ 거 같다(看起来会) + ~아서(因为) + 그래요(才那样)

---

### 台词3

**韩语：** 일방적으로 헤어지자고 통보해 놓고 사라졌어

**中文：** → 单方面通知分手然后消失了

- ** 부사일방적으로**

  일방(一方) + 적(的) + 으로(地)
- ** 동사헤어지다**
  → 分手、分别
  💬 어제 남친이랑 헤어졌어요. → 昨天和男朋友分手了。
- ** 동사통보하다**
  → 通知（通常单方面/正式）

- ** 동사사라지다**
  → 消失
  💬 갑자기 사라졌어요. → 突然消失了。
- **📌 문법~자고 하다**
  → 提议……（间接引语）
  "헤어지자" 直接说 → "헤어지자고 했어" 间接转述
- **📌 문법~아/어 놓고**
  → 做完……还……（带不满/反差语气）
  💬 일은 안 해 놓고 큰소리치네. → 活儿没干还嘴硬。

### 台词4

**韩语：** 전에도 그런 적 있었으니까 이번에도 그러려니 기다렸고

**中文：** → 之前也有过这种情况，这次也以为他会回来就等了

  → 想着会那样、当作如此

  💬 그러려니 하고 넘어갔어. → 当作那样就过去了。
- **📌 문법~(으)니까**
  → 因为……（理由，主观/经验）
  ~아/어서 强调客观因果，~(으)니까 主观＋后接命令/建议
- ** 부사전에**
  → 以前、之前
  💬 전에 만난 적 있어요. → 以前见过。

### 台词5

**韩语：** 구차하고 찌질해도 내 것도 사랑이에요

**中文：** → 就算寒酸又窝囊，我的也是爱情

- ** 형용사구차하다**

  💬 구차한 변명 그만해. → 别再苦涩地辩解了。
- ** 슬랭찌질하다**
  → 窝囊、不上台面（年轻人词）
  💬 정말 찌질하다. → 真是窝囊。
  💬 찌질남 = 窝囊男
- **📌 문법~아/어도**
  → 即使……也……
  💬 비싸도 살 거예요. → 再贵也要买。

### 台词6

**韩语：** 이러고 끝내고 싶지는 않다고요

**中文：** → 我不想这样收场

- ** 표현이러고**
  → 像这样、就这么
  이렇게 하고 缩略。"这样下去"的口语形
- **📌 문법~다고요**
  → ……呢/我说……（强调/重复）

  💬 사랑한다고요! → 我说我爱你呢！

### 台词7

**韩语：** 최소한의 품위는 지키면서 끝낼 수 있게

  → 最起码、最低限度

  💬 품위 있는 사람 = 有品位/有教养的人
- ** 동사지키다**

  💬 약속을 지켜 주세요. → 请遵守承诺。

  → 为了能够……（目的）
  ~(으)ㄹ 수 있다 + 게(为了)

## 🎬 守住品位·男主出手

> 🎭 **场景背景：** 男二出现了。朱浩进察觉到了气氛不对，「既然察觉到了就什么都别说」。但他并没有袖手旁观——「会伤害这个人内心的话，我不会传达」。他没有直接替她做决定，而是守护她体面地处理一切。
> **💡 这节学什么：** 「察觉到了(눈치챘다)」「快哭出来了(눈물 나올 거 같아)」「我来收拾(내가 수습해 볼게요)」「守住品位(품위 있게)」等表达。

### 台词1

**韩语：** 눈치챘으니까 아무 말 마요

**中文：** → 既然察觉到了就什么都别说

- ** 명사눈치**
  → 眼色、察言观色的能力

  💬 눈치 없다 = 不会看眼色

- ** 동사눈치채다**

  💬 눈치챘어요? → 察觉到了？
- **📌 문법~지 마요 / 마세요**

  "아무 말 마" = 아무 말 하지 마 的口语缩略

  💬 눈물이 나요. → 流眼泪。

  💬 다 먹어 버렸어. → 全吃完了。
  💬 잊어버렸어. → 忘掉了。

### 台词4

  汉字 收拾。常见: 사태를 수습하다(应对事态)
  💬 일을 수습해야 돼. → 必须收拾局面。
- **📌 문법~아/어 보다**

  💬 한번 먹어 봐요. → 试着尝一下。

  → 我会……（意志/承诺）

## 🎬 晚餐邀请·拐弯抹角的好意

> 🎭 **场景背景：** 事情解决了，但气氛还有些尴尬。朱浩进想请车武熙吃饭，又不想显得太刻意——「接下来要做什么？」「方便的话和我一起吃晚饭吗？」「两人座一个人吃太可惜」。明明是关心，却非要找一堆借口，这就是韩式「拐弯抹角的好意」。
> **💡 这节学什么：** 邀请相关的委婉表达！「方便的话…(~ㄴ/는다면)」「一个人吃太可惜」「再好吃的东西一个人吃也没味道」等。

> **📋 学习提示：** 以下内容建议在电脑或平板打开，**左边阅读内容，右边空白处做笔记**。

### 台词1

**韩语：** 지금부터 뭐 할 겁니까? 뭐 할 일 있습니까?

**中文：** → 接下来要做什么？有什么要做的事吗？

- **📌 문법~ㄹ 겁니까?**
  → 会做……吗？/ 打算……吗？(正式)
  ~ㄹ 거예요? 的正式形 ~ㄹ 겁니까?
  💬 어디 갈 겁니까? → 您要去哪里？
- **📌 문법~ㄹ 일**
  → 要做的事
  动词 ~(으)ㄹ + 일(事) 修饰名词。"할 일" = 要做的事
  💬 할 일이 많아요. → 要做的事很多。

🎵 ›지금부터→ [지금부터] (清晰)

---

### 台词2

**韩语：** 괜찮다면 저랑 같이 저녁 드실래요?

**中文：** → 方便的话和我一起吃晚饭吗？

- ** 동사드시다**
  → 吃/喝（敬语）
  먹다/마시다 → 드시다 (尊敬语)。"드세요" = 请吃/请喝
  💬 뭐 드실래요? → 您要吃什么？
- **📌 문법~ㄹ래요?**
  → ……吗？(询问对方意愿)
  用于询问对方"愿不愿意"。比 ~ㄹ까요?(咱要不要) 更聚焦对方
  💬 커피 마실래요? → 喝咖啡吗？
- **📌 문법~다면**
  → 如果……的话
  ~다고 하면 缩略，假设。"괜찮다면" = 如果方便的话

### 台词3

**韩语：** 아까는 싫다더니?

**中文：** → 刚才不是说不要吗？

- ** 부사아까**
  → 刚才
  💬 아까 뭐라고 했어요? → 刚才说什么了？
- **📌 문법~다더니?**
  → 不是说……吗？（反问/对比）
  ~다고 하더니 缩略。指出"前后不一致"
  💬 안 온다더니 왔네? → 不是说不来吗，怎么来了？

### 台词4

**韩语：** 굳이 같이 먹을 이유가 없다는 거였지

**中文：** → 我说的是没必要一起吃

- ** 부사굳이**
  → 非要、特意
  韩国年轻人爆款词。"굳이?" = "干嘛非要这样？"
  💬 굳이 가야 돼? → 非要去吗？
  💬 굳이 그럴 필요 있어? → 非那样不可吗？
- ** 명사이유**
  → 理由
  汉字 理由
- **📌 문법~다는 거**
  → 所谓……（前面整句作名词）
  ~다고 하는 것 = 所说的事。把陈述句变成名词
  💬 안 간다는 거예요? → 是说不去吗？

🎵 ›굳이→ [구지] (口蓋音化: ㄷ+ㅣ→ㅈ)

### 台词5

**韩语：** 두 사람 자리라는데 혼자 먹으면 아깝잖아요

**中文：** → 听说是两人座，一个人吃太可惜了

- ** 명사자리**
  → 座位、位子；位置
  💬 자리 있어요? → 有座吗？
- ** 형용사아깝다**
  → 可惜、舍不得
  💬 시간이 아까워요. → 时间可惜。
  💬 너무 아까워! → 太可惜了！
- **📌 문법~잖아요**
  → 不是……嘛（提醒已知事实）
  ~지 않아요? 的口语缩略。带"你也知道吧"的语气
  💬 그건 비싸잖아. → 那个不是很贵嘛。
- **📌 문법~ㄴ다는데**
  → 听说……（陈述）
  ~ㄴ다고 하는데 缩略，引用+继续

### 台词6

**韩语：** 그럼 내가 같이 가서 먹어 줄게요. 신세 진 것도 있는데

**中文：** → 那我陪你去吃。我也欠你人情

- ** 명사신세**
  → 人情、关照
  汉字 身世。常用搭配 신세를 지다(欠人情) ／ 신세를 갚다(还人情)
  💬 신세 졌어요. → 受您照顾了。
- ** 표현먹어 주다**
  → (帮你/给你面子)吃
  "~아/어 주다" 在这里带"勉强答应/给面子"语气
- ** 동사지다 (신세를 지다)**
  → 欠下、承受（人情/责任）

### 台词7

**韩语：** 왜 신세를 갚는 것처럼 생색을 내지?

**中文：** → 怎么像还人情似的还邀功？

- ** 동사갚다**
  → 偿还（钱/恩情）
  💬 돈을 갚아요. → 还钱。
  💬 은혜를 갚을게요. → 我会报恩。
- ** 명사생색**
  → 邀功、自夸
  汉字 生色。常用 생색을 내다(邀功显摆)
  💬 생색내지 마. → 别邀功。
- **📌 문법~처럼**
  → 像……一样
  💬 새처럼 날아요. → 像鸟一样飞。

### 台词8

**韩语：** 암만 맛있는 것도 혼자 먹으면 맛이 없고 많이 안 들어가요

**中文：** → 再好吃的东西一个人吃也没味道，吃不下

- ** 부사암만**
  → 再……、不管多……（口语）
  아무리 的口语缩略
  💬 암만 봐도 모르겠어. → 怎么看都不明白。
- ** 표현맛이 있다 / 없다**
  → 有/没有味道
  맛있다 (粘合) → 맛이 있다 (分开强调)
- ** 관용(음식이) 안 들어가다**
  → 吃不下（食物不进口）
  字面"东西进不去(嘴/胃)"
  💬 슬퍼서 밥이 안 들어가요. → 难过得吃不下。
