第29课 · 让我平静下来的人就是你 台词精讲 ①/⑥
냉정하고 차분하게 나를 진정시켜줄 사람이 옆에 있으면
→ 如果有个冷静又沉稳地能让我平静下来的人在身边的话
→ If there's a person next to me who can calmly and coolly soothe me
谁 사람（人）
← 前面修饰：「냉정하고 차분하게 나를 진정시켜줄」
-고 串"冷静"+"沉稳"；-아/어주다"帮谁做"；-ㄹ修饰
什么 没有宾语
干 있으면（如果有）→ -면 = 如果
📎 连接：不是两个句子拼的。修饰内部用-고把"冷静"+"沉稳"串在一起
📍 单词
냉정하다冷静 (cool-hearted)
차분하다沉稳 (calm/composed)
진정시키다使镇定 (calm down/soothe)
사람人 (person)
옆旁边 (next to/side)
있다有/在 (exist)
💬 短语
진정시켜주다 — 帮忙安抚（-아/어주다"给/帮"）
💬 造句
그녀가 나를 진정시켜줬어요. — 她让我平静下来了。
🔧 句型 / 接口
-고 连接两个形容词：냉정하고 차분하다 = 又冷静又沉稳（像英语 A and B）
-아/어주다 动词+"帮谁做"：진정시키줄 = 帮我平静（"给"塞进动词里）
-면 "如果"：있으면 = 如果有（条件句，if）
🔊 音变现象
냉정하고[냉정하고] · 차분하게[차분하게] → 无特殊音变

第29课 · 让我平静下来的人就是你 台词精讲 ②/⑥
어떨 것 같아요?
→ （你觉得）会怎么样呢？
→ What do you think it would be like?
谁 省略了（默认"你"）
什么 省略了（指前面说的那种情况）
干 어떨 것 같아요（会怎么样呢）→ -ㄹ 것 같다 = 推测
📎 简单推测疑问句，不是两个句子拼的。
📍 单词
어떻다怎么样 (how is it)
같다像/好像 (seems like)
💬 短语
어떨 것 같아요? — 你觉得怎么样？会怎么样呢？
💬 造句
내일 날씨가 어떨 것 같아요? — 明天天气会怎么样？
🔧 句型 / 接口
-ㄹ 것 같다 "好像会/觉得会"，推测语气。
어떻ㄹ 것 같아요 = 会怎么样呢？
→ 词干加 -ㄹ/-을 것 같다，英语 I think it will…
🔊 音变现象
같아요[가타요] → ㅌ+ㅇ 紧音化

第29课 · 让我平静下来的人就是你 台词精讲 03/⑥
그 사람이 마침 외국어도 잘하면 외국에 나가서도 안심이 될 거예요, 그쵸?
→ 那个人碰巧外语也好的话，去国外也能放心了，对吧？
→ If that person happens to be good at foreign languages too, even going abroad would be reassuring, right?
谁 그 사람（那个人）
← 마침 외국어도 잘하면（碰巧外语也好的话）条件在谁后面
什么 没有直接宾语
干 잘하면（擅长）→ -면 = 如果 | 안심이 될 거예요（会放心）→ -ㄹ 거예요 = 将来/推测
📎 两个简单句拼的！[条件] 마침 외국어도 잘하면（如果外语好）→ [结果] 안심이 될 거예요（会放心）
尾巴 + 그쵸?（对吧？确认一下）
📍 单词
마침恰好/碰巧 (coincidentally)
외국어外语 (foreign language)
잘하다擅长 (be good at)
외국外国 (foreign country)
나가다出去/去 (go out)
안심安心/放心 (relief)
되다成为/变得 (become)
💬 短语
안심이 되다 — 放心/安心
💬 造句
네가 같이 있으면 안심이 돼요. — 你在我身边我就放心了。
🔧 句型 / 接口
-도 表示"也"：외국어도 = 外语也
-면 条件句"如果"：잘하면
-서도 "即使/也"：나가서도
-ㄹ 거예요 将来/推测：될 거예요
-그쵸? 确认语气"是吧？"
🔊 音变现象
잘하면[잘하면] · 나가서도[나가서도] · 거예요[거예요] → 无特殊音变

그 사람이 마침 외국어도 잘하면 외국에 나가서도 안심이 될 거예요, 그쵸?

① 换词练习
외국어 → 운전
운전도 잘하면 좋겠어요. → 开车也好的话就好了。
외국에 나가서도 → 혼자 살아도
혼자 살아도 안심이 될 거예요. → 一个人生活也能放心。
② 场景迁移·旅行
같이 여행 가는 사람이 영어도 잘하면 안심이 돼요, 그쵸?
一起旅行的人英语也好的话就放心了，对吧？
③ 问答对话
Q: 어떤 사람이랑 여행 가고 싶어요?
你想跟什么样的人去旅行？
A: 길을 잘 찾는 사람이요!
会认路的人！
④ 句型扩展·-면+-ㄹ 거예요
비가 오면 집에 있을 거예요.
下雨的话就待在家里。
시간이 되면 도와줄 거예요.
有时间的话会帮忙的。
第29课 · 03/⑥ 台词应用
第29课 · 让我平静下来的人就是你 台词精讲 04/⑥
그런 사람이 옆에 있으면 버틸 만할 것 같은데
→ 那种人在身边的话，好像还能撑得住
→ With that kind of person by my side, I think I could endure it
谁 그런 사람（那种人）
← 그런（那样的）直接贴在사람前面
什么 没有宾语
干 있으면（在）→ -면 = 如果 | 버틸 만할 것 같은데（好像能撑住）→ -ㄹ 만하다 + -ㄹ 것 같다 + -는데
📎 两个简单句拼的。[条件] 사람이 옆에 있으면（人在旁边）→ [结果] 버틸 만할 것 같은데（好像能撑住）
「-는데」留个话尾巴等对方接。
📍 单词
그런那样的 (such/that kind of)
옆旁边 (next to)
버티다坚持/撑住 (endure/withstand)
만하다值得/足以 (worth/worthy)
💬 短语
버틸 만하다 — 还能撑得住/值得坚持
💬 造句
이 정도 일은 버틸 만해요. — 这种程度的事还能撑得住。
🔧 句型 / 接口
-면 "如果"：있으면
-ㄹ 만하다 "值得/足以"：버틸 만하ㄹ（볼 만하다=值得看）
-ㄹ 것 같다 推测"好像会"
-는데 铺垫语气，话说一半等接话
🔊 音变现象
버틸[버틸] · 만할[만할] → 无特殊音变

虽然一个人学很难，但有老师在身边就能撑住。
③ 问答对话
Q: 이 일이 버틸 만해요?
这事能撑得住吗？
A: 네, 좀 힘들지만 버틸 만해요.
嗯，虽然有点累但还能撑。
④ 句型扩展·-ㄹ 만하다
이 영화는 볼 만해요.
这部电影值得一看。
이 집 김치는 먹을 만해요.
这家的泡菜还能吃。

어떻게 생각해요?
→ 你怎么想？
→ What do you think?
谁 省略了（默认"你"）
← （无）
什么 省略了（默认"刚才说的事"）
干 생각해요（想/认为）← 前面加修饰：「어떻게」（怎么/如何）

📎 不是连接句。简单疑问句——"你怎么+想" = 你怎么想？
📍 单词
어떻게怎么/如何 (how)
생각하다想/思考 (think)
💬 短语
어떻게 생각해요? — 你怎么想？你觉得呢？（日常高频）
💬 造句
이 제안에 대해 어떻게 생각해요? — 对这个提议你怎么看？
🔧 句型 / 接口
-에 대해 "关于……"：이 제안에 대해
🔊 音变现象
생각해요[생가캐요] → ㅎ+ㅏ紧音化（각+해→가캐）

第29课 · 05/⑥ 台词应用
第29课 · 让我平静下来的人就是你 台词精讲 06/⑥
민희미 씨, 내가 알아듣게 얘기해요.
→ 闵熙美小姐，说我能听懂的话吧。
→ Ms. Min Huimi, speak so that I can understand.
谁 민희미 씨（闵熙美小姐 — 呼语，不是主语）
← （无）
什么 没有宾语
干 얘기해요（说吧）← 前面加修饰：「내가 알아듣게」（让我能听懂地）
→ 알아듣게 = -게（目的/方式）
→ 词尾：-아/어요 = 请求/命令语气
📎 不是连接句。含修饰的祈使句——"让我能听懂"地 + "说"。
📍 单词
민희미闵熙美 (Min Huimi/人名)
씨先生/小姐 (Mr./Ms.)
알아듣다听懂 (understand by hearing)
얘기하다说话/讲话 (talk/speak)
💬 短语
알아듣게 얘기하다 — 说得让人能听懂
💬 造句
쉽게 설명해 주세요. — 请简单说明一下。
🔧 句型 / 接口
-게 动词+게 = "以便…地/让…地"
알아듣게 = 让人能听懂地
-아/어요 请求/命令：얘기해요 = 说吧
🔊 音变现象
알아듣게[아라듣게] → ㄹ+아连音化
얘기해요[얘기해요] → 无特殊音变

Q: 이 내용이 너무 어려워요.
这个内容太难了。
A: 그러면 제가 쉽게 설명해 드릴게요.
那我简单说明一下。

请站到前面让大家都看到。

请切好让孩子方便吃。
