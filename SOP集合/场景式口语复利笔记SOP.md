# 场景式母语口语复利笔记 · 完整版SOP

> 适配手机/平板/电脑三端PDF

## 一、PDF版式规范

1. **页面尺寸**：竖版10:14（接近A4竖版），兼容手机竖滑、平板单页、电脑双页，支持打印装订
2. **背景**：底色 #F7F3EA（浅暖米白），叠加8%-12%透明度的细微纸张纤维纹理；浅灰细分割线 + 低透明度点阵辅助格，护眼不挡字
3. **封面**：开放式主题，可自定义
   - 上方放一**幅手绘风SVG小插画**，贴合当前场景主题（例：起床主题→太阳/云朵/床/闹钟/Zzz）
   - 插画下居中排文字

   主标题：《「_______」实景口语笔记》（横线处填场景名，如购物/职场/邀约）
   副标题：母语原版表达｜词汇精解｜句型套用｜实景落地练习
   页脚：场景式口语复利笔记 · 原版输入-改写输出闭环

4. **全书固定结构**：封面 → 开放式目录页 → SOP执行规范页 → 标准空白笔记模板页 → 周度复盘模板页 → 场景标签索引页

## 二、开放式目录（可无限新增场景）

1. 笔记标准执行SOP
2. 自定义场景分类合集（条目可随时追加）
3. 每日场景笔记记录专区
4. 每周句型复盘汇总专区

## 三、AI执行规范

### 用户发送格式
```
这是英语母语者场景教学的英文句子，场景：XXX。
按照本套竖版笔记模板处理。
```

### AI自动预处理规则
1. 收到外教讲解文字 → 自动剔除书面理论，拆分出独立口语短句
2. 自动打场景标签，方便后期检索归类
3. 文本分行规整，粘贴到笔记软件/PDF内不格式错乱

## 四、每句执行规范（固定模板）

### 句子展示
- **标题**：写英文句子（只写一次）
- **标题下方**：只写中文翻译，不重复写英文

### 每句固定几件套
按以下顺序，每个句子配齐全套：

1. **📍 单词** — 两列网格，英文→中文
2. **💡 多种表达** — 同一意思的不同说法（只列英文，不加中文说明）
3. **⚠️ 中式避雷** — 常见错误 ❌ + 正确说法 ✅
4. **🔧 万能句型** — 提炼可套用的句式结构
5. **✍️ 三层改写** — 基础 → 进阶 → 个人化
6. **🎭 实景对话** — A/B问答，贴近真实场景
7. **✍️ 反向仿写** — 给中文场景 → 参考答案

### 结尾
- 所有句子讲完后：**📌 今日总结**（汇总核心句式 + 场景标签）

## 五、周度复盘规则

1. 每周可将全部笔记汇总发AI
2. 合并同类句式，筛选2-3个最强句型，生成精简背诵清单
3. 整合所有场景标签，搭建完整口语句式索引库

## 六、硬性执行准则

1. **单日求精不求多** — 每天只吃透一组同场景表达，不堆句子
2. **素材筛选** — 只保留现实对话能用的口语，舍弃教材化书面长句
3. **完整输出** — 改写、模拟对话、反向仿写练习缺一不可，杜绝纯摘抄
4. **滚动复盘** — 周期性回看，依托标签检索，形成口语肌肉记忆

## 八、两种输出

- **完整版**：含全部解析、辨析、练习 → 存PDF长期归档

## 三、版面间距规范（2026.7.19 定稿）

根据 Notebook3 风格，确保页面疏朗可读：

### 字间距
- **letter-spacing: 1px** — 正文通用字间距
- 韩文与中文之间保持清晰间隔，如：**냉정하다 — 冷静**（左右各一空格）

### 行距
- **line-height: 2.0** — 正文行距
- 标题可稍紧，但正文必须疏朗

### 字体大小
- 正文：14~15px
- 韩文原文：18px 粗体
- 中文翻译：15px
- 英文注释：12~14px（灰色）
- 小字辅助：11px（如音变注释）

### 段落间距
- 单词网格：grid两列，gap 2px 16px，padding 24px
- 段落间：margin-bottom 5~8px
- 板块间：14px 留白

### 单词区规范
单词条目格式：
```html
<div class="word-item">
  <b>한국어</b>
  <span class="zh">中文</span>
  <span class="en"> (English)</span>
</div>
```
韩文与中文间不可挤在一起，必须有空格或边距。
