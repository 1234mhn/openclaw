### ② 直接从COS截取视频（🚫 不下载原片！）

| 步骤 | 操作 | 工具 |
|:---|:---|:---|
| 1 | 用COS脚本生成签名URL | `tencent-cos-skill` |
| 2 | ffmpeg从签名URL直接截取 + HDR→SDR转码 | `ffmpeg` |
| 3 | 上传截好的视频回COS | `tencent-cos-skill` |
| 4 | 拿到COS访问链接 | `tencent-cos-skill` |

```bash
# 生成签名URL（7200秒有效期）
node cos_node.mjs sign-url --key "原片路径" --expires 7200

# 直接从COS截取，不下载全片
ffmpeg -ss 起止时间 -to 结束时间 -i "签名URL" \
  -map 0:v -map 0:a:0 \
  -vf "zscale=t=linear:npl=100,format=gbrpf32le,zscale=p=bt709,format=yuv420p,scale=1920:1080" \
  -c:v libx264 -crf 22 -preset fast -c:a aac -b:a 192k \
  /tmp/ep2_lXX.mp4 -y
```

**🔴 HDR→SDR色调映射（必做，否则视频黢黑）**
**🔴 绝不在服务器上下载9GB原片——视频全程在COS上操作**

---

### ③ 提取台词（优先本地MKV + mkvextract，秒级完成）

**🎯 首选：本地MKV精简版 + mkvextract（几秒搞定）**

如果 `/tmp/ep02_combined.mkv` 或 `/tmp/ep02_combined2.mkv` 已存在：
```bash
# 提取韩语字幕（track 4=原版韩语）
mkvextract 文件.mkv tracks 4:输出.srt

# 提取中文字幕（track 6）
mkvextract 文件.mkv tracks 6:输出.srt

# 筛选目标时间范围
python3 -c "过滤 起止秒 到 结束秒"
```

**⚠️ HDR转码后的MP4会丢失字幕流！必须从原始MKV提取字幕，不能用whisper。**

**🔄 备选：从COS远程提取（仅当本地文件不存在时）**
```bash
timeout 300 ffmpeg -i "签名URL" -ss 起止时间 -to 结束时间 \
  -map 0:s:0 -c:s srt 输出.srt -y
```

**提取后：**
- 韩语原文 + 中文翻译 + 英文翻译（如有）
- 时间戳精确到0.1秒（取自字幕文件）

### ④ 按模板填充页面

根据 `课程模板说明.md` 的规范，依次填充：

| 填入位置 | 填入内容 |
|:---|:---|

| # | 问题 | 解决方法 |

| 1 | Whisper识别不准，台词文本有误 | 必须用片源字幕提取，不用whisper |
| 2 | 时间戳凭感觉写，跟视频对不上 | 从字幕文件精确抄写start/end |

| 5 | 单词一行堆砌 | 两列网格，每条一行 |

| 7 | 已完成按钮出现两次 | 检查是否只有一个 |
| 8 | 台词编号混乱 | 增删后重新编号 |
| 9 | handleSpeak参数是旧文本 | 更新台词时同步更新 |
| 10 | 时间戳没从字幕文件提取，手写了 | 🔴 必须用片源字幕或Whisper精确提取时间戳，误差不超过0.5秒。手动打点一定会跟视频对不上！|
