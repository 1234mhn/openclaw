# 每天写点什么？—— 系统说明

## 这是什么
每天记录思考/感悟/经验，由楠楠分析**如何应用到生活中**，并归档到知识库。

## 每次发内容，我会自动做3件事

### 1️⃣ 存档 → 归入知识库
你说的话 → 自动保存到对应日期

### 2️⃣ 三问分析
对你的内容回答三个问题：

| 问题 | 说明 |
|:----|:------|
| 🛠️ **能用吗？** | 这个想法怎么用到生活中？ |
| ♻️ **能复用吗？** | 以后还能拿出来用吗？怎么用？ |
| 📈 **能精进吗？** | 下次怎么做得更好？ |

### 3️⃣ 标签分类
自动打标签，方便以后查找复用

## 标签体系
- `#生活` — 生活感悟、经验教训
- `#学习` — 学习方法、备考心得
- `#产品` — 产品想法、创业思路
- `#工作` — 职场经历、兼职感悟
- `#旅行` — 旅行计划、见闻
- `#关系` — 人际关系、情感
- `#复盘` — 对过去经历的回顾
- `#灵感` — 一闪而过的想法
