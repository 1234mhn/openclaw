# 找工作参考手册

> 整理于 2026.06.26

---

## 一、按雇佣方式分

| 类型 | 特点 |
|------|------|
| **全职（正式员工）** | 签劳动合同，五险一金，稳定，灵活度低 |
| **兼职/小时工** | 按时间算钱，适合过渡 |
| **自由职业/接单** | 自己找活，时间自由，收入不稳定 |
| **创业/自己做生意** | 自己当老板，上限高下限也低 |
| **远程办公** | 人在国内拿国外薪水，越来越常见 |

## 二、按工作性质分

- **坐班型** — 朝九晚五，打卡上班（传统白领）
- **跑外型** — 销售、外卖、快递、地推
- **技术型** — 程序员、设计师、工程师、医生
- **服务型** — 餐饮、客服、家政
- **创意型** — 自媒体、写文案、拍视频、做设计

## 四、按公司类型分

### 🏢 体制内
- 公务员（国考/省考）
- 事业单位（学校、医院、研究所）
- 国企/央企（烟草、电网、石油、银行）
- **特点：** 稳定、福利好、升得慢、关系重要

### 🏗️ 大型民企
- 头部大厂：腾讯、阿里、字节、美团、比亚迪
- 传统民企：美的、格力、海尔
- **特点：** 薪资高、节奏快、竞争激烈、裁员也快

### 💼 外企
- 欧美系：微软、Apple、宝洁、四大
- 日韩系：松下、三星
- **特点：** WLB（工作生活平衡）、培训正规、天花板有限

### 🏪 中小企业（SME）
- 几十到几百人的公司
- **特点：** 灵活、身兼多职、学东西快、薪资低一些、不稳定

### 🚀 创业公司/Startup
- 几个人的团队到B轮融资的公司
- **特点：** 高风险高回报、期权画饼、一个人顶五个人用

### 🛠️ 个体/自由职业/工作室
- 自己接单、做项目
- **特点：** 自由、所有事自己扛、没有下限也没有上限

### 🌐 远程/全球化公司
- 在国内远程办公，给国外公司干活
- **特点：** 薪资按国外的水平、不用通勤、沟通有时差

## 五、怎么选（参考）

| 你在乎什么 | 适合去 |
|-----------|-------|
| 稳定、养老 | 体制内 / 国企 |
| 赚钱多、成长快 | 大厂 / 头部民企 |
| 平衡生活 | 外企 |
| 学东西、自由度 | 创业公司 / 小企业 |
| 谁都不管我 | 自由职业 |

> 传统眼光鄙视链：公务员 ≈ 大厂 > 外企 > 国企 > 创业公司 > 中小企业
> 但关键还是看你要什么，别被鄙视链带着走。
