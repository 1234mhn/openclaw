## P2 先颠覆一个认知

**传统的听力课：**
❌ 放录音 → 做题 → 对答案 → 讲语法
→ 美术生听了想睡觉

**今天的听力课：**
✅ 用你看画面的能力去听声音
✅ 把声音变成颜色、形状、位置
✅ 听到了，就画出来了

> 美术生的视觉记忆比听觉记忆强3倍。
> 那就别用耳朵硬扛，用眼睛帮忙。

## P3 听力本质：给你的耳朵配一只眼睛

中考听力就考这4件事，每件事对应一个「画面」：

| 考什么 | 对应画面 | 美术生理解方式 |
|--------|---------|---------------|
| 谁在说话？ | 人物轮廓 | 画一个人物速写 |
| 在哪说话？ | 场景背景 | 画一个场景草图 |
| 在说什么？ | 画面里的动作 | 画一个动态线条 |
| 什么态度？ | 色调冷暖 | 用颜色标记情绪 |

> 把对话画出来，答案就自己跳出来了。

---

## P4 第一招：画人物 · 关系题

**问人物关系时，不要背"doctor-patient"这种词，你要画轮廓**

人物关系速查画板：

```
医生 — 白大褂 + 听诊器 + 处方
师生 — 讲台 + 黑板 + 作业
店员顾客 — 收银台 + 商品 + 价格
家人 — 厨房 + 客厅 + 晚饭
朋友 — 咖啡厅 + 闲聊 + 笑
```

**真题演练：**
🎵 音频：2024年长沙中考听力第X题
听到 "Check your blood pressure" → 画：白大褂 + 听诊器 = 医生
答案：doctor-patient

**快速记法：**
听到一个词，画一个符号，不用写字。
- 医院 → 🏥
- 餐厅 → 🍽️
- 学校 → 📚
- 天气 → 🌧️❄️☀️

## P5 第二招：画场景 · 地点题

**地点题不是靠背单词，是靠"画面感"**

每个场景都有专属的"颜色和形状"：

| 场景 | 关键画面元素 | 关键词 |
|------|------------|--------|
| 餐厅 🍽️ | 菜单、服务员、账单、刀叉 | order, menu, bill, waiter |
| 图书馆 📚 | 书架、借书、安静、到期 | borrow, return, fine, quiet |
| 机场 ✈️ | 登机牌、行李、候机、延误 | gate, luggage, delay, board |
| 医院 🏥 | 白大褂、体温计、药方 | fever, medicine, check-up |
| 学校 📖 | 教室、考试、作业、老师 | exam, assignment, lecture |

**记忆诀窍：**
不是背单词，是**画场景速写**——
闭上眼睛，听到"menu" → 脑子里快速闪过餐厅的画面 → 就知道在哪了

## P6 用颜色标态度 · 态度题

**态度题是美术生的主场——你们天生懂颜色语言**

| 态度 | 颜色 | 信号词 |
|------|------|--------|
| 积极 👍 | 🟢 绿色 | great / wonderful / love / perfect |
| 消极 👎 | 🔴 红色 | terrible / hate / not good / awful |
| 中立 ➖ | 🔵 蓝色 | well / actually / I guess / not sure |
| 委婉拒绝 ⚠️ | 🟡 黄色 | I'd love to, but... / I wish... / Sounds great, but... |

**黄金法则：**
听到"I'd love to, but..." → 前面是客气，**but后面涂红色**
听到"That sounds great, but..." → 前面是客气，**but后面涂红色**

> 红色的话才是真话。

## P7 画线条 · 数字题

**数字不是背的，是画的**

数字题的陷阱就两个：

**陷阱①：单价 vs 总价**
- "$15 each, $30 in total" → 画：一个圈=15 / 两个圈=30
- 问什么圈什么

**陷阱②：-teen vs -ty**
- fifTEEN（尾巴重）↔ FIFty（前面重）
- 画一条波浪线：高音=teen / 低音=ty

**画数字诀窍：**
听到数字先画下来，不要急着选。
- 时间 → ⏰ 画个钟
- 价格 → 💰 画个钱袋
- 数量 → 📦 画个箱子

## P8 画出信号词 · 长对话导航图

长对话像一幅**构图复杂的画**——你得知道焦点在哪。

信号词就是画面的"视觉焦点"：

| 信号类型 | 信号词 | 像画画里的什么 |
|---------|--------|--------------|
| 转折 | but, however, actually | 画面里的**对比色** |
| 建议 | why not, how about | 画面的**高光点** |
| 强调 | especially, the most | 画面的**视觉中心** |
| 原因 | because, so, therefore | 画面的**构图逻辑线** |

**示例：**
> W: I'm thinking about where to go for vacation.
> M: How about Sanya? 🌟（高光点）
> W: But it might be too crowded. ⚡（对比色）
> M: The thing is, our budget is tight. 🔴（视觉中心）

听到"the thing is" → 这就是整幅画的中心点。

## P9 别画蛇添足 · 考场时间管理

**核心原则：先画框架，再填细节**

**30秒预读法（拿到卷子的第一时间）：**
不要读题，要**扫画面**：
- 看选项有几个"地点词" → 这题考哪里
- 看选项有几个"数字词" → 这题考多少钱/几点
- 看选项有几个"态度词" → 这题考心情

**听的时候：**
- 第一遍：画骨架（谁、在哪、干什么）
- 第二遍：填细节（具体时间、具体价格）

**卡住了？**
不要盯着那题不放 → 就像画错了不要一直擦 → 翻页，跟住下一段

**作业①：画场景词典**
选10个听力高频场景，每个场景用**一幅简笔画 + 5个关键词**表示
不写字，画出来。
下次课带过来交换猜。

**作业②：画听力地图**
找一段长沙中考听力真题（3分钟以内），边听边画→
画出一幅"对话地图"：人物位置 → 移动方向 → 关键事件
不用管画得好不好，你自己看得懂就行。

**作业③：颜色笔记**
下次课前，准备好红黄绿蓝四支笔。
上课时听到积极词→画绿线，听到消极词→画红线，听到信号词→画蓝线。
一节课下来，看看你的笔记是什么颜色最多。

## 给老师的上课话术备忘

**这堂课的底层逻辑：**
- 美术生 = 视觉型学习者
- 不要跟他们讲语法逻辑，要讲画面逻辑
- 每个听力技巧对应一个"画画动作"：画人物 → 画场景 → 涂颜色 → 画线条

**课堂节奏：**
- 前5分钟：破冰 → "你觉得学英语跟画画像不像"
- 中间25分钟：讲技巧，每讲一个马上练一道真题
- 最后10分钟：一起做一套真题，让他们用"画画法"记笔记
- 布置作业（画场景词典）

**骚话可以讲：**
- "你要是能画出一个人，你就能听懂他在说什么。"
- "颜色你都分得清，态度你还分不清？"
- "构图你都会，信号词你还找不到？"
