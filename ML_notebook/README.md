## 已学内容

### KNN（K近邻算法）
- 概念：算距离→排序→取K个→投票
- 鸢尾花数据集实战（K=3, 100%准确率）
- **交叉验证**：5折交叉验证科学选K值（K=7最优：98%）
- **归一化**：MinMaxScaler让特征公平投票（提升25%+）
- 代码：`03_KNN交叉验证.py` `04_KNN归一化对比.py`

### 逻辑回归
- 名字带回归，本质是分类
- Sigmoid函数：把任意分数变成0~1概率
- 多分类：softmax → 每个类一个概率，加起来=1
- 可解释性强：能告诉你每个特征的重要性
- 代码：`05_逻辑回归入门.py` `02_逻辑回归多分类.py`

### 三算法对比
- 线性回归：预测具体数值（播放量多少）
- KNN回归：找相似视频取均值
- 逻辑回归：判断能不能破万
- 代码：`01_三个算法对比.py`
