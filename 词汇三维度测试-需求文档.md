# 项目：三维度词汇测试

## 一句话描述
一个专为初高中生设计的词汇深度测试网页——从**词汇量（广度）**、**读音**、**词性**、**用法**四个维度诊断学生的单词掌握情况。

---

## 技术栈
- 前端：HTML + CSS + JS（单页应用，纯前端）
- 数据存储：localStorage（保存测试记录 + 错词本）
- AI出题：可选（用灵眸API生成例句或干扰项）

---

## 核心测试逻辑（两层筛选法）

### 第一层：快速筛选（测词汇量广度）
- 随机从词库中抽取 **30个词**
- 每个词只问："你认识这个词吗？" → 认识 / 不认识
- 30个词按难度分布：
  - 基础 800 词中抽 8 个
  - 中考 1600 词中抽 10 个
  - 高考 3500 词中抽 8 个
  - 拔高 5000+ 词中抽 4 个
- 根据"认识"的个数估算词汇量区间

### 第二层：深度诊断（测四个维度）
- 从第一层"认识"的词中再抽 **10个**
- 每个词考4个方面：

| 维度 | 题型 | 示例 |
|------|------|------|
| 🔊 读音 | 从4个音标中选正确的 | present → /ˈpreznt/ 还是 /prɪˈzent/? |
| 🏷️ 词性 | 可多选：这个词可以作什么词性 | present: 名词✅ 动词✅ 形容词✅ 副词❌ |
| 🔧 用法搭配 | 选正确的短语搭配 | make a present ✅ / do a present ❌ |
| ✍️ 语境应用 | 选正确的句子 | 4个句子中哪个用对了这个词 |

- 按这四个维度分别计分，出分维度报告

## 页面 / UI 描述

### 首页
- 标题：📖 三维度词汇测试
- 副标题：词汇量 · 读音 · 词性 · 用法 全面诊断
- 两个按钮：
  - 🟢 "开始测试"（默认中考词库）
  - 🟡 "选择词库"（中考 / 高考）

### 测试页（第一层）
- 顶部：进度条（1/30）
- 中间：单词大号显示 + 音标
- 底部：两个按钮「认识 ✓」「不认识 ✗」
- 右上角：已认 X / 30

### 测试页（第二层）
- 每个词分4步（通过进度条或分步切换）：
  - Step 1/4：读音题（4选1）
  - Step 2/4：词性题（可多选）
  - Step 3/4：搭配题（4选1）
  - Step 4/4：句子题（4选1哪个句子对）
- 每步答完自动进下一步
- 4步全答完 → 进入下一个词

### 报告页
- 顶部大卡：估算词汇量 + 综合掌握度（百分比）
- 四维雷达图（读音 词性 用法 语境）
- 分段详解：
  - 读音弱 → 建议：多练音标，注意重音
  - 词性弱 → 建议：背单词时注意词性标记
  - 用法弱 → 建议：多记词组搭配
  - 语境弱 → 建议：在句子中学单词
- 错词清单：列出所有答错的词，点击可看详情
- 底部按钮：重新测试 / 复习错词

## 词库格式

```javascript
// 每个词的完整数据格式
{
  word: "present",
  level: "中考",           // 基础 / 中考 / 高考 / 拔高
  phonetics: [
    { pron: "/ˈpreznt/", pos: "n./adj." },
    { pron: "/prɪˈzent/", pos: "v." }
  ],
  pos: ["n.", "v.", "adj."],   // 正确词性
  posTrap: ["adv."],           // 陷阱词性
  collocations: [
    { phrase: "at present", meaning: "目前" },
    { phrase: "present sb. with sth.", meaning: "向某人颁发某物" }
  ],
  example: {
    correct: "He presented his findings at the meeting.",
    traps: [
      "She present her homework yesterday.",
      "I want to present you a gift.",
      "She is present with the award."
    ]
  },
  // 读音题干扰项
  phoneticsTraps: ["/ˈprezɪdənt/", "/ˈperənt/"]
}
```

## 示例数据（以 present 为例）

### 第一层显示
```
present  /ˈpreznt/ 或 /prɪˈzent/
[认识 ✓]  [不认识 ✗]
```

### 第二层（4步）

**Step 1 - 读音**
```
present 的正确音标是？
A. /ˈpreznt/         ← ✅ 正确（名词/形容词读音）
B. /ˈprezɪdənt/      ← ❌ 干扰
C. /prɪˈzent/        ← ✅ 也正确（动词读音）
D. /ˈperənt/         ← ❌ 干扰
答案：A 和 C（可多选）
```

**Step 2 - 词性**
```
present 可以作哪些词性？（可多选）
☐ 名词 (n.)  ← ✅
☐ 动词 (v.)  ← ✅
☐ 形容词 (adj.)  ← ✅
☐ 副词 (adv.)  ← ❌ 陷阱
```

**Step 3 - 搭配**
```
以下哪个是 present 的正确搭配？
A. do a present     ← ❌
B. make a present   ← ✅
C. take a present   ← ❌（少见）
D. give a present   ← ✅（也正确）
答案：B 和 D（可多选）
```

**Step 4 - 语境**
```
哪个句子中 present 用得正确？
A. She present her homework to the class yesterday.
   ← ❌ 时态错误，应为 presented
B. He presented his findings at the meeting last week.
   ← ✅
C. I want to present you a gift.
   ← ❌ 双宾语用法有误，应为 present sb. with sth.
D. She is present with the award.
   ← ❌ 搭配不当
```
