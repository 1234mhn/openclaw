## 🏋️ 控笔训练（5min）

**① 弧形三点 × 10组**
- 在纸上画三点，呈弧形（像⌣形状）
- 第一点在上，第二点稍偏左，第三点偏右
- 感受三个点在弧线上而非垂直一线的排列
- 为三点水的弧形位置做准备

**② 水滴形练习 × 10个**
- 轻入→渐按→渐提→出锋，一笔画出一个水滴形状
- 左上到右下方向练习
- 模拟三点水中每个点的"轻-重-提"节奏

## 🖌️ 偏旁练习

### 三点水（氵）— 写20遍

**口诀：** 弧形排列，上中下呼应

```
① 第一点（上）：靠近右边，短而轻
② 第二点（中）：稍靠左，与第一点呈弧线
③ 第三点（下）：又靠右变提，指向右上方
```

**🔑 三点水呈弧形（⌣），不是垂直一条线**

三点位置关系：
```
  •    ← 第一点（靠右）
•      ← 第二点（偏左）
  ↗   ← 第三点（挑向右上）
```

### 两点水（冫）— 写20遍

**口诀：** 上点下提，两点呼应

```
① 上点：轻入重顿，靠右
② 下提：起笔偏左，向右上提出锋
```

**🔑 两点间距适中，提画指向右上方**

## 📖 例字（各写4遍）

### 「江」

- 右边"工"字横平竖直
- 注意左右比例 — 三点水约占1/3

### 「河」
- 三点水+可字
- "可"字横的起笔在三点水第二点附近高度
- 口字扁，竖钩挺直

### 「海」
- 三点水+每字
- "每"字上横要长，盖住下方
- 母字部分笔画多，注意间距均匀

### 「冰」
- 两点水+水字
- 两点水要紧凑
- 水字撇捺舒展，注意与两点水的呼应

### 「冷」
- 两点水+令字
- "令"字人字头打开
- 下方两点与两点水呼应

### 「净」
- 两点水+争字
- "争"字上中下结构
- 竖钩要挺直有力

## 💡 要点提醒

1. **三点水不是三个点垂直排队** — 呈弧形才有味道
2. 三个点的 **第二点最靠左**，形成弧度
3. **第三点是提**，不是点 — 指向右边偏上
4. 两点水比三点水紧凑，上点重下提出锋
