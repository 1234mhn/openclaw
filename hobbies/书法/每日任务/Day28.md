## 🖌️ 偏旁练习

### 广字头（广）× 各4遍 / 例字

**结构要点：**
- 点：居中偏右（靠右！你言字旁的点也偏位，这次注意）
- 横：左低右高微斜
- 撇：长撇舒展，从横的左端起笔覆盖下部

**例字：** 「店」「底」「庆」

```
店 → 广字头盖住"占"，撇要长
底 → 广字头下"氏"勿太大
庆 → 广字头覆盖"大"，撇捺舒展
```

### 月字旁（月）× 各4遍 / 例字

**结构要点：**
- 竖撇：先直下再左撇出（今天重点！）
- 横折钩：折角方正、钩底驻锋
- 内部两横：靠左，间距均匀

**例字：** 「朋」「期」「背」

```
朋 → 左右月，左小右大，两月间距不要贴
期 → 左"其"右"月"，左右等宽
背 → 上"北"下"月"，上宽下窄
```

## 🔁 薄弱巩固追加

**竖撇** — 先竖后撇，弧度适中
- 追加例字：「千」「用」各写4遍
- 💡 竖写够长再左撇，别一上来就拐

**横折钩** — 折角方正，钩底驻锋
- 追加例字：「而」「同」各写4遍
- 💡 折角往回带0.5mm再下行，钩踢出干脆

## 💡 要点提醒

1. **广字头** — 撇要长、要覆盖，像一把伞撑开 ☂️
2. **月字旁** — 竖撇先直后弯，横折钩折角要方
3. **竖撇的核心** — 上半段是"竖"，下半段才是"撇"，别搞反
