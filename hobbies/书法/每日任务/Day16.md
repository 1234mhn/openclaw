## 🖌️ 笔画练习（30min）

### 横撇 × 15遍

**口诀：** 横是铺垫，撇是主体，转角一驻即走

```
起笔轻顿 → 右上行笔（短横）→ 折角稍驻 → 向左下撇出
                         ↕                    ↕
                     不要拖泥带水          锋送到尾
```

**🔑 节奏：** 横→（停）→撇，心里默念"1-2-送"

**⚠️ 你的薄弱环节回顾：**
> 上次Day11反馈：**横撇出锋太快导致发飘**
> 今天特训：每笔心里数"1-2"再送出锋，感受笔尖在纸上送到底再离开

### 横折 × 15遍

**口诀：** 折角方正，顿笔有力

```
起笔轻顿 → 右行 → 折角顿笔 → 下行（稍向内收）
   ↕              ↕              ↕
不要过重        方正          不是死直
```

**🔑 要点：**
- 折角要"方"不要"圆"
- 起笔像羽毛碰纸——**别砸下去！**（你的老毛病）
- 折角往回带约0.5mm，让转角有自然弧度（Day12教练建议）

## 📖 例字（各写4遍）

| 例字 | 要点 |
|------|------|
| **又** | 横撇是主笔，撇出舒展，与捺形成夹角 |
| **双** | 两个"又"左小右大，左窄右宽 |
| **口** | 上宽下窄，右上角略高，竖折折角有自然弧度 |
| **日** | 竖长形，三横等距，最后一横托住 |
| **目** | 比"日"更长，四横间距相等 |
| **田** | 上宽下窄，中间十字交叉居中 |

## ⚠️ 要点提醒

### 今日特别需要注意：

1. **🔴 起笔轻！起笔轻！起笔轻！** 
   > 这是贯穿至今的老毛病，横折起笔别像砸钉子
   > 想象笔尖像羽毛轻轻碰纸

2. **🔴 横撇出锋放慢节奏**
   > 心里数"1-2-送"，不要急着把笔甩出去
   > 空手模拟3遍折角节奏再落纸

3. **🟡 折角要方不要圆**
   > 横折的折角要有"咔"的顿感
   > 但不要过度用力变成死疙瘩

4. **🟡 写之前先看字帖3秒**
   > 默念结构比例再下笔
   > 不急那3秒，写歪了更费时间
