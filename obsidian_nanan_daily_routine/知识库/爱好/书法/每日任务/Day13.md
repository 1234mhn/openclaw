## 🏋️ 控笔训练（5min）

**① 锯齿撇折线 × 10组**
- 画连续「V」形锯齿：↘→↙→↘→↙
- 每个折角角度一致，像锯条齿
- 🔑 为撇折的折角转换热身

**② 横→提转换 × 10组**
- 先写一短横，到末端不停，直接向右上挑出
- 横平稳，提斜上45°，夹角干净利落
- 🔑 为提画的热身

**③ 横折慢写 × 5组（巩固）**
- 放慢速度写横折，注意起笔轻、折角自然弧
- 昨天的问题今天多加留意

## 🖌️ 新笔画：撇折（㇀）— 写20遍 ⭐

**口诀：** 撇→折→提，两段分明

```
① 起笔：撇起笔轻顿入纸
② 撇走：向左下撇出（约45°，跟普通撇一样）
③ 转折：到撇末端轻轻一按，折向上
④ 提出：从折处向右上提出（约45°），出锋干脆
```

**🔑 关键点：**
- 这是**一笔两段**：先撇后提，中间一折
- 撇段要干脆，不能拖
- 折角角度要明显，不能圆滑成弧线
- 提出方向：右上45°，干净利落

```
写法示意：

    ↘
     ↖      ← 撇→折→提
      ⬆
```

### 常见错误

```
❌ 撇折写成了弧线（分不出两段）
✅ 两段角度分明：左斜下→折→右斜上

❌ 撇段太长，提段太短
✅ 撇约占2/3，提约占1/3

❌ 折角太圆滑，像"弯钩"
✅ 折角要有"切面感"，像刀切一下
```

## 🖌️ 新笔画：提（㇀）— 写15遍 ⭐

**🔔 提不是新东西！** Day 6（提/横折）和 Day 10（提手旁）都练过，今天正式归入笔画补齐。

**口诀：** 左下起→右上出锋

```
① 起笔：左下方向，顿笔入纸
② 行笔：向右上中锋行笔，由重渐轻
③ 出锋：到右上末端，快速提出锋
```

**🔑 关键点：**
- 方向：**左下→右上**（约45°），别太平
- 由重到轻：起笔重、收笔尖
- 干净利落，不要拖泥带水

## 📖 例字（各写4遍）

### 「去」— 撇折 + 土
- 上横短，撇折开口大
- 下面"土"字上横短下横长
- 撇折的提段不能太陡

### 「云」— 撇折 + 二（加一点）
- 上横短，撇折覆盖
- 下方"云"撇折 + 点
- 整体上窄下宽

### 「玄」— 撇折 + 幺
- 上横短，下撇折
- 底部"幺"写紧凑
- 撇折的提段指向下方"幺"的中间

### 「虫」— 口 + 中竖 + 提 + 点
- 上部"口"扁
- 中竖垂直穿过口
- 提画在竖的中下部，向右上挑出
- 最后一点收尾

### 「打」— 提手旁巩固
- 提手旁写紧，右边"丁"舒展
- 注意提手旁的"横→竖钩→提"三笔顺序

### 「拉」— 提手旁 + 立
- 左边提手旁占1/3
- 右边"立"字上点居中
- 左右留白均匀

## 💡 要点提醒

1. **撇折是"一笔两段"** — 不是撇+折，是一笔写出两段方向
2. **撇折和提的区别：** 撇折是起笔撇后折上去；提是直接左下起笔右上挑
3. **折角要分明** — 撇折的折角是"切面"不是"弧线"
4. **提的方向右上45°** — 别太平（30°）也别太陡（60°）
