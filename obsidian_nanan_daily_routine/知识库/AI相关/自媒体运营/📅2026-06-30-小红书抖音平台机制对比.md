# 小红书 vs 抖音：平台机制深度拆解（2026版）

## 📌 为什么这两套逻辑要分开理解

同一篇内容，在两个平台的命运可能完全不同——一个爆了、一个沉了。核心原因在于：**推荐算法的底层逻辑不一样。**

---

## 🎵 抖音：从「标签匹配」到「用户行为预测」

### 2026年核心变化
- 完播率权重**下降**，用户长期价值权重**上升**
- 算法不再靠标签推内容，而是**给每条视频打分**，推"高分内容"
- 平台定向分配 **~20%流量给小众兴趣**，打破信息茧房
- AI 能精准识别视频实际内容（不只是看标题和标签）

### 权重优先级
1. 🔼 收藏 + 复访 → 做值得反复看的内容（攻略、干货、教程）
2. 🔼 关注 + 追更 → 引导用户长期关注
3. 🔼 搜索行为 → 探索用户潜在需求，跨赛道扩展

### 运营打法
- **别只追"完播率高"的短内容**，做有知识密度、能收藏的内容
- 封面打出"含金量"（"3000字干货，先收藏再看"）
- 适度跨赛道：做房产的也可以聊财经、家装

---

## 📕 小红书：CES评分 + 阶梯式流量池

### 2026年核心变化
- 星云5.0 算法升级（2026年5月发布）
- AI语义理解深度升级→**不再靠堆关键词引流**
- 新号与大V初始曝光平等→**内容质量是唯一通行证**
- 搜索流量占比大幅上升（约30%~60%）

### CES评分公式（2026新版）
```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```
**互动质量也被纳入：**
- 长评论（>15字）权重 > 短评
- 完读率（用户完整读完笔记）
- 截屏行为（被视为"极高价值"信号）
- 二次回访（收藏后又回来看）

### 流量池闯关地图
| 小眼睛范围 | 阶段 | 通关关键 |
|:----------:|:----:|:---------:|
| 0-200 | 冷启动池（前2小时） | 封面/标题/标签 |
| 200-500 | 初级流量池 | CTR达标 + CES互动 |
| 500-5000 | 中级流量池 | 互动率持续优秀 |
| 5000-1w+ | 热门预备池 | 人工审核介入 |
| 1w+ | 大爆款 | 大规模分发 |

### 运营打法
- **文字写够600字以上**，低质内容会被判违规
- **必须开原创声明**，拿到额外流量倾斜
- 用提问/争议点刺激评论（"圆脸适合这款发型吗？"）
- 标题+正文+话题保持高度一致，精准匹配标签

## ⚡ 核心差异总结

| 维度 | 抖音 | 小红书 |
|:----|:----|:-------|
| 推荐机制 | 行为预测 + 内容价值评分 | 标签匹配 + CES评分 |
| 核心指标 | 收藏+复访，关注+追更 | 评论4分，关注8分 |
| 流量形态 | 短视频为主，长尾弱 | 搜索+长尾，生命周期长 |
| 内容策略 | 短、有价值、可收藏 | 高质量原创+强互动引导 |
| 对新手 | 门槛偏高，需投流辅助 | 相对友好，内容为王 |
