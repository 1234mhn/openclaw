# 2026.7.5 每日一句

**주호진 씨 사진 안 찍어요**

## 单词

| 韩文 | 中文 |
|:----|:-----|
| 사진 | 照片 |
| 안 | 不（放在动词前面表否定） |
| 찍다 | 拍；찍어요是口语形式，사진 찍다 拍照 |

## 句型

**안 +动词어요?** 不...吗？

### 例句

1. 밥 안 먹어요? 不吃饭吗？
2. 집에 안 가요? 不回家吗？
3. 커피 안 마셔요? 不喝咖啡吗？
4. 영화 안 봐요? 不看电影吗？
5. 한국어 안 배워요? 不学韩语吗？

---

## 韩语否定3种

### ① 안 + 动词
**意思：** 不~（自己不想，主观）

| 怎么用 | 例子 |
|:------|:----|
| 안 + 动词 | 안 가요 不去 |
| | 안 먹어요 不吃 |
| | 안 찍어요 不拍 |

### ② 못 + 动词
**意思：** 不能~（没法做，客观上）

| 怎么用 | 例子 |
|:------|:----|
| 못 + 动词 | 못 가요 去不了 |
| | 못 먹어요 吃不了 |
| | 못 찍어요 拍不了 |

### ③ ~지 마(요)
**意思：** 别~（叫别人别做）

| 怎么用 | 例子 |
|:------|:----|
| 动词词干 + 지 마요 | 가지 마요 别去 |
| | 먹지 마요 别吃 |
| | 찍지 마요 别拍 |

### 一句话区别

| | 안 | 못 | ~지 마 |
|:--|:---|:---|:-------|
| **意思** | 不做 | 不能做 | 别做 |
| **对谁说** | 对自己/陈述 | 对自己/陈述 | 对别人说 |
| **例句** | 안 먹어요（不吃） | 못 먹어요（不能吃） | 먹지 마요（别吃） |
