## 分步说明

### ① 你发我

```
┌──────────────────────────────┐
│ 课号：第几课                    │
│ 视频开始时间：00:05            │
│ 视频结束时间：00:55            │
│ 台词内容（如有）               │
└──────────────────────────────┘
```

**示例：**
> 「楠楠，加第13课，视频00:05～00:55」

台词可以一起发，也可以我后面补。

### ② COS上截取视频（全程不下本地）

| 步骤 | 操作 | 用到的技能 |
|:---|:---|:---|
| 1 | 用COS CI媒体处理，对源视频提交截取任务 | `tencent-cos-skill` |
| 2 | 指定开始时间、结束时间、输出格式 H264+AAC 1080p | `tencent-cos-skill` |
| 3 | 截取完成的视频片段直接存回COS（scene13_xxx.mp4） | `tencent-cos-skill` |
| 4 | 获取COS访问链接 | `tencent-cos-skill` |

**整个过程中视频不下本地，全在COS上完成。**

### ③ 页面挂COS链接

页面中第X课的 `video` 路径直接填COS链接，播放时从COS加载。

### ④ 按模板填充页面

**⚠️ 原则：首页模板不动，课程内容页结构不动，只填新内容。**

根据 `课程模板说明.md` 的现有规范，在对应位置填入新数据：

| 填入位置 | 填入内容 |
|:---|:---|
| 首页课程卡片 | 新课卡片（照抄已有卡片格式） |
| LESSONS 数据 | 课名、视频=COS链接、单词表 |
| 台词区 | 每条台词6件套（📍单词→💬短语→💬造句→📌语法→💬语法例句→🎵音变） |
| 单词区 + 小贴士 | 照抄已有格式 |
| LESSON_TIMES | 时间戳数组 |

**不改变：**
- ❌ 首页卡片样式（class、结构、颜色）
- ❌ 课程内容页布局（顶部导航、视频区、台词区、展开讲解样式）
- ❌ 功能按钮（测验、已完成、笔记、口语练习）

**🔴 核心要求：台词和视频必须一一对应，严丝合缝**

| 要求 | 说明 |
|:---|:---|
| 台词顺序 | 台词讲解的排列顺序 = 视频中对话出现的先后顺序 |
| 台词内容 | 每句台词必须与视频中人物说的完全一致，不能多词不能少词 |
| 时间戳 | 每条台词的 `data-time-start` 和 `data-time-end` 精准对应视频中说话的开始和结束时间 |
| 点击联动 | 点击下方任意一句台词 → 视频跳转到该句对应时间播放 |
| 播放联动 | 视频播放到某句时 → 下方对应台词自动高亮 |

> ⚡ **检查方法：** 写完后按顺序点一遍每句台词，看视频是否跳到对应的画面和对话。有一句对不上就是不合格。

整理好后 → 备份 → 交给 Claude Code 注入。
