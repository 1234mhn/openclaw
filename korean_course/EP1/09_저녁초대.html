단어

지금부터 뭐 할 겁니까? 뭐 할 일 있습니까?

→ 接下来要做什么？有什么要做的事吗？

문법~ㄹ 겁니까?
→ 会做……吗？/ 打算……吗？(正式)
~ㄹ 거예요? 的正式形 ~ㄹ 겁니까?
💬 어디 갈 겁니까? → 您要去哪里？
문법~ㄹ 일
→ 要做的事
动词 ~(으)ㄹ + 일(事) 修饰名词。"할 일" = 要做的事
💬 할 일이 많아요. → 要做的事很多。
🎵 音变说明
› 지금부터 → [지금부터] (清晰)

괜찮다면 저랑 같이 저녁 드실래요?

→ 方便的话和我一起吃晚饭吗？

동사드시다
→ 吃/喝（敬语）
먹다/마시다 → 드시다 (尊敬语)。"드세요" = 请吃/请喝
💬 뭐 드실래요? → 您要吃什么？
문법~ㄹ래요?
→ ……吗？(询问对方意愿)
用于询问对方"愿不愿意"。比 ~ㄹ까요?(咱要不要) 更聚焦对方
💬 커피 마실래요? → 喝咖啡吗？
문법~다면
→ 如果……的话
~다고 하면 缩略，假设。"괜찮다면" = 如果方便的话

아까는 싫다더니?

→ 刚才不是说不要吗？

부사아까
→ 刚才
💬 아까 뭐라고 했어요? → 刚才说什么了？
문법~다더니?
→ 不是说……吗？（反问/对比）
~다고 하더니 缩略。指出"前后不一致"
💬 안 온다더니 왔네? → 不是说不来吗，怎么来了？

굳이 같이 먹을 이유가 없다는 거였지

→ 我说的是没必要一起吃

부사굳이
→ 非要、特意
韩国年轻人爆款词。"굳이?" = "干嘛非要这样？"
💬 굳이 가야 돼? → 非要去吗？
💬 굳이 그럴 필요 있어? → 非那样不可吗？
명사이유
→ 理由
汉字 理由
문법~다는 거
→ 所谓……（前面整句作名词）
~다고 하는 것 = 所说的事。把陈述句变成名词
💬 안 간다는 거예요? → 是说不去吗？
🎵 音变说明
› 굳이 → [구지] (口蓋音化: ㄷ+ㅣ→ㅈ)

두 사람 자리라는데 혼자 먹으면 아깝잖아요

→ 听说是两人座，一个人吃太可惜了

명사자리
→ 座位、位子；位置
💬 자리 있어요? → 有座吗？
형용사아깝다
→ 可惜、舍不得
💬 시간이 아까워요. → 时间可惜。
💬 너무 아까워! → 太可惜了！
문법~잖아요
→ 不是……嘛（提醒已知事实）
~지 않아요? 的口语缩略。带"你也知道吧"的语气
💬 그건 비싸잖아. → 那个不是很贵嘛。
문법~ㄴ다는데
→ 听说……（陈述）
~ㄴ다고 하는데 缩略，引用+继续

그럼 내가 같이 가서 먹어 줄게요. 신세 진 것도 있는데

→ 那我陪你去吃。我也欠你人情

명사신세
→ 人情、关照
汉字 身世。常用搭配 신세를 지다(欠人情) ／ 신세를 갚다(还人情)
💬 신세 졌어요. → 受您照顾了。
표현먹어 주다
→ (帮你/给你面子)吃
"~아/어 주다" 在这里带"勉强答应/给面子"语气
동사지다 (신세를 지다)
→ 欠下、承受（人情/责任）

왜 신세를 갚는 것처럼 생색을 내지?

→ 怎么像还人情似的还邀功？

동사갚다
→ 偿还（钱/恩情）
💬 돈을 갚아요. → 还钱。
💬 은혜를 갚을게요. → 我会报恩。
명사생색
→ 邀功、自夸
汉字 生色。常用 생색을 내다(邀功显摆)
💬 생색내지 마. → 别邀功。
문법~처럼
→ 像……一样
💬 새처럼 날아요. → 像鸟一样飞。

암만 맛있는 것도 혼자 먹으면 맛이 없고 많이 안 들어가요

→ 再好吃的东西一个人吃也没味道，吃不下

부사암만
→ 再……、不管多……（口语）
아무리 的口语缩略
💬 암만 봐도 모르겠어. → 怎么看都不明白。
표현맛이 있다 / 없다
→ 有/没有味道
맛있다 (粘合) → 맛이 있다 (分开强调)
관용(음식이) 안 들어가다
→ 吃不下（食物不进口）
字面"东西进不去(嘴/胃)"
💬 슬퍼서 밥이 안 들어가요. → 难过得吃不下。
📌 单词·语法 总览

📍 单词

▸ 드시다 — 吃/喝(敬语)

▸ 굳이 — 非要、特意

▸ 이유 — 理由

▸ 자리 — 座位

▸ 아깝다 — 可惜

▸ 신세를 지다/갚다 — 欠/还人情

▸ 생색 — 邀功

▸ 암만 — 不管多么(口语)

📍 核心语法

▸ ~ㄹ래요? — 想……吗？

💬 영화 볼래요? → 想看电影吗？

▸ ~다더니 — 不是说……吗

💬 안 먹는다더니 다 먹었네. → 说不吃，结果都吃完了。

▸ ~잖아요 — 不是……嘛

💬 우리 친구잖아. → 我们不是朋友嘛。

▸ ~처럼 — 像……一样

💬 너처럼 똑똑한 사람. → 像你一样聪明的人。

💡 小贴士

① 굳이 是当下韩国最 hot 的反问词。"굳이?" 单独用，意思是"干嘛非要这样？"。

② 韩国人吃饭文化注重 "혼자 먹기 NG" — 一个人吃没仪式感。这是邀请的常用借口。

③ 신세를 지다 / 갚다 是关键短语，韩国职场/人情往来必备。

④ 호진 用拐弯抹角的方式邀请 무희 吃饭("两人座可惜了")，是典型韩国男主"嘴硬心软"。
