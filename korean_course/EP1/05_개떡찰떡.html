늙은이 쫓아다니면서 통역하느라 수고 많았어

→ 跟着我这老头子翻译辛苦了

명사늙은이
→ 老头子（自嘲/略带贬义）
늙다(老) + 은(冠形) + 이(人)。委婉敬称: 어르신
동사쫓아다니다
→ 跟着、追着跑
쫓다(追) + 아 + 다니다(往返)
💬 강아지가 나를 쫓아다녀요. → 小狗跟着我。
문법~느라(고)
→ 因为做……（消耗时间/精力）
表负面后果常用："因为忙做A，所以B(没做/没能)"
💬 공부하느라 잠을 못 잤어요. → 学习累得没睡。
표현수고 많았어요
→ 辛苦了
敬语: 수고하셨습니다 ／ 平等: 수고했어 ／ 注意: 수고하세요 不能对长辈说！
🎵 音变说明
› 늙은이 → [늘그니] (双终声+连音化)

개떡같이 말해도 찰떡같이 통역해 주는 거는

→ 我说得乱七八糟你也能翻得明明白白

속담개떡같이 말해도 찰떡같이 알아듣다
→ 别人说得不清不楚也能完全理解（=韩国超高频谚语）
개떡(粗糙糙的米糕，粗糙混乱) ↔ 찰떡(粘糕，紧实漂亮)。"乱说也准懂"
💬 우리 엄마는 개떡같이 말해도 찰떡같이 알아들어요. → 我妈即使我说得乱七八糟也能完全听懂。
유사 속담관련 음식 비유
▸ 떡 줄 사람은 생각도 안 하는데 김칫국부터 마신다 = 还没给糕，自己先喝泡菜汤(自作多情)
▸ 그림의 떡 = 画中之饼(可望不可即)
▸ 누워서 떡 먹기 = 躺着吃糕(易如反掌)
🎵 音变说明
› 개떡같이 → [개떡까치] (紧音化+ㅌ→ㅊ口蓋音化)
› 찰떡같이 → [찰떡까치] (同上)

스페인 가는 거를 좀 미루지?

→ 去西班牙的事推迟一下吧？

동사미루다
→ 推迟、拖延
💬 회의를 미뤘어요. → 把会议推迟了。
💬 일을 미루지 마. → 别拖延工作。
문법~지?
→ 不是……吗？／……吧？（提议/确认）
尾音上扬+轻语气=委婉提议。"미루지?" = "推一下不行吗？"
🎵 音变说明
› 가는 거를 → [가는거를] (口语中常说 "가는거 좀")

기어이 일본까지 끌고 와 써먹으셨으면 됐지

→ 硬把人拽到日本来用都用了

부사기어이
→ 硬要、终究、非要
💬 기어이 가야겠어? → 非要去吗？
동사끌고 오다
→ 拽过来、带过来
끌다(拉) + 고 + 오다(来)
동사써먹다
→ 用上、利用上(略含负面)
쓰다(用) + 어 + 먹다(掉)。"用掉、消耗"
💬 그 지식을 써먹을 데가 없어요. → 那知识没地方用。
문법~(으)면 됐지
→ ……就行了/就够了吧（带反讽/无奈语气）
💬 그만하면 됐지. → 那样就够了吧。

그거 너무 뻔뻔하신 거 아닙니까?

→ 那也太厚脸皮了吧？

형용사뻔뻔하다
→ 厚脸皮、不知羞耻
💬 정말 뻔뻔하다. → 真厚脸皮。
💬 뻔뻔한 사람이에요. → 厚脸皮的人。
유의어관련 형용사
▸ 부끄러움이 없다 = 不知羞
▸ 낯이 두껍다 = 脸皮厚（直译）
▸ 철판을 깔다 = 像铺铁板（一样厚脸）
문법~신 거 아닙니까?
→ 不是太……了吗？（敬语反问）
~시 + ㄴ 거 = 您是……的吗。带轻微抗议语气。

술 줄이세요

→ 少喝点酒

동사줄이다
→ 减少、缩小
줄다(减少) → 줄이다(使减少)。사동사
💬 살을 줄이고 싶어요. → 想减肥。
💬 소리를 줄여 주세요. → 把声音调小。
관용술 줄이다 / 끊다
→ 减少喝酒 / 戒酒
💬 담배 끊었어요. → 戒烟了。

일본 온 김에 가 볼 데가 있어서요

→ 趁来日本想去个地方

문법~(으)ㄴ 김에
→ 趁……的机会、顺便……
~(으)ㄴ 김(势头/机会) + 에。"既然已经……不如再……"
💬 시장 간 김에 우유도 사 와. → 既然去市场了顺便买点牛奶。
💬 일어난 김에 운동할까? → 既然起来了不如运动？
문법~아/어 보다
→ 试着……、~过
"가 보다" = 去看看 / 去过
대명사데
→ 地方
"가 볼 데" = 想去看看的地方。곳 의 구어체

굼벵이도 구르는 재주가 있다더니

→ 都说蛴螬也有滚动的本事

속담굼벵이도 구르는 재주가 있다
→ 即使是看似无能的人也有自己的本事(俗语)
굼벵이 = 蛴螬（看似笨重的虫子）／ 구르다 = 滚 ／ 재주 = 本事。是夸赞性俗语。
문법~다더니
→ 都说……(果然如此)
~다고 하더니 缩略。引用过去听过的话+(果然/没想到)。
💬 한국 김치 맛있다더니, 정말 맛있어요! → 都说韩国泡菜好吃，果然！
🎵 音变说明
› 재주가 → [재주가] (清晰)
› 있다더니 → [읻따더니] (紧音化)
📌 单词·语法 总览

📍 单词

▸ 미루다 — 推迟

▸ 기어이 — 硬要

▸ 써먹다 — 用掉

▸ 뻔뻔하다 — 厚脸皮

▸ 줄이다 — 减少

▸ 굼벵이 — 蛴螬

▸ 재주 — 本事

📍 韩式俗语·谚语

▸ 개떡같이 말해도 찰떡같이 알아듣다 — 说得糙听得准

▸ 굼벵이도 구르는 재주가 있다 — 蛴螬也有看家本事

▸ 그림의 떡 — 画饼充饥

▸ 누워서 떡 먹기 — 易如反掌

📍 核心语法

▸ ~느라(고) — 因忙于……所以

💬 일하느라 못 갔어요. → 忙工作没去成。

▸ ~(으)ㄴ 김에 — 趁……顺便

💬 청소하는 김에 빨래도 했어. → 打扫顺便洗了衣服。

▸ ~다더니 — 都说……果然

💬 일본은 비싸다더니 정말 비싸. → 都说日本贵真贵。

💡 小贴士

① 韩国谚语爱用떡(年糕)。떡 是韩国传统美食，引申"美好事物"或"生活百态"。

② 注意 수고하세요 ❌ 不能对长辈/上司说！应该说 수고하셨습니다 或 고생하셨어요。

③ ~느라 不能与"积极结果"搭配：✗ 공부하느라 합격했어요 ✓ 공부해서 합격했어요

④ ~(으)ㄴ 김에 表达 "既然……不如顺便……"，超高频生活用语。
