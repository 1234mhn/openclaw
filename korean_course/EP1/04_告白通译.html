단어

지금 뭐라고 한 거예요?

→ 刚才说什么了？

문법~라고 하다
→ 说……(间接引语)
陈述句间接引语：动词어요/예요 → ~다고 하다 ／ 名词 → ~(이)라고 하다
💬 학생이라고 했어요. → 说是学生。
문법~ㄴ/은 거예요?
→ 是……的吗？（确认事实）
~ㄴ 것이에요? 的口语形。比 ~었어요? 更带追问意味。
💬 어디 간 거예요? → 你去哪儿了？
🎵 音变说明
› 뭐라고 한 거예요 → [뭐라고한거에요] (流畅连读)

지금 인사하는 거예요?

→ 现在是在打招呼吗？

명사인사
→ 问候、招呼
汉字词 人事。常见搭配：인사하다(打招呼) / 작별 인사(告别) / 첫인사(初次问候)
문법~는 거예요?
→ 在做……吗？（询问当下行为意图）
动词 ~는 것 + 이에요 = 做……这件事
💬 뭘 보는 거예요? → 在看什么呢？
🎵 音变说明
› 인사하는 → [인사하는] (清晰)

주호진 씨 통역해요

→ 朱浩进先生，请翻译

명사통역
→ 口译
통역(口译) ↔ 번역(笔译)。동사: 통역하다, 통역해 주다
호칭~씨
→ ……先生／女士（中性礼貌称呼）
名字 + 씨。比 님(尊敬) 平和，比 야/아(亲昵) 礼貌。
💬 김민수 씨, 안녕하세요. → 金敏洙先生您好。
문법~해요
→ 既可陈述也可命令(口语命令)
语调上扬=疑问 / 平=陈述 / 降调=命令(轻命令)

당신이 말을 전해 줘야 내가 답을 하지

→ 你得把话传过来，我才能回答呀

동사전하다
→ 传达、转达
💬 안부를 전해 주세요. → 请代我问好。
문법~아/어야
→ 必须……才能……（条件）
필요조건 = 必要条件。后续句意"才能/才会"。
💬 일찍 자야 안 늦어요. → 早睡才不会迟到。
문법~지
→ ……呀／嘛（自然结论语气）
~지 결句尾，常表"理所当然""明显的事"
💬 그렇게 해야 되지. → 当然得那样啊。
🎵 音变说明
› 줘야 → [줘야] (자연 발음)

준비한 작별 인사는 하지 않아도 되겠네요

→ 看来准备好的告别问候用不上了

명사작별 인사
→ 告别、道别
作别人事 = 분별 인사 / 마지막 인사
문법~지 않아도 되다
→ 不用……也行
~지 않다(不) + ~아/어도 되다(也行)
💬 안 와도 돼요. → 不来也没关系。
문법~겠네요
→ 看来……呢（推测＋感叹）
겠 (推测) + 네요 (感叹)。表"看样子是这样"
💬 비가 오겠네요. → 看来要下雨。

통역해 드리겠습니다

→ 我来帮您翻译

敬语~아/어 드리다
→ ~아/어 주다 的尊敬形
주다(给) → 드리다(奉献给)。对上位者的"为您做某事"
💬 가르쳐 드릴게요. → 我来教您。
💬 도와 드리겠습니다. → 我来帮您。
문법~겠습니다
→ ……(将要)、(我会……)
表意志 + 礼貌。공식적인 말투（正式场合）。
💬 잘 부탁드리겠습니다. → 请多指教。
🎵 音变说明
› 드리겠습니다 → [드리겓씀니다] (紧音化+鼻音化)

나는 진심으로 차무희라는 사람이 좋습니다

→ 我真心喜欢车武熙这个人

명사진심
→ 真心
진심으로 = 真心地。반대 = 거짓(虚假)
💬 진심이에요. → 是真心的。
문법~(이)라는
→ 叫做……的；称为……的
~(이)라고 하는 的缩略。后接名词，定义/介绍。
💬 김치라는 음식 알아요? → 知道叫泡菜的食物吗？
💬 사랑이라는 게 뭐예요? → 所谓爱情是什么？
문법이 좋다 / 을 좋아하다
→ "喜欢"两种说法
N이/가 좋다 = N(对我而言)是好的(感觉) ／ N을/를 좋아하다 = 喜欢N(动作)
💬 너가 좋아. = 너를 좋아해. → 喜欢你。
🎵 音变说明
› 좋습니다 → [조씀니다] (ㅎ脱落+紧音化+鼻音化)

제대로 전달이 됐습니까?

→ 准确传达到了吗？

부사제대로
→ 好好地、正确地
💬 제대로 좀 해 봐. → 好好做。
명사전달
→ 传达、传递
汉字词 传达。동사: 전달하다 / 전달되다(被传达)
💬 메시지를 전달했어요. → 传达了消息。
문법~이/가 되다
→ 成为……、变得……
전달이 되다 = 被传达。被动 1 - 名词 + 이/가 되다
🎵 音变说明
› 됐습니까 → [됟씀니까] (紧音化+鼻音化)
📌 单词·语法 总览

📍 单词

▸ 통역 — 口译

▸ 진심 — 真心

▸ 작별 인사 — 告别问候

▸ 전하다 / 전달하다 — 传达

▸ 제대로 — 好好地

📍 核心语法

▸ ~(이)라는 — 叫做……的

💬 차무희라는 사람 → 叫车武熙的人

▸ ~아/어 드리다 — 为您……（敬语）

💬 안내해 드리겠습니다. → 我来为您引导。

▸ ~아/어야 — 必须……才能

💬 먹어야 살죠. → 得吃才能活啊。

▸ ~지 않아도 되다 — 不用……也行

💬 가지 않아도 돼요. → 不去也行。

▸ ~겠네요 — 看来……呢

💬 늦겠네요. → 看来要迟到了。

💡 小贴士

① 韩语两种"喜欢": 좋다(主+이/가) = (我)觉得好 ／ 좋아하다(목적어+을/를) = 喜欢(动作)。

② 씨 vs 님：씨 是中性礼貌(平辈/陌生)，님 是尊敬(长辈/上司/网络昵称)。

③ ~아/어 드리다 是 ~아/어 주다 的敬语，对长辈/客户必用。

④ 这场告白巧妙点：히로的日语→호진的韩语→무희听到。三层语言，男主把握着"传话权"。
