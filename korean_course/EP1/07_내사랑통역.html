회화

남의 치정 싸움을 통역하고 싶지는 않습니다

→ 我不想替别人翻译三角恋的争吵

명사남
→ 别人、他人
💬 남의 일에 끼어들지 마. → 别管别人的事。
명사치정
→ 痴情/感情纠纷
汉字 痴情。常见: 치정 싸움 = 三角恋纠纷 ／ 치정극 = 情感剧
명사싸움
→ 争吵、打架
动词 싸우다 → 名词 싸움
문법~고 싶지(는) 않다
→ (倒是)不想……
~고 싶다(想) + 否定 + 주제 ~는 = 强调"恰恰不想"
💬 가고 싶지는 않아. → 倒是不想去。

파파고로 까이면 너무 비참할 거 같아서 그래요

→ 用Papago被甩太悲惨了，所以我才……

슬랭까이다
→ 被甩、被拒绝（口语）
까다(剥/损) → 까이다(被剥/被损)。年轻人常用："고백 까였어" = 告白被甩
💬 어제 까였어요. → 昨天被甩了。
형용사비참하다
→ 悲惨
汉字 悲惨。同义 = 처참하다(更书面)
💬 비참한 결말이었어요. → 是悲惨的结局。
문법~ㄹ 거 같아서 그래요
→ (我)是觉得会……才这样
~ㄹ 거 같다(看起来会) + ~아서(因为) + 그래요(才那样)

일방적으로 헤어지자고 통보해 놓고 사라졌어

→ 单方面通知分手然后消失了

부사일방적으로
→ 单方面地
일방(一方) + 적(的) + 으로(地)
동사헤어지다
→ 分手、分别
💬 어제 남친이랑 헤어졌어요. → 昨天和男朋友分手了。
동사통보하다
→ 通知（通常单方面/正式）
通報하다。비교: 알리다(告知) 较中性
동사사라지다
→ 消失
💬 갑자기 사라졌어요. → 突然消失了。
문법~자고 하다
→ 提议……（间接引语）
"헤어지자" 直接说 → "헤어지자고 했어" 间接转述
문법~아/어 놓고
→ 做完……还……（带不满/反差语气）
💬 일은 안 해 놓고 큰소리치네. → 活儿没干还嘴硬。

전에도 그런 적 있었으니까 이번에도 그러려니 기다렸고

→ 之前也有过这种情况，这次也以为他会回来就等了

표현그러려니 (하다)
→ 想着会那样、当作如此
"应该会那样吧" 自我安慰式接受。고급 표현！
💬 그러려니 하고 넘어갔어. → 当作那样就过去了。
문법~(으)니까
→ 因为……（理由，主观/经验）
~아/어서 强调客观因果，~(으)니까 主观＋后接命令/建议
부사전에
→ 以前、之前
💬 전에 만난 적 있어요. → 以前见过。

구차하고 찌질해도 내 것도 사랑이에요

→ 就算寒酸又窝囊，我的也是爱情

형용사구차하다
→ 寒酸、苦涩、勉强
💬 구차한 변명 그만해. → 别再苦涩地辩解了。
슬랭찌질하다
→ 窝囊、不上台面（年轻人词）
💬 정말 찌질하다. → 真是窝囊。
💬 찌질남 = 窝囊男
문법~아/어도
→ 即使……也……
💬 비싸도 살 거예요. → 再贵也要买。

이러고 끝내고 싶지는 않다고요

→ 我不想这样收场

동사끝내다
→ 结束（使……结束）
끝나다(自动结束) → 끝내다(他动使其结束)。사동사
표현이러고
→ 像这样、就这么
이렇게 하고 缩略。"这样下去"的口语形
문법~다고요
→ ……呢/我说……（强调/重复）
~다고 + 요(尊敬) = "我说过……" 强调情感
💬 사랑한다고요! → 我说我爱你呢！

최소한의 품위는 지키면서 끝낼 수 있게

→ 至少能保留最起码的尊严收场

명사최소한(의)
→ 最起码、最低限度
汉字 最少限。반대 = 최대한(最大限)
명사품위
→ 品位、尊严、体面
汉字 品位。常见搭配 품위를 지키다(保持尊严)
💬 품위 있는 사람 = 有品位/有教养的人
동사지키다
→ 守护、遵守
💬 약속을 지켜 주세요. → 请遵守承诺。
문법~(으)ㄹ 수 있게
→ 为了能够……（目的）
~(으)ㄹ 수 있다 + 게(为了)

내 사랑도 통역해 주세요. 부탁해요

→ 也请翻译我的爱情。拜托你了

동사부탁하다
→ 拜托、请求
💬 잘 부탁드립니다. → 请多多关照。(初次见面常用)
💬 한 가지만 부탁할게요. → 拜托一件事。
표현本剧主题句
→ 这是 EP1 的高潮台词，也是剧名"이 사랑 어떻게 통역해"的来源。
爱情通常无法翻译，但 무희 让 호진"翻译我的爱"。
🎵 音变说明
› 통역해 주세요 → [통여캐주세요] (激音化)
› 부탁해요 → [부타캐요] (激音化)
📌 单词·语法 总览

📍 单词

▸ 치정 싸움 — 三角恋纠纷

▸ 까이다 — 被甩(俗)

▸ 비참하다 — 悲惨

▸ 일방적 — 单方面

▸ 헤어지다 — 分手

▸ 사라지다 — 消失

▸ 구차하다 — 寒酸

▸ 찌질하다 — 窝囊

▸ 품위 — 品位/尊严

▸ 부탁하다 — 拜托

📍 核心语法

▸ ~자고 하다 — 提议……（间接引语）

💬 같이 가자고 했어요. → 提议一起去。

▸ ~아/어 놓고 — 做完了还……（反差）

💬 약속해 놓고 안 왔어. → 答应了还没来。

▸ ~다고요 — 我说……（强调）

💬 안 한다고요! → 我说不做了！

▸ ~아/어도 — 即使……也

💬 힘들어도 해야지. → 再累也得做。

▸ ~(으)ㄹ 수 있게 — 为了能……

💬 잘 잘 수 있게 도와줘. → 为了能睡好帮我。

💡 小贴士

① 찌질하다 是年轻人高频词，不仅指"窝囊"也包含"心眼小、抠门、纠缠"的负面感。

② 까이다(被甩) 来自动词 까다(剥)。"까이다" 在年轻人口语很常用，但不要在正式场合用。

③ 그러려니 是高级表达"看开一点／自我安慰", 学会能瞬间提升口语层次。

④ 这一段是剧情转折：무희 第一次表现出弱点和恳求，호진 也答应"翻译她的爱情"。
