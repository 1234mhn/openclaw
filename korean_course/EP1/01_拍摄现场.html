단어문법

이런 데인 줄 알았으면 운동화를 신으라고 했어야지

→ 早知道是这种地方，就该让我穿运动鞋

단어이런 데
→ 这种地方（데 = 곳，口语化）
💬 이런 데 자주 와요? → 你经常来这种地方吗？
💬 이런 데 처음이에요. → 这种地方我是第一次来。
단어운동화 / 신다
→ 运动鞋 / 穿（鞋袜类）
신다 = 穿鞋袜 / 입다 = 穿衣 / 쓰다 = 戴帽戴眼镜 / 끼다 = 戴戒指手套 / 차다 = 戴手表
💬 운동화 신는 게 편해요. → 穿运动鞋舒服。
문법~ㄴ/은 줄 알았으면
→ 早知道是……的话（事后假设，常带后悔）
~ㄴ/은 줄 알다(以为是…) + ~았/었으면(如果…的话)
💬 비 올 줄 알았으면 우산을 가져왔지. → 早知道下雨就把伞带来了。
💬 이렇게 어려운 줄 알았으면 안 했어. → 早知道这么难就不做了。
문법~(으)라고 하다
→ 让……做（命令的间接引语）
신다 → 신으라고 하다(让穿) / 가다 → 가라고 하다(让去)
💬 엄마가 약을 먹으라고 했어요. → 妈妈让我吃药。
💬 선생님이 일찍 오라고 했어요. → 老师让我早点来。
문법~았/었어야지
→ 本该……才对（埋怨/后悔语气）
💬 일찍 일어났어야지! → 就该早点起来啊！
💬 미리 말했어야지. → 就该提前说嘛。
🎵 音变说明
› 줄 알았으면 → [주 라라쓰면] (连音化：ㄹ 받침 + ㅇ → 顺移)
› 신으라고 → [시느라고] (连音化：ㄴ 받침 + ㅇ → 顺移)
› 했어야지 → [해써야지] (连音化：ㅆ 받침 + ㅇ → 顺移到下一音节)

아, 잔뜩 차려입으라더니 이게 뭐야

→ 啊，让我盛装打扮，结果是这种场面？

단어잔뜩
→ 满满地、狠狠地（程度副词）
💬 잔뜩 먹었어요. → 吃得饱饱的。
💬 화가 잔뜩 났어요. → 气得不行。
단어차려입다
→ 盛装打扮、穿戴整齐
차리다(整理/准备) + 입다(穿) → 把衣服讲究地穿好
💬 오늘 뭐 이렇게 차려입었어? → 今天怎么穿这么正式？
💬 정장으로 잔뜩 차려입었어요. → 穿了一身西装盛装。
문법~(으)라더니
→ 之前让……，结果却……（前后矛盾/对比）
~(으)라고 하더니 的缩略
💬 일찍 오라더니 본인은 늦었네. → 让我早点来，自己反倒迟到了。
💬 조용히 하라더니 자기가 더 시끄럽네. → 让别人安静，自己更吵。
회화이게 뭐야
→ 这算什么 / 这是什么啊（不满/质疑语气）
💬 이게 다 뭐야? → 这都是些什么啊？
🎵 音变说明
› 잔뜩 → [잔뜩] (词中本来就是紧音 ㄸ)
› 차려입으라더니 → [차려이브라더니] (连音化：ㅂ 받침 + ㅇ → 顺移)
› 뭐야 → [뭐야] (口语自然连读)

아, 다리 아파

→ 啊，腿好疼

단어다리
→ 腿（注意：다리 也有"桥"的意思，靠语境分辨）
💬 다리가 길어요. → 腿很长。
💬 다리가 저려요. → 腿麻了。
단어아프다
→ 疼、痛、生病（으脱落不规则）
아프다 + 아 → 아파（으脱落） / 아프다 + 았어요 → 아팠어요
💬 머리가 아파요. → 头疼。
💬 어디가 아프세요? → 您哪里不舒服？
회화다리 아파
→ 腿疼（口语省略主格 가，半语反映心情）
敬语：다리가 아파요 → 半语口语：다리 아파
🎵 音变说明
› 아파 → 아프다 + 아 = 아파（으脱落不规则，词干末尾 으 + 아/어 时 으 脱落）

차무희 씨, 오디오 들어갑니다

→ 车武熙小姐，音频要进了（导播提示对讲）

단어~씨
→ ……先生 / 小姐（成人之间的尊称，接姓名后）
💬 김민수 씨, 안녕하세요. → 金敏秀先生你好。
단어오디오
→ 音频、音响（外来语 audio）
💬 오디오 체크할게요. → 我来检查一下音频。
단어들어가다
→ 进入、进去；（节目/录制场景）开始进入某状态
들다(进) + 가다(去) 复合动词。剧组术语：오디오/카메라/방송 들어갑니다 = 现在进入音频/镜头/直播
💬 회의실에 들어갔어요. → 进了会议室。
💬 곧 방송 들어갑니다. → 节目马上开始。
문법~ㅂ니다 / 습니다
→ 最正式的"格式体"陈述句尾
动词 + ㅂ니다(母音/ㄹ结尾) / 습니다(收音结尾)。现场对讲、播报、商务最常用。
💬 곧 도착합니다. → 马上到。
🎵 音变说明
› 들어갑니다 → [드러감니다] (连音 + 鼻音化：ㅂ + ㄴ → ㅁ + ㄴ)
› 차무희 씨 → [차무히 씨] (注意 ㅎ 在母音间常弱化)

액션 캠 똑바로 들어 주세요

→ 请把运动相机拿正

단어액션 캠
→ 运动相机 (action cam，外来语缩写)
💬 액션 캠으로 찍었어요. → 用运动相机拍的。
단어똑바로
→ 笔直地、端正地、好好地（强调"对/正"）
💬 똑바로 앉으세요. → 请坐直。
💬 똑바로 말해. → 给我老实说。
단어들다
→ 拿、举（用手抬起或握住）
注意区分：들다(拿/举) ≠ 들어가다(进入)。这里是"举/握"。
💬 가방 좀 들어 주세요. → 请帮我拎一下包。
문법~아/어 주세요
→ 请（帮我）做……（请求 + 受惠）
动词词干 + 아/어 주세요。比单纯命令更礼貌。
💬 문을 열어 주세요. → 请帮我开门。
💬 사진 찍어 주세요. → 请帮我拍张照。
🎵 音变说明
› 똑바로 → [똑빠로] (紧音化：ㄱ 받침 后的 ㅂ → ㅃ)
› 들어 → [드러] (连音化：ㄹ 받침 + ㅇ → 顺移)

됐나요?

→ 这样行了吗？

단어되다 / 됐다
→ 行、可以、成了。됐다 = 되었다 的口语缩略
高频万能动词。"행了"、"够了"、"算了"都能用 됐어요
💬 됐어요, 그만! → 够了，别说了！
💬 이만하면 됐어요. → 这样就差不多了。
문법~나요?
→ ……吗？（柔和、含蓄的疑问尾，比 ~아요? 更礼貌随和）
动词/形容词 + 나요?（过去 았/었나요?）。常见于客服、确认。
💬 시간 있으세요? = 시간 있나요? → 有时间吗？
💬 다 끝났나요? → 都结束了吗？
🎵 音变说明
› 됐나요 → [됀나요] (鼻音化：ㅆ 收音遇 ㄴ → ㄴ)

아, 굽 다 나가겠네

→ 啊，鞋跟都要废了

단어굽
→ 鞋跟（특히 高跟鞋的跟）
💬 굽이 너무 높아요. → 鞋跟太高了。
💬 굽이 부러졌어요. → 鞋跟断了。
단어다
→ 全部、都（程度副词）
💬 다 먹었어요. → 都吃完了。
단어나가다
→ 出去；这里口语义 = 坏掉、报废、不能用
"X 나가다" 在口语里常表示 X 失灵/坏了：퓨즈가 나갔어 = 保险丝烧了 / 정신이 나갔어 = 神不守舍了
💬 노트북이 나갔어요. → 笔记本坏了。
문법~겠~
→ 表示推测、即将（这里 = "快要…了"）
动词词干 + 겠 + 终结词尾。表猜测/预感。
💬 비가 오겠네. → 看来要下雨了。
💬 늦겠어요. → 要迟到了。
문법~네
→ 半语感叹尾（"啊~"、"呢"），表新发现/感叹
💬 진짜 예쁘네. → 真漂亮啊。
🎵 音变说明
› 굽 다 → [굽 따] (紧音化：ㅂ 收音 + ㄷ → ㄸ)
› 나가겠네 → [나가겐네] (鼻音化：ㅆ → ㄴ 在 ㄴ 前)

저기요, 거기 우리 매니저 있죠?

→ 喂，那边有我们经纪人吧？

회화저기요
→ "喂"/"不好意思"（叫人引起注意，店员/陌生人通用）
💬 저기요, 길 좀 물어볼게요. → 不好意思，问个路。
💬 저기요, 여기 메뉴판 좀 주세요. → 服务员，给一下菜单。
단어거기
→ 那里（指对方那边/听者一侧）
여기(这边·我侧) / 거기(那边·你侧) / 저기(那边·远处)
단어매니저
→ 经纪人 (manager，外来语)
💬 매니저한테 물어볼게요. → 我去问问经纪人。
문법~죠? (~지요?)
→ ……吧？（向对方确认已知信息）
~지요? 的口语缩略。下降语调时表"是这样吧"，上升语调时表确认询问。
💬 여기 있죠? → 在这儿吧？
💬 김 선생님이시죠? → 您是金老师吧？
🎵 音变说明
› 있죠 → [읻쬬] (받침 ㅆ → ㄷ + ㅈ 紧音化为 ㅉ)

네, 누나 말씀하세요

→ 是，姐你说

단어누나
→ 姐姐（男性对姐姐的称呼）
男称：형(哥) / 누나(姐) ；女称：오빠(哥) / 언니(姐)。剧中是男经纪人称呼武熙。
💬 누나, 이거 뭐예요? → 姐，这是什么？
단어말씀
→ 话（말 的尊敬形）
말씀하다 = 말하다 的尊敬形。"请讲、您说"
💬 할 말씀 있으세요? → 您有话要说吗？
문법~(으)세요
→ 请……（尊敬命令尾，温和的请求）
动词词干 + (으)세요。这里 말씀하 + 세요 = 请说。
💬 들어오세요. → 请进。
💬 천천히 말씀하세요. → 您慢慢说。
회화네
→ 是、好（应答词，万能）
🎵 音变说明
› 말씀하세요 → [말쓰마세요] (连音化：ㅁ 后 ㅎ 弱化)

이거 구두 증정이야, 반납이야?

→ 这双高跟鞋是送的，还是要还的？

단어구두
→ 皮鞋 / 高跟鞋（区别于运动鞋 운동화）
💬 이 구두 예뻐요. → 这双鞋真好看。
단어증정
→ 赠送（书面词，常用于品牌/活动）
💬 사은품 증정 이벤트. → 赠品赠送活动。
💬 이건 증정품이에요. → 这是赠品。
단어반납
→ 归还、退还（图书/借品/试用装常用）
💬 도서관에서 책을 반납했어요. → 在图书馆还书了。
💬 회사 노트북은 퇴사할 때 반납해야 돼요. → 公司笔记本离职时要还。
문법A이야, B이야?
→ 是 A 还是 B ？（半语二选一疑问）
名词 + 이야(收音) / 야(无收音) ?。敬语版：A예요, B예요?
💬 커피야, 콜라야? → 咖啡还是可乐？
💬 농담이야, 진심이야? → 是开玩笑还是认真的？
🎵 音变说明
› 증정이야 → [증정이야] (无音变，平铺直读)
› 반납이야 → [반나비야] (连音化：ㅂ 받침 + ㅇ → 顺移)

어, 그건 반납이에요

→ 嗯，那个是要还的

회화어
→ 嗯（半语应答；比 네 随意，对熟人用）
단어그건
→ "그것은" 的缩略 = 那个（话题主语）
이건(this) / 그건(that) / 저건(that over there) — 都是 ~것은 缩略
💬 그건 제 거예요. → 那个是我的。
문법~이에요 / 예요
→ 是……（"~이다" 敬语）
收音结尾 + 이에요 / 无收音 + 예요。반납 有收音 ㅂ → 반납이에요
💬 학생이에요. → 是学生。
💬 제 친구예요. → 是我朋友。
🎵 音变说明
› 반납이에요 → [반나비에요] (连音化：ㅂ 받침 + ㅇ → 顺移)

까지지 않게 조심하세요

→ 小心别刮花了

단어까지다
→ 擦破、剐蹭、磨损（皮/漆面）
💬 무릎이 까졌어요. → 膝盖擦破了。
💬 차가 까졌어요. → 车被剐花了。
단어조심하다
→ 小心、留神
💬 차 조심하세요. → 当心车。
💬 감기 조심하세요. → 小心感冒。
문법~지 않게
→ 为了不……、别让……（目的状语，否定式）
动词词干 + 지 않게 = "不让……地"。后接动作动词。
💬 늦지 않게 출발하세요. → 别迟到，早点出发。
💬 잊지 않게 메모해 둬요. → 为了不忘记，记下来。
문법~(으)세요
→ 请……（尊敬温和命令）
💬 조심하세요. → 请小心。
🎵 音变说明
› 까지지 않게 → [까지지 안케] (激音化：ㅎ + ㄱ → ㅋ)
› 조심하세요 → [조시마세요] (ㅎ 在 ㅁ 后弱化/脱落)

📍 核心语法

▸ ~ㄴ/은 줄 알았으면 — 早知道是……的话

💬 이렇게 비싼 줄 알았으면 안 샀지. → 早知道这么贵就不买了。

▸ ~(으)라고 하다 — 让做……（命令引语）

💬 의사가 술을 끊으라고 했어요. → 医生让我戒酒。

▸ ~았/었어야지 — 本该……才对（埋怨）

💬 미리 예약했어야지. → 就该提前预约嘛。

▸ ~(으)라더니 — 之前让…，结果却…（前后矛盾）

💬 조용히 하라더니 자기가 더 시끄러워. → 让别人安静，自己更吵。

▸ ~아/어 주세요 — 请帮忙做……

💬 여기에 앉아 주세요. → 请坐这里。

▸ ~나요? — ……吗？（柔和疑问）

💬 다 끝났나요? → 都结束了吗？

▸ ~죠? (~지요?) — ……吧？（确认）

💬 맞죠? → 对吧？

▸ ~겠~ — 推测/即将

💬 비가 오겠네. → 看来要下雨了。

▸ ~지 않게 — 为了不……

💬 늦지 않게 출발해요. → 为了不迟到，早点出发。

▸ A이야, B이야? — 是 A 还是 B？

💬 농담이야, 진심이야? → 是开玩笑还是认真的？

▸ ~이에요 / 예요 — 是……（敬语判断）

💬 제 친구예요. → 是我朋友。

💡 小贴士

① 신다 / 입다 / 쓰다 / 끼다 / 차다 韩国人对"穿/戴"分得很细：鞋袜 신다、衣服 입다、帽眼镜 쓰다、戒指手套 끼다、手表 차다。

② 됐다 = 되었다 的口语缩略，万能词。"됐어!" 既能表示"行了！"也能表示"够了！" — 看语气。

③ 들다 和 들어가다 别搞混：들다 = 拿/举（手部动作），들어가다 = 进入（移动）。剧组术语 "오디오/방송 들어갑니다" 表示"音频/节目要开始了"。

④ 나가다 在口语里常表示"坏掉/报废"：굽 나가다(鞋跟废了) / 정신 나가다(神不守舍) / 퓨즈 나가다(保险丝烧了)。

⑤ 剧中 무희 称呼男经纪人 매니저，男经纪人称呼她 누나（姐），是亲近又有上下尊卑的职场关系。
