단어

어, 거기 기대면 안 됩니다, 물러서세요

→ 那里不能靠，请退后

동사기대다
→ 倚靠、靠着
💬 벽에 기대지 마세요. → 不要靠墙。
문법~(으)면 안 됩니다
→ 不能/不可以做……（禁止）
条件 ~(으)면 + 안 되다(不行)。比 ~지 마세요 更书面、正式。
💬 여기서 사진 찍으면 안 돼요. → 这里不能拍照。
동사물러서다
→ 退后、退开
물러(=退) + 서다(=站) → 站到后面
💬 한 발짝 물러서 주세요. → 请退后一步。
🎵 音变说明
› 안 됩니다 → [안됨니다] (鼻音化: ㅂ+ㄴ→ㅁㄴ)
› 물러서세요 → [물러서세요] (流音清晰)

통역사님이 웬일로 불쑥 끼어드시나?

→ 通译师怎么突然插话了？

명사통역사
→ 口译/通译师
통역(口译) + 사(职业人) ／ 翻译书面：번역가
표현웬일로
→ 怎么会、什么风把……吹来
웬일(=何事) + 로，表"难得地、出乎意料地"。
💬 웬일로 일찍 왔네? → 怎么这么早就来了？
부사불쑥
→ 突然、冷不防
💬 불쑥 찾아오지 마. → 别突然找上门来。
동사끼어들다
→ 插嘴、插队、插话
끼다(夹) + 어 + 들다(进)，构成"硬挤进来"的合成动词
💬 남의 대화에 끼어들지 마세요. → 别插别人的话。
🎵 音变说明
› 웬일로 → [웬닐로] (ㄴ添加)
› 끼어드시나 → [끼어드시나] (敬语 시 + 의문 ~나)

나 걱정해 주는 거예요? 바람에 날아갈까 봐?

→ 是在担心我吗？怕我被风吹走？

표현걱정해 주다
→ (替别人)担心、关心
~아/어 주다 = 为某人做……
💬 항상 걱정해 줘서 고마워. → 谢谢你一直为我担心。
명사바람
→ ① 风 ② 期望（同形异义）
💬 바람이 세요. → 风很大。
문법~(으)ㄹ까 봐
→ 担心/怕……（推测+顾虑）
动词 ~(으)ㄹ까 + 보다 → 怕会发生……。常省略后续动词。
💬 늦을까 봐 택시 탔어요. → 怕迟到所以打车了。
💬 비 올까 봐 우산 챙겼어. → 怕下雨带了伞。
🎵 音变说明
› 날아갈까 봐 → [나라갈까봐] (连音化: ㄹ+ㅇ)

그 앞의 표지판에 쓰여 있습니다

→ 前面的指示牌上写着呢

명사표지판
→ 标识牌、指示牌
표지(标识) + 판(板)
문법~아/어 있다
→ ……着（状态持续）
쓰다(写) → 쓰이다(被写) → 쓰여 있다(写着)。表示动作完成后的状态。
💬 문이 열려 있어요. → 门开着。
💬 의자에 앉아 있어요. → 坐在椅子上。
대조~아/어 있다 vs ~고 있다
→ 状态持续 vs 动作进行
앉아 있다 = (已坐着) ／ 앉고 있다 = (正在坐下)
🎵 音变说明
› 쓰여 있습니다 → [쓰여 읻씀니다] (鼻音化)

이탈리아어 못하는 사람은 어떡하라고?

→ 不会意大利语的人怎么办？

문법~지 못하다 / 못하다
→ 不会、不能（能力否定）
"못 + 동사" 或 "동사 + 지 못하다"。区别于 안 (意愿否定)。
💬 수영을 못해요. → 我不会游泳。
회화어떡하다
→ 怎么办（어떻게 하다 的口语缩略）
💬 나 어떡해? → 我怎么办呀？
문법~(으)라고?
→ 让(我)……？（反问命令）
命令引语 ~(으)라고 하다 的省略，表"叫我怎样？"的反问。
💬 이 시간에 오라고? → 这时候叫我来？
🎵 音变说明
› 못하는 → [모타는] (激音化: ㄷ+ㅎ→ㅌ)
› 어떡하라고 → [어떠카라고] (激音化: ㄱ+ㅎ→ㅋ)

영어로도 써 놨어야지

→ 也该用英语写上才对啊

문법~(으)로
→ 用……（手段、工具、语言）
💬 한국어로 말해 주세요. → 请用韩语说。
문법~아/어 놓다
→ ……好(放着)（动作完成后状态保留）
써 놓다 = 写好放着 ／ 사 놓다 = 买好放着 ／ 해 놓다 = 做好放着
💬 문을 열어 놨어요. → 门开着了（开好了放在那）。
문법~었어야지
→ 本该做……（带责备语气）
复习: 일찍 일어났어야지! → 就该早点起来啊！
🎵 音变说明
› 써 놨어야지 → [써놔써야지] (썼어 → 써 놨어, 紧音化)

그쪽이 못 읽은 거지

→ 是你没读出来罢了

대명사그쪽
→ 那边／你（中性指代陌生人）
用于不熟、平辈或下位者。比 당신 更口语。
💬 그쪽이 먼저 사과하세요. → 你先道歉。
문법~(으)ㄴ 거지
→ ……罢了/不就是……嘛（确认/解释语气）
~(으)ㄴ 것이지 的口语缩略。陈述+轻微反驳。
💬 모르는 거지 못하는 게 아니야. → 是不知道罢了，不是不会。
🎵 音变说明
› 못 읽은 → [몬닐근] (ㄴ添加+鼻音化)

둘이 비슷비슷해서 헷갈려서 그런 거죠

→ 两个长得很像才会混淆

형용사비슷비슷하다
→ 差不多、相似
叠词 비슷 + 비슷하다 → 强调"都差不多"
💬 가격이 다 비슷비슷해요. → 价格都差不多。
동사헷갈리다
→ 混淆、搞不清
💬 이름이 헷갈려요. → 名字搞混了。
💬 헷갈리지 마. → 别搞混了。
문법~아/어서 그런 거죠
→ 因为……才那样的吧（说明原因）
💬 피곤해서 그런 거죠. → 因为累才那样的吧。
🎵 音变说明
› 비슷비슷해서 → [비슫삐스태서] (紧音化+激音化)
› 헷갈려서 → [헫깔려서] (紧音化)
📌 单词·语法 总览

📍 单词

▸ 통역사 — 通译师/口译员

▸ 표지판 — 标识牌

▸ 바람 — 风

▸ 비슷비슷하다 — 差不多

▸ 헷갈리다 — 混淆

▸ 끼어들다 — 插话/插队

▸ 물러서다 — 退后

📍 核心语法

▸ ~(으)면 안 됩니다 — 不能/不可以做……

💬 여기서 담배를 피우면 안 돼요. → 这里不能抽烟。

▸ ~(으)ㄹ까 봐 — 怕会……

💬 잊어버릴까 봐 메모했어요. → 怕忘了所以记下来了。

▸ ~아/어 있다 — ……着（状态持续）

💬 책상 위에 책이 놓여 있어요. → 书桌上放着书。

▸ ~아/어 놓다 — 做好(放着)

💬 음식을 만들어 놨어요. → 做好饭了。

▸ ~(으)라고? — 让(我)……？（反问）

💬 이걸 다 먹으라고? → 这些全要我吃？

💡 小贴士

① 韩剧里 '그쪽' 很常见，是对陌生人不算很客气但也不算粗鲁的称呼。比 '당신' 更安全。

② 웬일 vs 왠지：웬일=何事(为什么会这样) ／ 왠지=不知为何(理由不明)。容易拼错。

③ 韩国职场常见反问 "~었어야지"，含责备意味，慎用！

④ 통역사 무희가 무희한테 '그쪽'이라고 한 건 거리감을 두는 표현이에요.
