## 四、line-exp 字段说明

| 字段 | 说明 | 示例 |
|:----|:-----|:-----|
| 📍单词 | 本条台词涉及的单词+语法词尾拆解 | 이름 — 名字(name) ｜ 뭐 — 什么(what) ｜ -예요 — 是(is/am) |
| 💬短语 | 用重点单词造的短搭配 | 연락처가 뭐예요? — 联系方式是什么？(What's your contact?) |
| 💬造句 | 用该句语法点的完整造句 | 이 책 제목이 뭐예요? — 这本书叫什么名字？(What's the title of this book?) |
| 📌语法 | 语法点讲解 + English explanation | -이/가 뭐예요? 询问名称 (asking for a name/identity) |
| 💬语法例句 | 语法点的延伸例句（中英双语） | 내일이라고요? — 你说明天？(Did you say tomorrow?) |
| 🎵音变 | 实际发音+音变规则 | 뭐예요 → [뭐예요] |

每条台词在 `.cn` 翻译下方增加 `.en` 英语翻译行：
```html
<p class="cn">→ 中文翻译</p>
<p class="en" style="font-size:12px;color:#888;margin:2px 0 0 16px">→ English translation</p>
```

## 五、LESSON_TIMES

```javascript
n:[
  {start:秒,end:秒},{start:秒,end:秒},...
]
```

### ⚡ 时间戳注意事项

1. **精准匹配视频实际说话时间**，每段台词结束时间延伸到下一句开始前，避免间隙跳转
2. **前10秒无对话**时，第一句从0秒开始高亮，避免打开页面时无台词亮着
3. 时间戳数组长度必须与 `lines` 数组完全一致

## 六、功能说明

- **视频固定在页面顶部不动**（position:sticky;top:52px），下面的台词讲解可以自由滚动
- 播放时台词**自动高亮联动**，当前台词自动滚入可视区域
- **点击台词**（韩文或中文）视频跳转到对应位置播放
  - ⚡ **即时高亮反馈**：点击后立刻高亮所选台词，不等视频加载
  - ⏳ 如果视频未加载完成，等待加载后自动跳转
  - 🔇 点击🔊朗读按钮不会触发跳转
- 点▸展开/收起单词·短语·造句·语法·音变
- 测验模式：从已学课程抽单词四选一
- 已完成标记 + 自动保存笔记

### 口语练习
- 点击「🎤 口语练习」按钮进入
- **5种练习模式**：

| 模式 | 说明 | 数据来源 |
|:----|:-----|:---------|
| 🎯 对话练习 | 看中文说韩语句子 | `LESSONS[n].lines` |
| 📝 单词抽测 | 中韩互译（交替出题） | `LESSONS[n].vocab` |
| ✏️ 单词造句子 | 用给定单词造完整韩语句 | `vocab` + `wordSentences` |
| ✏️ 短语造句子 | 用短语扩写成完整句子 | `phrases`（自动从lines提取） |
| 🔤 句型造句子 | 按句型框架造句 | `patterns`（自动从tip提取） |

- 自动抽取5题一轮，逐词比对显示正确/错误/漏词
- 支持打字输入（所有浏览器）+ 语音录音（Chrome桌面）
- 每轮结束显示正确率，可回顾错题

> ⚠️ 口语练习数据自动从 LESSONS[n] 读取，所以 LESSONS 中的 lines 和 vocab 必须与视频内容一致

## 七、生成规则

每条台词必须包含完整6项讲解：
```
📍单词 → 💬短语 → 💬造句 → 📌语法 → 💬语法例句 → 🎵音变
```

六个字段缺一不可，短语和造句不能共用同一个例句。
