## 🖌️ 偏旁练习

### 大字底 — 「奖」「类」

**口诀：** 大字底撇捺撑开，托稳上部

```
奖 → 左上"将"+ 右下"大"为底
      大字的撇捺要左右撑开，稳稳托住上面
类 → 上"米"+ 下"大"为底
      米字收紧，大字撇捺舒展展开
```

🔑 大字底是整字的支撑，撇捺角度要开，撑出气势

### 大字头 — 「奇」「牵」

**口诀：** 大字头撇捺盖住，如伞覆下

```
奇 → 上"大"盖住下"可"
      大字的撇捺要向两侧打开，覆盖下面的"可"
牵 → 上"大" + 下"玄"（中间凸宝盖）
      大字头撇捺像伞一样盖住下面的部件
```

🔑 大字头撇捺伸展开张，像一把伞为下部遮风挡雨

## 💡 要点提醒

1. **大字底** — 撇捺要撑开，别缩着写，底部要稳如磐石
2. **大字头** — 撇捺像伞盖，盖住下部但不要压抑
3. **横折** — 折角往回带0.5mm，不是直角刀切
4. **点** — 宁慢勿虚，轻入→重顿→回锋，三拍到位
