# 语法填空智能练习产品 — 需求文档（给Claude Code看）

---

## 一句话描述

一款面向初中/高中学生的语法填空在线练习工具。学生做题 → 自动批改 → 分析错因 → 总结薄弱点 → 针对性出题，形成学习闭环。老师可在课上使用，学生也可回家自学。

---

## 技术栈

- 前端：HTML + CSS + JavaScript（单页应用）
- 后端：不需要后端，纯前端
- 数据存储：localStorage（学生做题记录 + 错题本 + 薄弱点统计）
- 外部依赖：无（所有功能前端实现）

---

## 目标用户 & 受众规模

- 🟡 中等 — 初中/高中英语老师及学生
- 核心用户：你（英语老师），用来课堂教学 + 布置作业
- 最终用户：你的学生

## MVP 功能（按优先级）

### P0（必须有）

#### 1. 题目录入
- 学生通过粘贴/打字输入文章
- 系统自动识别空位（用 `___` 或括号标记）
- 每个空位生成对应的 **输入框**

#### 2. 答题 & 批改
- 学生在每个空位输入答案
- 点击提交 → 系统自动判对错
- 正确的标 ✅，错误的标 ❌ 并显示正确答案

#### 3. 解题思路展示
- 每个空位可点击展开「查看思路」
- 下拉显示完整解题步骤（基于你的教学体系）
- 思路格式：
  - 有提示词/无提示词
  - 决策树路径（先看什么 → 再看什么 → 得出结论）
  - 为什么是这个答案

#### 4. 自动纠错 + 错因分析
- 答错后自动弹出错因分析
- 错因分类：
  - 时态/语态错误
  - 非谓语形式错误
  - 介词搭配错误
  - 从句引导词错误
  - 词性判断错误
  - 长难句理解错误
  - 生词导致错误
  - 其他

#### 5. 学习报告
- 做完一篇题后生成总结报告：
  - 得分率
  - 各知识点正确率
  - 薄弱知识点排名
  - 建议优先复习哪些知识点

#### 6. 针对性出题
- 根据薄弱点，自动生成新的语法填空题
- 题型风格与真题一致，难度适配初中/高中

### P1（最好有）

#### 7. 初中/高中分级
- 学生选择学段
- 题目难度、讲解深度、词库自动匹配

#### 8. 历史记录
- 查看过去做的所有题
- 薄弱点变化趋势（这周 vs 上周）

### P2（后续加）

#### 9. 拍照识别
- 拍照纸质试卷 → 自动提取题目

#### 10. 语音输入
- 语音朗读 → 转文字录入

## 页面 / UI 描述

### 首页
- 标题「语法填空智能练习」
- 两个大按钮：初中 / 高中
- 底部显示历史统计数据

### 做题页
- 顶部：文章标题 + 学段标识
- 中间：文章正文，每个空位显示为一个**输入框**（短下划线样式）
- 右侧/底部：提交按钮
- 提交后：每个空下方显示 ✅/❌ 标记和正确答案

### 思路弹窗
- 点击空位旁的「💡 思路」按钮
- 弹出下拉卡片，展示：
  - 该空位的考点类型
  - 决策树路径
  - 具体解析
  - 易错提示

### 报告页
- 得分概览（环形图）
- 知识点雷达图
- 薄弱点列表 + 推荐练习按钮
- 生成新题按钮

## 交互流程

```
① 选择初中/高中
    ↓
② 粘贴/输入题目文章（用 ___ 标记空位）
    ↓
③ 系统识别空位 → 显示带输入框的文章
    ↓
④ 学生逐空填写
    ↓
⑤ 点击提交 → 自动批改
    ├─ 正确 → ✅ 绿色标记
    └─ 错误 → ❌ 红色标记 + 显示正确答案 + 弹出错因分析
    ↓
⑥ 学生可点击每个空的「💡 思路」查看解题过程
    ↓
⑦ 完成所有题 → 生成学习报告
    ↓
⑧ 点击「生成针对性练习」→ 根据薄弱点出新题
```

## 示例数据

### 输入示例
```
Text: "I _____ (go) to school by bus every day. My mother _____ (be) a teacher. She _____ (work) in a hospital near our home."
```

### 输出示例
```
空1: go → ✅ 正确
空2: is → ✅ 正确  
空3: works → ❌ 错误（正确答案: works）
   错因分析: 时态错误。主语she是第三人称单数，一般现在时加-s
   思路: 有提示词work → 句子无其他谓语 → 缺谓语 → 一般现在时主谓一致 → works
```

- 解题思路基于你的英语教学体系
