第1课 · 拍摄现场

📝 台词 · 详解

1. 이런 곳인 줄 알았으면 운동화를 신으라고 했어야지

→ 早知道是这种地方，就该让我穿运动鞋

📖 单词

• 곳 — 地方 ｜ • 운동화 — 运动鞋 ｜ • 신다 — 穿（鞋）

🔍 语法

• -(으)ㄴ 줄 알았으면: '还以为…'，表示事实与事实相反
• -라고 하다: 间接引语，'让/叫某人做某事'
• -았/었어야지: '本该…'，表后悔或责备

2. 아 참 잘할만하네, 이게 뭐야

→ 啊真是的，这什么啊

📖 单词

• 참 — 真/真是 ｜ • 잘하다 — 做得好 ｜ • 이게 — 这个（이거+가）

🔍 语法

• -ㄹ 만하다: '值得…/还算…'，表示对某事的评价
• -뭐야: '什么啊'，口语中表示不满或惊讶

3. 발 진짜 아파

→ 脚真的好痛

📖 单词

• 발 — 脚 ｜ • 진짜 — 真的 ｜ • 아프다 — 痛

🔍 语法

• 韩语中形容词可以直接作谓语，아파是아프다+아(해체，半语)

4. 아 잠도 차려입으라더니

→ 啊，让我打扮得漂漂亮亮地来

📖 单词

• 잠도 — 觉都(-도表示'也/都') ｜ • 차려입다 — 盛装打扮

🔍 语法

• -라고 하더니: '说要…结果却…'，表他人说过的话与实际情况不符
• -라더니: 对别人说过的话进行回顾，带有转折语气

5. 아 다리 아파 아이씨

→ 啊腿好痛，靠

📖 单词

• 다리 — 腿 ｜ • 아파 — 痛（아프다的活用） ｜ • 아이씨 — 感叹词（类似'靠/ damn'）

🔍 语法

• 아이씨/아이고: 韩语中表示疼痛、无奈、不满的感叹词，语气比'아이고'更强

6. 참호희 씨 오디오 들어갑니다

→ 车武熙小姐，音频要进去了

📖 单词

• 씨 — 小姐/先生（敬称） ｜ • 오디오 — 音频(audio) ｜ • 들어가다 — 进入

🔍 语法

• -ㅂ니다/습니다: 敬语陈述句尾，正式场合使用
• 这是拍摄现场导演提醒演员的用语

7. 액션캠 똑바로 들어 주세요

→ 请把运动相机拿好

📖 单词

• 액션캠 — 运动相机 ｜ • 똑바로 — 端正地 ｜ • 들다 — 拿

🔍 语法

• -아/어 주세요: '请为我做…'
• 똑바로 하다: '好好做/端正地做'

8. 됐나요?

→ 好了吗？

📖 单词

• 되다 — 行/完成

🔍 语法

• -았/었나요?: 确认疑问句尾，比-아요?更柔和

9. 아 곧 더 나가겠네

→ 啊，鞋跟要坏了

📖 单词

• 곧 — 马上 ｜ • 나가다 — 坏掉

🔍 语法

• -겠네: 推测+感叹，'好像要…了'

10. 저기요 거기 우리 매니저 있죠

→ 喂，我们经纪人在吗？

📖 单词

• 저기요 — 喂 ｜ • 매니저 — 经纪人

🔍 语法

• -죠?: 确认语气，'…吧？'

11. 네 누나 말씀하세요

→ 是，姐姐您请说

📖 单词

• 누나 — 姐姐 ｜ • 말씀하다 — 说（敬语）

🔍 语法

• -세요: '请…'的敬语命令式

12. 이거 구두 증정이와 반납이요

→ 这高跟鞋是赠送的还是归还的？

📖 单词

• 구두 — 高跟鞋 ｜ • 증정 — 赠送 ｜ • 반납 — 归还

🔍 语法

• -이야? / -이요?: 疑问句尾，이야更随意이요稍客气

13. 어 그건 반납이에요

→ 啊，那是要归还的

📖 单词

• 그건 — 那个是 ｜ • 반납 — 归还

🔍 语法

• -이에요: '是…'，肯定陈述

14. 까지지 않게 조심하세요

→ 小心别磨破了

📖 单词

• 까지다 — 磨破 ｜ • 조심하다 — 小心

🔍 语法

• -지 않게: '以免…'

15. 어 오케이

→ 嗯 OK

📖 单词

• 어 — 嗯 ｜ • 오케이 — OK

🔍 语法

• 韩语口语中常用英语外来词

16. 예 지금처럼 참으시고 컬러 해주세요

→ 好的，像现在这样忍着，请给颜色

📖 单词

• 지금 — 现在 ｜ • 참다 — 忍耐 ｜ • 컬러 — 颜色

🔍 语法

• -처럼: '像…一样' ｜ • -시-: 敬语词尾

17. 네 이분들은 창원 씨 정면으로 더 들어가 주세요

→ 好的，请从昌原先生正面再靠近一些

📖 单词

• 이분들 — 各位 ｜ • 정면 — 正面

🔍 语法

• -(으)로: 方向助词 ｜ • -아/어 주세요: 请做

第2课 · 翻译到达

📝 台词

1. 박 피디, 통역 인이어 연결됐지?

→ 朴PD，同传耳机连上了吧？

2. 통역사님 오셨어요?

→ 翻译老师您来了？

3. 통역사님, 준비되셨어요?

→ 翻译老师，准备好了吗？

4. 네, 저도 준비됐습니다

→ 嗯，我也准备好了

📌 单词

통역사 → 翻译（口译员） ｜ 인이어 → 入耳式耳机(in-ear) ｜ 연결되다 → 连接上 ｜ 준비되다 → 准备好了 ｜ 오셨어요 → 您来了

💡 小贴士

'-셨어요?'是'-(으)셨어요?'的缩略，表示敬语过去疑问句。'통역사님 오셨어요?'=翻译老师您来了？是韩语职场最常用的问候之一。
