# 📓 AI创作记录 · 提示词

## 用途
把自己用AI工具做产品的心路历程记录下来，存档用，风格自由。

---

## 背景信息（如实填）

**我是谁：**
- 一个素人，不是程序员

**时间跨度：**
- 大约2026年5月开始

**用到的工具：**
- OpenClaw + Claude Code + DeepSeek / ChatGPT + 豆包 + 腾讯云

**做过的项目：**
- 词根精讲PPT / 五维词汇测试网站 / 语法填空工具
- 高考听力PPT素材 / 外刊精讲PPT生成器 / 电影学单词材料

---

## 记录要求

不需要任何修饰，就当写日记，想到哪写到哪：

1. **每天做了啥**
   - 今天在搞什么项目
   - 遇到了什么问题
   - 怎么解决的

2. **有啥感受**
   - 觉得这个工具有意思的地方
   - 被什么坑到了
   - 做成什么之后有点爽

3. **学到了什么**
   - 什么技巧下次能用上
   - 什么方法不好用
   - 对AI或者做产品有什么新理解

4. **下一步打算**
   - 接下来想搞什么
   - 有什么想法冒出来了

**没有格式限制，没有字数要求，就是真实记录。** 以后想翻的时候知道自己走过哪些路。
